Velkommen til SendBlaster, den enkle og kraftige softwaren for administrering av mailinglister fra din PC:

  • Håndtere multiple lister
     
  • Generere nyhetsbrev med vedlegg og embedded'e bilder
       De nyeste operasjonsmiljøer tillater ikke visning av remote bilder: bilder er derfor embedded i meldingstekstern, dermed unngås alle problemer relatert til lesing av meldinger
     
  • Importer HTML-sider og meldinger generert med e-postklient (.eml-format)
       Ekstern bildereferanse i HTML-kode blir embeddet i meldingen, og alle vedlegg funnet i .eml-filen blir bevart.
     
  • Send personaliserte meldinger
       Hver mottaker kan motta en personalisert melding: e-postmeldinger kan inneholde tag'er som erstattes med mottakerens data
     
  • Muliggjør direct send, uten SMTP-server
       Ved hjelp av direct send-modus, kan du bruke mange internett-tilkoblinger uten å måtte modifisere innstillinger for å koble til tilbyderens SMTP-tjeneste: meldingene vil bli overført direkte til mottakerenes e-postservere.
     
  • Send mange meldinger fort
       Takket være den kraftigeSmartSend -motoren kan mange meldinger sendes samtidig, og dermed økes sendehastigheten dramatisk.
     
  • Administrer abonnering og av-abonnering fra en dedikert e-postadresse
       Brukere kan abonnere og av-abonnere ganske enkelt ved å sende til en brukerdefinert e-postadresse med "Subscribe" eller "Unsubscribe " i emnefeltet. SendBlaster laster ned meldingene via POP3 og oppdaterer listene.
     
  • Importer from Outlook Express' adressebok
     
  • Importer lister fra .csv-filer og web-ressurser
       Hvis ditt web-område allerede er utstyrt med et system for abonnering/av-abonnering og et eksporterinsscritp, kan SendBlaster motta og laste data, og deretter oppdatere din lokale database
     
  • Enkelt å integrere med ethvert web-område
       Å legge til funksjonalitet for abonnering/av-abonner til mailinglister på ditt web-område er enkelt: ingen script nødvendig, bare legg til en enkel link
