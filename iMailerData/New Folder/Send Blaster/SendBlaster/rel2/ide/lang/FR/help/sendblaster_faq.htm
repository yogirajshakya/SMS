Foire aux Questions (FAQ)

Quand j'envoie un message à plus d'un destinataire, le destinataire voit-il les adresses des autres destinataires ?

Non, SendBlaster procède pour chaque message à un envoi séparé et protège les données personnelles des destinataires. L'utilisation de SendBlaster pour l'envoi de courriels à destinataires multiples vous empêche également de révéler accidentellement les adresses des autres destinataires, ce qui constituerait une violation de la protection des données personnelles.
 

Comment filtrer les listes ?

En utilisant les fonctions de classement et de regroupement de l'onglet Gérer les listes, et en sélectionnant ou en désélectionnant momentanément un groupe d'adresses, vous pouvez envoyer un courriel à un sous-ensemble de la liste sélectionnée. Vous pouvez également utiliser le bouton Filtre

Comment inclure dans le corps du message un lien pour se désabonner de la liste de diffusion ?

SendBlaster peut gérer ses listes de diffusion en téléchargeant les messages d'une adresse de courriel dédiée, qui doit être différente pour chaque liste de distribution (Gérer les inscriptions). En utilisant le bouton Insérer un lien pour se désabonner, lors de la composition d'un message, vous inserez ainsi un lien tel que: 

<a href="mailto:#courieldegestion#?Subject=Unsubscribe">Se désabonner</a>

Le code HTML sera inséré dans la section HTML du message. Vous pouvez le réviser/éditer en cliquant sur Afficher le code.

Lorsque j'envoi un message par le biais d'une liste de diffusion, certains messages ne sont pas livrés et me sont renvoyés avec un message d'erreur

Lorsque vous envoyez à nombre d'adresses de courriels, il est normal que tous les envois de messages n'aboutissent pas: des adresses pourraient ne plus être actives ou bien momentanément engorgées, certains serveurs pourraient être surchargés et la connexion pourrait être momentanément ralentie ou interrompue.

Quand le pourcentage de messages envoyés est trop bas, ou si aucun message ne part, il y a lieu de vérifier les Paramètres de l'Envoi.

En particulier, il pourrait être nécessaire de modifier la modalité d'envoi (direct ou SMTP) et, si votre serveur SMTP est doté d'un filtre anti-pourriel, diminuer le nombre de connexions et augmenter la pause entre les messages. Le serveur SMTP pourrait avoir été configuré pour empêcher l'envoi d'un nombre excessif de messages pour un intervalle donné (par exemple, un serveur peut être configuré pour envoyer un maximum de 50 messages à l'heure); dans ce cas, après le premier envoi, le reste des messages sera complété plus tard en utilisant les fonctions du Journal.

Si la configuration anti-pourriel du SMTP ne peut être contourné et que si l'envoi ne fonctionne qu'avec des paramètres d'envoi très lents, nous conseillons d'essayer l'envoi direct ou de se connecter à un serveur SMTP plus "tolérant" (sur le réseau Internet vous pouvez trouver des services SMTP particulièrement efficaces à un coût limité) .

Si le problème se manifeste durant l'envoi direct, il peut être nécessaire de saisir manuellement l'adresse IP du serveur DNS.
 

J'ai perdu le mot de passe et je ne peux plus démarrer le logiciel. Que puis-je faire ?

Soyez conscients que si cela devait arriver, toutes vos données seraient perdues. Toutefois, la procédure qui suit vous permettra de rétablir les fonctions du logiciel.

1. Désinstaller le logiciel à travers le Panneau de Configuration de Windows.

2. Effacer manuellement le répertoire d'installation du logiciel (par ex. C:\Programmes\SendBlaster) ainsi que tous les sous–répertoires; pour des raisons de sécurité, même en cas de désinstallation du logiciel, la base de données n'est pas automatiquement éliminée (c'est pourquoi vous devrez l'éliminer manuellement).

3. Réinstaller le logiciel.
