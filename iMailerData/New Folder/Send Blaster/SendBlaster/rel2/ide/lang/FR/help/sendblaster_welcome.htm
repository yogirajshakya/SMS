SendBlaster, le logiciel puissant et versatile qui vous permet de gérer vos listes de diffusion depuis votre PC, vous souhaite la bienvenue:

  • Gérez des listes de diffusion multiples
     
  • Créez des lettres d'information contenant des pièces jointes et des images incorporées
       Les plus récents systèmes d'exploitation empêche les messages de faire appel à des images résidant ailleurs, ce logiciel incorpore l'image dans le message et prévient les problèmes de lecture.
     
  • Importez des pages HTML et des messages générés par votre client de courriel (format .eml)
       Les images externes sont incorporées dans le code HTML et toutes les pièces jointes du fichier .eml sont conservées
     
  • Envoyez des messages personnalisés
       Chaque destinataire peut recevoir un message personnalisé : le texte du courriel peut contenir des étiquettes qui seront remplacées au moment de l'envoi par les données relatives au contact
     
  • Envoyez des messages directement sans passer par un serveur SMTP
       En choisissant l'envoi direct, si vous utilisez plus d'un fournisseur d'accès, vous ne devrez plus modifier les paramètres de connexion au serveur SMTP de votre f.a.i.: les messages seront directement livrés aux serveurs de courriels des destinataires
  • Envoyez vos messages rapidement
       Le moteur SmartSend peut envoyer plusieurs messages simultanément. Ce qui augmente dramatiquement la vitesse d'envoi.
     
  • Gérer automatiquement les abonnements et les désabonnements sur une adresse de courriel réservée
       Vos abonnés peuvent s'abonner et se désabonner de vos listes en expédiant un message ayant pour objet "Abonner" ou "Désabonner" à une adresse de courriel définie. SendBlaster télécharge ces messages reçus par le biais d'un POP3 et met à jour vos listes de diffusion.
     
  • Importez les contacts présents dans le carnet d'adresses de Outlook Express
     
  • Importez des listes d'adresses venant de fichier en format .csv et de ressources sur le Web
       Si votre site Web est doté d'un système d'abonnement et de désabonnement, et qu'il possède un script pour l'exportation des adresses, SendBlaster peut alors directement prendre et télécharger des données pour mettre à jour la base de données locale.
     
  • Intégrez-le facilement à n'importe quel site Web
       Doter votre site Web des fonctionnalités d'abonnement/désabonnement à une lettre d'information est simple: vous n'avez pas besoin de script, il vous suffit de poser un lien vers l'adresse électronique de gestion
