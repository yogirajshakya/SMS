Paramètres avancés

Dans cette fenêtre, vous pouvez définir quelques options avancées du programme.

Interface:

  • Aspect: permet de spécifier si le panneau latéral devrait être montré avec une barre de style Explorer (éventuellement avec des petites icônes, afin d'économiser de l'espace), ou le style Barre d'outils.
     
  • TrackReports affiche un lien dans le panneau: permet d'afficher / cacher le lien TrackReports dans le panneau principal. Si vous ne prévoyez pas utiliser TrackReports, vous pouvez masquer ce lien pour économiser l'espace sur la barre.
     
  • Afficher les liens Google Analytics dans le menu: permet d'afficher / cacher le lien Google Analytics dans le panneau principal. Si vous ne prévoyez pas utiliser Google Analytics, vous pouvez cacher ce lien pour économiser l'espace sur la barre.
     
  • Afficher des conseils au démarrage: permet d'afficher la fenêtre de conseils lorsque SendBlaster démarre. C'est une bonne idée de laisser cette option activée jusqu'à ce que vous vous familiarisez avec SendBlaster.
     
  • Synchronize la liste de sélection à travers les menus: c'est une option très puissante, qui vous permet de continuer à travailler sur la même liste dans tous les menus jusqu'à ce que vous la changiez. A titre d'exemple, si vous sélectionnez une liste dans l'écran Envoyer, puis allez à la section de Gérer les listes afin d'ajouter une nouvelle adresse, la liste que vous avez sélectionné dans l'écran Envoyer sera déjà sélectionné dans la section Gérer les listes. L'inconvénient est que si vous avez des listes trop longues et un PC bas de gamme, cette option pourrait ralentir le programme un peu: si vous rencontrez des retards gênants dans la réactivité de l'interface, simplement désactiver cette option.
     
  • Utilisez le bouton Langue pour changer la langue de l'interface utilisateur de Sendblaster. La langue sélectionnée sera utilisée la prochaine fois que vous démarrerez SendBlaster.

Messages:

  • Le codage par défaut pour les nouveaux messages: Cette option vous permet de définir le caractère de codage par défaut utilisé pour la composition de nouveaux messages.

Abonnement par courriel:

  • Cette section permet de spécifier les lignes d'objet qui devrait être utilisé pour les messages d'abonnement et de désabonnement. Vous pouvez aussi choisir si les messages doivent exactement correspondre ou non (afin d'être traités comme des abonnements ou des annulations) ou si les messages ont tout simplement besoin d'inclure les mots spécifiés. Pour des raisons de compatibilité avec les versions précédentes du programme, les objets ayant «Abonnement» et «Désabonnement» seront toujours acceptés (en plus des sujets que vous définissez ici).

Formats personnalisés des rebonds:

  • En utilisant le bouton Modifier formats, vous pouvez ajouter ou modifier les formats de rebonds sur mesure. Puisque les formats de courriels de rebonds ont été initialement conçu pour être lu par les humains, et qu'il n'y a pas de standard, SendBlaster peut traiter tous les formats de rebonds les plus fréquents en utilisant sa base de données interne. Mais vous pourriez rencontrer quelques formats qui ne sont pas reconnus par le logiciel. Si cela arrive, vous pouvez les ajouter manuellement à la base de données des rebonds de Sendblaster afin que SendBlaster puisse également traiter ces nouveaux formats de rebonds.

Planifier:

  • Cette section permet de configurer SendBlaster pour démarrer en mode calendrier (minimisé dans la zone de plateau de système) au démarrage de Windows; de cette façon des envois réguliers de courriels peuvent être démarrer automatiquement à échéance.

Envoyer:

  • Cette section permet de spécifier si vous voulez que SendBlaster envoie des duplicatas de courriels à vos contacts lorsque les contacts existent dans des listes plus d'une fois. Habituellement, vous n'avez pas besoin d'envoyer plus d'une copie de chaque courriel à vos contacts. Nous vous conseillons de laisser cette option désactivée. Sauf si vous avez une raison très particulière et spécifique pour envoyer des duplicatas de courriels à des duplicatas de contacts.

Extensions (plugiciels):

  • Cette section vous permet actuellement d'activer ou de désactiver les extensions installées dans SendBlaster (sans les désinstaller, ce que vous pouvez faire à partir du Panneau de configuration Windows ou par l'utilisation du programme de désinstallation de chaque extension). Vous pouvez également télécharger d'autres extensions SendBlaster en utilisant le bouton Plus d'Extensions.

Base de données:

  • En utilisant le bouton Dossier Base de Données, cela ouvrira le dossier Windows où les données utilisateur sont enregistrées. Ce qui est utile pour la maintenance, la sauvegarde et d'autres opérations de base de données.
