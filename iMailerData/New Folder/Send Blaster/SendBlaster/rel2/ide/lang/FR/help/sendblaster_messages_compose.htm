Composer un message

La fenêtre pour la composition du message permet de créer un message qui sera envoyé à une liste de diffusion.

La composition du message est semblable à la composition d'un message dans un client de courriel électronique traditionnel: saisir l'Objet du message, ajouter les éventuelles Pièces Jointes*, spécifiez un destinataire de courrier électronique (utilisé uniquement pour la fonction de prévisualisation du courriel) et tapez le texte du message. Si nécessaire, vous pouvez utiliser des alphabets spéciaux (cyrillique, grec, etc...) en spécifiant un Encodage particulier (vous pouvez définir l'encodage prédéfini dans la fenêtre Paramètres des Champs).

Les Étiquettes peuvent être insérées et utilisées en appuyant sur les boutons d'étiquettes (#) à côté des champs Sujet et Destinataire afin d'envoyer des en-têtes de courriers personnalisés.

Attention: pour un affichage correct du message, il est important de définir l'encodage avant de commencer la composition de votre texte. Dans le cas contraire, le texte que vous avez composé sera illisible.

L'écran affiche deux zones de texte:
  • la zone principale, caractérisée par une barre d'outils, permet de formater du texte au format HTML. Vous avez la possibilité d'ajouter des images et des liens. Vous pouvez à tout moment contrôler (et modifier) votre code HTML généré automatiquement en cliquant le bouton Afficher le code présent dans la barre supérieure.
     
  • Dans la zone Texte pur, vous pouvez saisir le contenu du message au format texte. Certains destinataires pourraient ne pas être en mesure de lire les messages HTML et il est bon de fournir une version alternative du contenu, sans images et/ou liens. Le texte pur peut être généré automatiquement à partir du texte mis en forme en cliquant le bouton HTML > Texte lorsque la composition du message est terminée.

Pour envoyer non pas un message général mais un message personnalisé, dans l'Objet et le corps du message (aussi bien dans la zone HTML que dans la zone texte pur), vous pouvez utiliser les étiquettes. Chaque étiquette correspond à un champ: lorsque le message sera envoyé, l'étiquette sera remplacée par le texte qui se trouve dans le champ correspondant pour chaque destinataire. Supposons que la liste possède un champ "Société" et un champ "Adresse" (sans oublier que le nom des champs peut être modifié). Un message qui contient le texte

#Société#
#Adresse#

parviendra personnalisé aux destinataires, comme dans cet exemple:

eDisplay srl
via Wildt 19/3

Puisque le nom des champs peut être modifié, vous pouvez avoir recours aux étiquettes "de réparation", qui indépendamment du nom du champ se réfèrent à sa position (#1#, #2#, et ainsi de suite jusqu'à #10#) - consultez la position des champs dans la fenêtre Paramètres des Champs. Il existe aussi des étiquettes spéciales: #nomdelaliste# est remplacé par le nom de la liste de diffusion; #baldegestion# par l'adresse utilisée pour la gestion des abonnements/désabonnements à la liste, que vous avez définie dans le zone Gérer les inscriptions. Cela vous permet en toute simplicité de poser un lien qui permette au destinataire de se désabonner de la liste. Enfin #datesysteme# qui est remplacée par la date actuelle formatée selon les paramètres de Windows.

     

Poser un lien pour permettre au destinataire de se désabonner

SendBlaster peut gérer vos listes de diffusion en téléchargeant les messages directement d'une adresse de courriel dédiée, différente pour chaque liste (Gérer les inscriptions). Utiliser les étiquettes pour inclure dans le code HTML un lien qui permet de se désabonner est extrêmement simple:

<a href="mailto:#gestioncourriel#?Subject=Unsubscribe">Se désabonner </a>

Ce code HTML peut être facilement inclus dans le message HTML en utilisant le bouton Insérer du lien de désabonnement de la barre des boutons, qui permet également de spécifier le texte qui sera lié à l'URL de suppression.

      

Vous pouvez enregistrer le message courant au format .eml (un format que la plupart des clients de courriel sont capable d'interpréter) en utilisant le bouton Enregistrer.

Pour créer un message, vous pouvez partir de rien (bouton Nouveau ou Choisir un modèle) ou, en utilisant le bouton Ouvrir, ouvrez un message enregistré précédemment:

  • importer un fichier .eml (enregistré avec Outlook par ex.) en choisissant Importation / Importer courriel .eml
  • récupérer un courriel que vous avez déjà envoyé avec Importation / Importer courriel du Journal
  • importer le texte formaté d'une page HTML se trouvant sur votre disque dur et qui inclut éventuellement des images avec Importation / Importer fichier HTML
  • importer du texte pur d'un fichier externe avec Importation / Importer du texte pur

Le bouton HTML > Texte permet d'obtenir automatiquement du texte pur venant d'un texte formatté HTML; à utiliser lorsque la composition du message est terminée.

Les fonctionnalités de suivi de courriel peuvent être ajoutées au message par l'utilisation de Google Analytics et / ou des boutons TrackReports dans le panneau de l'HTML; les liens dans le message sont automatiquement modifiés selon le système choisi de suivi et le code nécessaire est alors ajouté. L'ajout du code de Google Analytics ou de TrackReports doit toujours être la dernière opération effectuée sur le message - les deux méthodes modifient le code final; donc les utilisateurs doivent premièrement avoir terminés de composer le message. Le suivi du message en utilisant les deux systèmes est possibles: dans ce cas, l'utilisateur doit ajouter premièrement le suivi de Google Analytics.

Avant d'envoyer le message, il est possible (et fortement recommandé) de vérifier comment le message sera affiché dans les clients de courriel, en utilisant le bouton Prévisualisation du Courriel. La fonction de prévisualisation du courriel utilisera un prélèvement des données de l'expéditeur défini dans la section Mes données personnelles dans l'écran Paramètres des Champs.

Il est possible d'effectuer une vérification du score de pourriel sur le moment; lors de l'édition du message en utilisant le bouton à Vérifier Pourriel, la vérification du score de pourriel est basée sur le populaire moteur SpamAssassin, c'est l'un des filtres anti-pourriel le plus commun dans les serveurs de courriels. En utilisant cette fonctionnalité avancée de SendBlaster, vous éviterez à votre message d'être considéré comme un pourriel par erreur.

*La gestion des pièces jointes n'est possible qu'avec la version Pro
