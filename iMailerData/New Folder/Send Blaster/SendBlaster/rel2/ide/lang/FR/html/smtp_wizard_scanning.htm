Scansion

L’assistant fera une scansion de votre PC pour chercher les portes disponibles, essayez de vous connecter à votre serveur SMTP et d’envoyer un message de test. Cela vous demandera quelques minutes, selon le temps requis pour trouver une porte disponible.

Si vous avez un programme de sécurité (firewall / antivirus), il pourrait vous demander si vous voulez permettre SendBlaster d’accéder à Internet; si il vous le demande, répondez "oui" ou "permettre" pour donner plein accès à Internet à SendBlaster. Nous vous conseillons aussi de désactiver temporairement la scansion des mails sortantes.

Entre temps, vous pouvez visiter services SMTP.com dédiés exclusivement conçus pour les utilisateurs de SendBlaster.
