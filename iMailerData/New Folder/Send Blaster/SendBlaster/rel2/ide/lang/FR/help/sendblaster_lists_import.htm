Importer

Vous pouvez importer dans la liste courante des données provenant d'une source externe. Elles peuvent provenir de quatre sources:

  • Vous pouvez importer le contenu d'un fichier texte formaté (.csv, .txt) se trouvant sur votre PC ou sur un réseau local, en sélectionnant Importer un fichier texte . Le format "texte formaté" est un format d'échange largement utilisé par les bases de données et les feuilles de calcul: presque tous les logiciels de ce type sont capables d'exporter leurs données dans ce format.
     
  • Vous pouvez importer depuis une adresse internet en choisissant Importer depuis un site web via http://. L'importation est en tout semblable à celle du texte formaté, mis à part que le fichier pour l'importation ne se trouve ni sur le PC de l'utilisateur, ni sur un réseau local mais peut être atteint sur internet par le protocole http. Les données à importer peuvent être produites dynamiquement par un script (asp ou php par exemple). Cette méthode vous permettra d'importer le contenu d'une base de données en ligne disposant d'un script d'exportation. Le lien du site sera indiqué comme source des données.
     
  • Vous pouvez importer le contenu du presse-papiers de Windows (après l'avoir copié les donnés d'un autre logiciel) en sélectionnant l'option Importer depuis le Presse-Papiers .
     
  • Vous pouvez importer le carnet d'adresses de Outlook Express (aka Livre d'Adresses de Windows) en sélectionnant Importer depuis Outlook Express.

Ces données importées d'une source externe peuvent servir pour une des opérations suivantes :

  • vous pouvez les insérer dans la liste sélectionnée, en choisissant Ajouter à la liste et inscrire les nouvelles adresses .
     
  • vous pouvez les éliminer de la liste sélectionnée en choisissant Eliminer les adresses de la liste. Si les adresses contenues dans la liste importée se trouvent dans la liste, elles seront éliminées.
     
  • Vous pouvez désabonner les adresses de la liste (sans les effacer, c-à-d. en les conservant). Sélectionnez Désinscrire les nouvelles adresses. L'opération est similaire à la précédente, mais les adresses sont simplement désabonnées et non plus éliminées. De cette façon, vous conservez les données mais tout en évitant d'utiliser les adresses de courriels dans les opérations d'envoi de courriels.

Après avoir sélectionnée la liste, la source des données et l'opération, vous pouvez cliquer le bouton Importer. Une procédure guidée vous demandera d'indiquer les éléments nécessaires pour compléter l'importation:

  • le Parcours du fichier à importer , ou bien l'URL du fichier à importer si l'importation a lieu au départ du web (pour l'importation du Presse-Papiers et du carnet d'adresses de Outlook Express, il n'est pas nécessaire que vous spécifiez le parcours);
     
  • le Format de vos données (sauf pour Outlook Express): en choisissant Délimité, vous devrez également spécifier le Champs servant à Délimiter et si la Première ligne contient le nom du champs. En sélectionnant le format Libre , seules les adresses électroniques, reconnues automatiquement, seront importées, et vous ne devrez donc pas indiquer le format des données;
     
  • S'il s'agit d'une véritable importation (c-à-d. l'ajout de données par l'opération Ajouter à la liste et inscrire les nouvelles adresses), deux options supplémentaires sont disponibles. Elles indiquent au logiciel comment il doit se comporter lorsqu'une adresse contenue dans les données importées est déjà présente dans la liste: vous pouvez choisir de Ne pas ajouter les doublons et adresses déjà présentes dans la liste de distribution (ces adresses seront ignorées, dans le cas contraire l'adresse est ajoutée et crée un doublon). Si vous choisissez Inscrire les adresses qui figurent dans la liste, les adresses présentes dans la liste importée et qui ne sont pas encore inscrites seront abonnées à la liste de distribution sélectionnée (consulter Gérer les listes). Dans ce cas, les adresses ne sont pas importées mais si elles figurent déjà dans la liste, elles sont abonnées à cette liste.
     
  • L'étape suivante vous permet d'indiquer des correspondances entre les champs importés et les champs du logiciel SendBlaster. Enfoncer le bouton Suivant démarre l'importation.

Les Captures sont particulièrement très utiles lorsque vous importez: si vous importez avec succès un fichier, prenez une capture de votre configuration afin de sauvegarder les paramètres que vous utilisez; vous serez capables de les réutiliser plus tard à l'aide de quelques clics.

Attention: indiquer correctement quel est le champ de la source extérieure qui contient l'adresse électronique est fondamental car:

  1. tous les enregistrements externes qui ne contiennent pas d'adresse électronique formellement valable dans le champ que vous avez indiqué sont systématiquement ignorés. Si vous n'indiquez pas le bon champ, aucun enregistrement ne contiendra d'adresse valable et toutes vos données seront ignorées.
  2. la comparaison entre la source extérieure et les données déjà présentes dans le logiciel, pour vérifier si une adresse est déjà dans la liste, se fait sur la base de l'adresse électronique trouvée dans le champ spécifié.
