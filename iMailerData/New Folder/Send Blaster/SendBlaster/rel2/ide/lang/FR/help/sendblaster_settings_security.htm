Sécurité

Vous pouvez protéger l'accès au logiciel avec un mot de passe *.

  • Pour choisir un nouveau mot de passe (au cas où l'accès au logiciel n'avait pas déjà été protégé), saisir un mot de passe dans le champ Nouveau mot de passe et le saisir de nouveau dans le champ Confirmer votre nouveau mot de passe , et enfin choisir Enregistrer.
     
  • Pour modifier un mot de passe, mettre votre mot de passe actuel dans le champ Ancien mot de passe , et insérer le nouveau mot de passe dans le champ Nouveau mot de passe, répéter l'opération pour le champ Confirmer votre nouveau mot de passe , et enfin choisir Enregistrer.
     
  • Pour éliminer un mot de passe (et rendre l'accès au logiciel possible pour tous), saisir le mot de passe actuel dans l'espace Ancien mot de passe , ne rien indiquer dans l'espace Nouveau mot de passe ni dans l'espace Confirmer votre nouveau mot de passe , et enfin choisissez Enregistrer.

Les changements auront effet au prochain démarrage du logiciel.

Attention: Si vous optez pour protéger l'accès au logiciel, conservez avec soin votre mot de passe ! Si vous deviez le perdre, il n'existe aucun système pour le récupérer et l'accès au logiciel vous sera impossible. Vous devrez alors le désinstaller, éliminer à la main les fichiers résiduels (et tous les sous-répertoires) dans le répertoire d'installation (des raisons de sécurité empêchent l'élimination automatique de la base de données ), et le réinstaller. Dans ce cas, toutes vos données seront perdues!

*La protection par mot de passe n'est disponible qu'avec l'édition Pro
