Champs et interface

Ce menu vous permet de définir quelques options.

Donner un nom aux champs : pour chaque adresse, SendBlaster permet de donner un nom à quinze champs (le champ Courriel est indiqué en rouge, car quelque soit son nom, c'est le champ qui sera utilisé pour l'envoi des messages et ne peut contenir que l'adresse électronique de votre contact). Pour renommer les autres champs, saisissez le nom et choisissez Enregistrer.

N.B.: Si vous modifiez le nom des champs, les étiquettes disponibles pour la composition du messages seront également modifiées. Vous deviez donc revoir le texte de votre message qui aura alors été modifié.

Mes données personnelles: Dans cette section, vous pouvez ajouter vos détails personnels qui seront utilisées pour les caractéristiques de fusion lors du prévisionnement d'un message de courriel dans l'écran de composition, cette caractéristique vous permettra d'avoir une idée plus précise du message de courriel qui sera envoyé.
