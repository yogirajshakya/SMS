Historique

Cette fenêtre vous permet de voir la liste de tous les messages envoyés. Si une opération d'envoi n'a pas été complété (par exemple si votre serveur SMTP ne vous permet pas d'envoyer plus qu'un certain nombre de courriels en un court laps de temps) il est possible de le résumer du point d'interruption, d'envoyer le message à tous les destinataires non inclus dans l'opération d'envoi originelle.

Chaque opération d'envoi de courriels est notée avec la date d'envoi, le nom de la liste d'envoi du message et le sujet.

Cliquez sur le bouton Aperçu de courriel (ou sélectionner la commande correspondante avec le clic droit de la souris) pour montrer le message envoyé pour l'entrée sélectionnée.

La Liste d'Adresse vous permet de voir une liste de toutes les adresses auxquelles le message a été envoyé. Toutes les adresses à lesquelles le message a été envoyé avec succés sont montrées en vert, alors que les adresses montrées en rouge sont celles ayant eu un échec à la livraison.

Le bouton Journal montre le journal de l'envoi, qui est le dialogue qui a eu lieu avec le serveur SMTP (ou avec les serveurs de destination en cas d'envoi direct).

Dans le cas où une ou plusieurs adresses n'ont pas été incluses dans l'opération d'envoi, la dernière colonne de la liste (celle des entrées incomplètes) montrera le bouton . En cliquant dessus, cela vous permettra d'achever l'opération d'envoi en utilisant les options en cours pour l'envoi, comme spécifié dans l'onglet Envoyer. Le message sera alors envoyé à tous les destinataires ne l'ayant pas encore reçu (les adresses en rouge dans la fenêtre de la liste d'Adresses).

Les éléments de l'historique peuvent être supprimés en les sélectionnant (utilisez la touche Ctrl du clavier pour sélectionner plusieurs éléments ou utilisez Maj pour sélectionner un groupe) et en cliquant sur le bouton Supprimer les éléments.

Les adresses délivrées et non délivrées peuvent être exportées dans un fichier texte en utilisant les boutons Exportation délivrée et Exportation non délivrée. Exporter des adresses non délivrés peut également être utile lorsque vous souhaitez purger les listes de mauvaises adresses: vous pouvez ensuite importer le fichier en sélectionnant supprimer ou à l'aide des options de désabonnement.

     

Quand est-ce qu'il est utile d'utiliser les fonctionnalités de l'Historique ?

L'achèvement d'une opération d'envoi d'un message par les fonctionnalités de l'Historique est une excellente façon de contourner les restrictions d'un serveur SMTP qui ne permettraient pas autrement l'envoi de plusieurs messages en un court laps de temps. Les restrictions du serveur SMTP pourraient être telles que, en dépit de paramètres dans les Paramètres Avancés de l'onglet Envoyer, cela ne permettra pas l'achèvement d'une opération d'envoi en une seule étape. Par exemple, dans une situation où vous avez un serveur qui ne permettra pas d'envoyer plus de 50 messages par heure et une liste de distribution avec 100 adresses, seule la première moitié ira à travers au cours de la première tentative. Après une heure d'attente l'opération peut être complétée à partir de la fenêtre de l'Historique.

L'exportation d'adresses non délivrées (en rouge) pourrait également être utile lorsque vous souhaitez purger les listes de mauvaises adresses.
