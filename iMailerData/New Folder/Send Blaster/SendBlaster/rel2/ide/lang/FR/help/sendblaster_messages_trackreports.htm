TrackReports

Ce menu vous permet de naviguer facilement le site Web de TrackReports sans ouvrir une fenêtre de navigateur externe. La barre de navigation est masquée par défaut, mais vous pouvez la rendre visible en cliquant sur la barre Naviguer; les boutons pour les options familières Précédent, Suivant, Accueil, Actualiser et Arrêter apparaîtront alors.

TrackReports est un service en ligne qui vous permet de suivre vos campagnes: combien de personnes ont ouvert votre lettre d'information, combien d'entre eux ont effectivement cliqué sur des liens, et ainsi de suite. Il affiche les données recueillies à l'aide de tableaux détaillés. Afin de commencer le suivi de votre envoi, il suffit d'utiliser le bouton de TrackReports dans panneau d'édition de message de SendBlaster:

  1. Composez votre message et insérer des liens
  2. Une fois que vous avez terminé, vous connecter à l'Internet (si vous n'êtes pas déjà connecté) et cliquez sur le bouton TrackReports
  3. Entrez votre compte TrackReports (vous devez vous inscrire si vous ne possédez pas déjà un compte)
  4. Entrez un nouveau et unique titre pour votre envoi ou choissisez un modèle existant (s'il vous plaît voir le site Web de TrackReports si vous avez besoin d'aide)

Tous les liens dans le message seront automatiquement mis à jour afin qu'ils comprennent l'encodage nécessaire TrackReports et le système de redirection. Une "image de suivi" est également ajoutée au message. Comme le code HTML d'origine est modifié, l'ajout de code TrackReports doit toujours se faire seulement après que la rédaction du message soit terminée (en d'autres termes, cela doit être la dernière chose que vous faites lorsque vous composez votre message)*.

Vous devez être en ligne afin d'ajouter le code TrackReports à votre message. Car cette opération nécessite l'accès à l'API TrackReports en ligne.

TrackReports est actuellement en phase bêta, et donc, au moment d'écrire ces lignes, c'est un service proposé gratuitement aux utilisateurs SendBlaster. S'il vous plaît soyez conscient que, bien que SendBlaster a une pleine intégration avec le service, TrackReports ne fait pas partie de SendBlaster (le logiciel) en soi et pourrait être proposé comme un service d'abonnement payant à l'avenir. Il est actuellement offert «tel quel» sans aucune garantie.

Avertissement: TrackReports utilise un mécanisme de redirection (un peu comme un raccourcisseur URL). Si le serveur web TrackReports tombait, tous les liens de suivis seront inaccessibles (vos destinataires ne seront alors pas en mesure d'ouvrir vos liens). Nous vous suggérons de ne pas utiliser TrackReports ou d'autres systèmes de redirection (raccourcisseurs url, etc) pour des communications critiques.

* Sauf si vous voulez que quelques liens ne soient pas suivis: dans ce cas, vous devez ajouter les liens de non-suivis après que le code TrackReports a été ajouté au message. Si vous voulez utiliser Google Analytics et TrackReports, ajouter en premier lieu Google Analytics et TrackReports en dernier (car il utilise un système de redirection).
