Exporter

SendBlaster permet l'exportation au format .csv des adresses d'une liste* de diffusion. Sélectionner la liste à exporter et choisir parmi les options possible d'exportation:

  1. Exporter toutes les adresses : toutes les adresses de la liste sélectionnée seront exportées.
     
  2. N'exporter que les adresses des abonnés : dans la liste sélectionnée, seules les adresses inscrites seront exportées (Dans l'onglet Gérer les listes, elles sont identifiées par le symbole ).
     
  3. N'exporter que les adresses des non abonnés : dans la liste sélectionnée, seules les adresses des non-abonnés seront exportées (elles sont identifiées par le symbole du menu Gérer les listes ).
     
  4. N'exporter que les adresses actives : n'exportera que les adresses cochées. En utilisation les options avancées de filtrage disponibles dans la fenêtre Gérer les listes , cette méthode vous permettra de définir des exportations précises et sélectives.

En cliquant sur le bouton Exporter une fenetre s'ouvrira où vous définirez le nom du fichier pour l'exportation; l'option Champs délimités par  devra également être spécifiée (c'est le caractère de séparation de chaque colonne) et vous serez également capable de spécifier si la Première ligne contient le nom des champs . Vous pouvez enfin sélectionner les champs pour l'exportation et cliquer sur Suivant pour démarrer l'exportation.

*Les fonctions d'exportation ne sont disponibles que dans la version Pro
