Google Analytics

Ce panneau vous permet de naviguer aisément dans le site Web de Google Analytics sans ouvrir la moindre fenêtre de navigateur externe. La barre de navigation est masqué par défaut, mais vous pouvez la rendre visible en cliquant sur la barre Naviguer; les boutons familiers Précédent, Suivant, Accueil, Recharger et Arrêter vont alors apparaître.

Google Analytics est solution d'analyze Web la plus répandue, il vous donne un riche éclairage du trafic de votre site Web et l'efficacité de votre marketing. Si vous utilisez déjà Google Analytics sur votre site, vous serez également en mesure de suivre facilement vos campagnes de courriels: utilisez simplement le bouton Google Analytics dans le panneau d'édition de message de SendBlaster:

  1. Composez votre message et insérer les liens
  2. Une fois fait, cliquez sur le bouton Google Analytics
  3. Entrez les détails de votre campagne (svp voir le site Web de Google Analytics si vous avez besoin d'aide)

Tous les liens dans le message seront automatiquement mis à jour afin d'inclure le code nécessaire de Google Analytics. Puisque le code HTML original est modifié, rajouter le code de Google Analytics doit toujours se produire après que l'édition du message soit terminée (en d'autres mots, cela doit être la dernière chose à faire lorsque vous composez votre message)*.

Rappelez-vous SVP que Google Analytics doit déjà être installé sur votre site Web (ou sur le site Web où les destinaires se trouvent rédirigés). Vous avez besoin d'être familier avec le service afin d'être en mesure de comprendre l'information de suivi fournie.

*Sauf lorsque vous voulez que les liens ne soient pas suivis: dans ce cas, vous avez besoin de rajouter les liens non suivis après que le code de Google Analytics ait été ajouté au message. De plus, si vous voulez utiliser aussi bien Google Analytics que TrackReports, ajouter le code de Google Analytics premièrement et ensuite celui de TrackReports (car il utiliser un système de redirection).
