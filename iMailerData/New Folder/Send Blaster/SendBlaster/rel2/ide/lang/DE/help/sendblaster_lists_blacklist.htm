Sperrliste

Eine Sperrliste ist ein Filter, der verhindert, dass bestimmte Email-Adressen oder ganze Domänen Emails erhalten, unabhängig davon, ob ihr Zustand angemeldet/abgemeldet ist.

Wenn Sie eine Nachricht an eine Liste versenden, überprüft SendBlaster vor dem eigentlichen Versand, ob die Empfängeradresse oder Domäne in der Sperrliste enthalten ist: Ist dies der Fall, wird der Empfänger übersprungen (die Meldung "<Adresse>: ist in der Sperrliste" wird im Log-Bereich angezeigt).

Sendblaster unterstützt zwei verschiedene Sperrlisten:

  • Adressen-Sperrliste: Geben Sie hier die Email-Adressen ein, denen Sie niemals eine Nachricht zukommen lassen möchten.
     
  • Domänen-Sperrliste*: Statt einer einzelnen Adresse können Sie auch ganze Domänen angeben (eine Domäne ist jener Teil einer Email-Adresse, der nach dem "@"-Zeichen folgt: Achten Sie bitte darauf, nicht "www." oder andere Third-Level-Domänennamen einzufügen). Wenn eine Domäne gesperrt ist, wird niemals eine Nachricht an eine Email-Adressen dieser Domäne geschickt werden.

Sie können Einträge als freien Text vornehmen. Jede Zeile darf jedoch nur eine Adresse / Domäne enthalten (Sonderzeichen sind nicht zulässig: Sie dürfen keine sogenannten Wildcards / Platzhalter verwenden). Sie können Daten über die Zwischenablage austauschen, um Sperrlisten mit anderen Programmen zu bearbeiten oder klicken Sie den Mit Notepad editieren-Button, um eine Sperrliste mit dem in Windows integrierten Texteditor zu bearbeiten (die Ausführung der Software wird dabei unterbrochen bis Sie das Bearbeiten beenden und den Editor schließen).
