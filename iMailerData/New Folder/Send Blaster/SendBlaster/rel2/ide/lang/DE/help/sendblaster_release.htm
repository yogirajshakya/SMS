Was ist neu in SendBlaster 2

  • Die verbesserte und neu gestaltete Benutzeroberfläche
    von SendBlaster ist nun noch einfacher zu handhaben. Die Benutzer können zwischen der unveränderten klassischen und der neu entworfenen Programmansicht wählen. Diese ist nicht nur visuell ansprechender gestaltet, sondern unterstützt nun auch große sowie kleine Icons und verfügt nun über eine Werkzeugleiste, welche auch horizontale Bildschirmabstände speichern kann. Alle Bedienfelder/Programmfenster können nun mit dem Maus-Rad nach unten oder oben gescrollt werden; die Listenauswahl kann jetzt über alle Bedienfelder/Programmfenster synchronisiert werden und wird alphabetisch sortiert; der Benutzeroberfläche wurden ebenfalls viele kleinere Verbesserungen hinzugefügt.
     
  • Geplanter Mailingversand
    Ab jetzt ist es auch möglich, ein Mailing zu erstellen und dieses zu einem späteren Zeitpunkt über SendBlaster 2 automatisch versenden zu lassen. Außerdem können Mailings nun als "geparkt" gespeichert werden, so dass die Nutzer diese später auch selbst manuell starten können. SendBlaster kann nun automatisch beim Start von Windows im minimierten Zeitplanmodus/Terminmodus ausgeführt werden.
     
  • E-mail Tracking/ Analyse
    SendBlaster 2 integriert die Nutzung von Google Analytics, dem kostenlosen Mailmonitoring, sowie von TrackReport, unserem erweiterterten online EMail-Tracking- und Analyseservice.
     
  • Neue, effektive Datenfilter
    Es wurde ein kraftvoller Filter hinzugefügt, welcher Benutzern die Feineinstellung eines Mailings ermöglicht. Diese erfolgt über die Auswahl zutreffender, oder auszuschließender Bedingungen. Er unterstützt unterschiedlichste Optionen und hochkomplexe Anwendungen, wodurch Sie eine SQL-Abfrage erstellen können, ohne den Code selbst schreiben zu müssen. Duplikate können jetzt automatisch, wie auch schon in SendBlaster 1, herausgefiltert werden, oder vom Benutzer mit Hilfe einer neuen, effektiven Suchmaschine selbst gemanagt werden.
     
  • Erweiterte Versandoptionen
    Benutzer können jetzt eigene, spezifische Email-Header hinzufügen, Lesebestätigungen anfordern, Prioritäten festlegen, definieren von kundenspezifischen To: Kopfzeilen welche die Kombination aus mehreren Tags unterstützen, aktivieren und deaktivieren des automatischen Filters von doppelten Email-Adressen beim Versand von Nachrichten.
     
  • Weitere Bearbeitungsmöglichkeiten von Nachrichten
    Viele neue Buttons im Bearbeitungsfenster ermöglichen dem Nutzer das leichte Hinzufügen und Entfernen von Links in der Nachricht, das Hinzufügen von speziellen Links für Google Analytics und TrackReport, sowie das Übernehmen kombinierter Tags in die Betreffs- und Kopfzeilen einer Nachricht. Jetzt können auch Nachrichten, welche auf reinem Text basieren (als Alternative zu HTML-Nachrichten), nur durch das Auswählen einer Option ganz leicht erstellt werden.
     
  • Überprüfen von Spam
    Es ist nun auch möglich, schon während der Erstellung einer Nachricht eine Spam-Bewertung durchzuführen; der Prüfmechanismus des Spam Scores basiert auf der erfolgreichen Anwendung "SpamAssassin", welche vor allem für ihren Antispamfilter für Server bekannt ist.
     
  • Vorschau mit kombinierten Tags
    Die Vorschau unterstützt nun auch mehrfach kombinierte Tags. Dadurch erhält der Anwender eine genauere Vorstellung darüber, wie sich der zu versendende Inhalt darstellt.
     
  • Bessere Geschwindigkeit
    SendBlaster 2 unterstützt nun bis zu 50 Verbindungen - diese können die Geschwindigkeit der Mailings deutlich erhöhen, sollten Sie einen schnellen SMTP Server verwenden. Außerdem bietet SendBlaster jetzt eine Option, welche den Abbruch von Mailings verhindert, sollte die Internetverbindung abbrechen.
     
  • Anpassbare Anmelde- und Bounceback Formate
    Anwender können nun einfach An- und Abmeldungen an Ihre Bedürfnisse anpassen. Nicht zustellbare E-Mails können dank vielseitig anpassbarer Bounce-Back Filter automatisch verarbeitet werden.
     
  • SMTP und POP3 Profile
    Das Nutzen unterschiedlicher SMTP und POP3 Einstellungen ist nun einfacher als je zuvor - die Accounteinstellungen können nun in verschiedenen Profilen gespeichert werden, so dass alle SMTP und/oder POP3 Einstellungen mit nur ein paar Klicks geändert werden können. Das ist besonders hilfreich, wenn Nachrichten im Namen anderer versendet werden.
     
  • Snapshots
    Speichern Sie alle Einstellungen der vorhandenen Benutzeroberflächen mit nur einem Klick, um Sie später wieder nutzen zu können - es ist spielend einfach, einen Snapshot zu erstellen! Mit den eingeführten Snapshots wird das Konzept der "Profile" auf ein neues Level erweitert - fast alle Einstellungen können als Snapshot gespeichert werden.
     
  • Unterstützung externer Plugins
    Nutzer können kostenlose Plugins (kleine Programme, welche spezielle Aufgaben, wie z.B. das Importieren verschiedener Email Programme beinhalten) von unserer Webseite herunterladen. Es werden in Zukunft weitere hinzukommen. Die Plugin API ist öffentlich und Partnerfirmen sind bereits damit beschäftigt, weitere nützliche Plugins zu erstellen.
     
  • Neue, einfacher zu bearbeitende Templates
    SendBlaster 2 beinhaltet mehr als 100 neue professionell gestaltete Vorlagen, welche auf flexiblen Layouts basieren. Diese können ganz einfach angepasst werden.
     
  • Einfacherer Datenzugang
    Der direkte Zugriff auf die Daten in der Datenbank aufgrund von Datensicherungen ist nun wesentlich einfacher. Die Dateien sind nun im Dokumentenordner von Windows gespeichert.    .
     
  • Keine Administrationsrechte erforderlich
    SendBlaster 2 kann von den Standardnutzern unter Windows ausgeführt werden
     
  • Bereit für Windows 7
    Sendblaster 2 ist auf dem aktuellen technischen Stand und zukunftssicher dank Windows 7 Kompatibilität.
     
  • Und vieles mehr!
    SendBlaster 2 beinhaltet viele kleine zusätzliche Verbesserungen, welche dem Nutzer die Anwendung verbessern und noch mehr Freude bereiten.
