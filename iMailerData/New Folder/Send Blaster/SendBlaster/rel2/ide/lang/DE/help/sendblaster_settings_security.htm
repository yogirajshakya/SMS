Sicherheit

Die Software kann durch ein Passwort geschützt werden*.

  • Um ein neues Passwort zu setzen (falls der Zugang zur Software vorher nicht geschützt war) geben Sie es in das Feld Neues Passwort ein und wiederholen Sie es in dem Feld Neues Passwort bestätigen, schließlich klicken Sie auf den Speichern-Button.
     
  • Um ein bestehendes Passwort zu ändern, geben Sie das aktuelle Passwort im Feld Altes Passwort ein, geben Sie dann Ihr neues Passwort im Feld Neues Passwort ein und wiederholen Sie es im Feld Neues Passwort bestätigen, schließlich klicken Sie auf den Speichern-Button.
     
  • Um ein Passwort zu löschen (um die Software wieder für jeden zugänglich zu machen) geben Sie das aktuelle Passwort in das Feld für das alte Passwort ein, lassen Sie dann das Feld Neues Passwort und Passwort bestätigen frei, klicken Sie schließlich auf den Speichern-Button.

Die neuen Einstellungen gelten ab dem nächsten Start der Anwendung.

Achtung: Falls Sie sich dazu entschließen, einen Zugangsschutz einzurichten, bewahren Sie Ihr Passwort gut auf! Es gibt keine Funktion die den Zugang zu einem verlorenen Passwort ermöglicht und Sie werden keinen Zugang mehr zur Software haben. Um wieder Zugriff auf die Software zu erlangen, müssen Sie sie deinstallieren, manuell alle Dateien und Verzeichnisse im Installationspfad entfernen (aus Sicherheitsgründen werden die Datenbankdateien nicht deinstalliert), und schließlich die Software neu installieren. In diesem Fall gehen all Ihre Daten verloren!

* Der Passwortschutz ist nur in der Pro-Version verfügbar.
