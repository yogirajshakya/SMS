Erweiterte Einstellungen

In diesem Fenster können Sie einige der erweiterten Programmeinstellungen vornehmen.

Oberfläche: :

  • Erscheinungsbild: erlaubt es einzustellen, ob das Bedienfeld am Rand mit einer Explorerleiste (optinal mit kleinen Icons in platzsparender Reihenfolge), oder einer Werkzeugleiste angezeigt werden soll.
     
  • TrackReport Link im Bedienfeld anzeigen: ermöglicht es, den TrackReport Link im Hauptbedienfeld anzuzeigen oder auszublenden; falls Sie nicht vorhaben, TrackReport zu nutzen, können Sie diesen Link ausblenden, um Platz in der Leiste zu sparen.
     
  • Google Analytics Link im Bedienfeld anzeigen: erlaubt es, den Google Analytics Link im Hauptbedienfeld anzuzeigen oder auszublenden; falls Sie nicht vorhaben, Google Analytics zu nutzen, können Sie diesen Link ausblenden, um Platz in der Leiste zu sparen.
     
  • Tipps beim Programmstart anzeigen: Lassen Sie die Programmtippsanzeige beim Start von SendBlaster so lange aktiviert, bis Sie mit der Software vertraut sind.
     
  • Synchronisieren der bedienfeldübergreifenden Listenauswahl: diese leistungsstarke Option ermöglicht Ihnen das Arbeiten an allen bedienfeldübergreifenden Listen, bis Sie diese ändern. Falls Sie zum Beispiel eine eine Liste im "Senden" Bildschirm auswählen, und dann zum Listenmanagement gehen, in der Reihenfolge in der Sie neue Adressen hinzufügen, werden die Listen, welche Sie bereits im "Senden" Bildschirm ausgewählt haben, wird automatisch als ausgewählt im Listenmanagement angezeigt. Der Nachteil an dieser Option ist die Verlangsamung des Programms bei sehr langen Listen oder langsamen PCs: Falls Sie eine unerwartet lange Wartezeit in der Antwortoberfläche bemerken, deaktivieren Sie diese Funktion.
     
  • Nutzen Sie den Sprache Button, um die Sprache der Oberfläche von SendBlaster zu ändern. Die ausgewählte Sprache wird beim nächsten Start von SendBlaster ausgeführt.

Nachrichten:

  • Voreingestellter Zeichensatz für neue Nachrichten: Diese Option ermöglicht es Ihnen, einzustellen, welcher Zeichensatz/ welche Zeichencodierung bei der Erstellung neuer Nachrichten verwendet werden soll.

Anmeldungen per Email:

  • Diese Sektion erlaubt es Ihnen, die Betreffzeilen zu spezifizieren, welche bei Anmeldungen oder Abmeldungen per Email verwendet werden sollen. Sie können außerdem festlegen, welche Nachrichten diesen Betreffzeilen exakt entsprechen müssen, in der Reihenfolge, in welcher diese als Anmeldung oder Abmeldung behandelt werden sollen bzw. welche speziellen Wörter nur beinhalten soll. Um die Funktion auch in älteren Versionen gewährleisten zu können, funktionieren die Betreffzeilen "Anmeldung" und "Abmeldung" ebenfalls (in der Reihenfolge, in der Sie die Betreffzeilen hier angeben).

Nutzerspezifische Formate für unzustellbare E-Mails:

  • Durch Klick auf den Hinzufügen oder Editieren von Bounce-Back-Formaten Button, können Sie Bounce-Back-Formate hinzufügen oder entfernen. BounceBack Email Formate wurden ursprünglich erstellt um von Menschen gelesen zu werden. Aus diesem Grund gibt es kein einheitliches Standardformat. SendBlaster ist aufgrund der integrierten Datenbank in der Lage, die meisten Formate für unzustellbare Emails zu bearbeiten. Die vorhandenen Formate können durch individuelle BounceBack-Formate der Sendblaster Datenbank hinzugefügt werden, so dass SendBlaster diese Formate ebenfalls erkennen kann.

Zeitplanung:

  • Durch den Zeitplanmodus kann SendBlaster minimiert in der Schnellstartleiste gestartet werden soll, wenn Windows startet; über diesen Weg können dann zeitgesteuert die gewünschten Mailingaktionen ausgeführt werden.

Senden:

  • Diese Einstellung ermöglicht es Emails an mehrfach vorhandene Einträge in Ihrer Empfängerliste mehrfach zu senden. Wir empfehlen diese Option nicht zu nutzen, es sei denn es gibt einen speziellen Grund, dass einige Empfänger mehrfach benachrichtigt werden.

Plugins:

  • Diese Sektion erlaubt es Ihnen die gerade installierten SendBlaster Plugins (ohne diese Deinstallieren zu müssen, welches Sie mit Hilfe der Systemsteuerung oder dem jeweiligen Plugin Deinstaller durchführen können) zu aktivieren oder zu deaktivieren. Sie können außerdem weitere SendBlaster Plugins downloaden, in dem Sie den Weitere Plugins Button klicken.

Datenbank:

  • Nach dem Anklicken des Datenbank Buttons öffnet sich der Windows-Dateiordner, in dem die Benutzerdaten von Sendblaster gespeichert werden; Dies benötigen Sie evtl. für die Wartung, zum Speichern /Backup oder andere Datenbankarbeiten.

Versionsinfo und Lizenz:

  • Hier finden Sie Informationen zur Softwareversion und Lizenz. Es können hier keine Konfigurationen vorgenommen werden. (Sollten Sie ggf. Ihren Lizenzschlüssel ändern müssen, können Sie dies während des Willkommensbildschirms beim Start von Sendblaster tun.)
