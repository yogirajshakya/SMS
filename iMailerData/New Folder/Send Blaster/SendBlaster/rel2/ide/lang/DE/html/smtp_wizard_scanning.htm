Überprüfen

Der Assistent wird nun Ihren PC auf verfügbare Ports überprüfen, versuchen eine Verbindung herzustellen und eine Test-Nachricht zu versenden. Dies kann einige Minuten dauern, je nachdem wie schnell ein verfügbarer Port gefunden wird.

Wenn Sie eine Sicherheitssoftware verwenden (z.B. Firewall /Antivirensoftware), könnten Sie gefragt werden, ob Sie SendBlaster den Zugriff auf das Internet gestatten möchten. Wenn Sie gefragt werden, antworten Sie bitte mit "Ja" oder "Erlauben" um SendBlaster den vollen Zugriff auf das Internet zu gewährleisten. Wir empfehlen weiterhin, dass Sie Ihre Antivirus-Anwendung so konfigurieren, dass ausgehende Emails zeitweise nicht untersucht werden.

Wenn Sie mögen, können Sie in der Zwischenzeit einen Blick auf die SMTP.com Dienste werfen, welche exklusiv für SendBlaster-Anwender entwickelt wurden.
