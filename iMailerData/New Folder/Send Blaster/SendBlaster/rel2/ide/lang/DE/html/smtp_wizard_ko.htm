Verbindung zum Server fehlgeschlagen

Leider war das Verbinden mit dem Server und das Senden einer Test-Nachricht nicht erfolgreich. Dies kann mehrere Gründe haben:

  1. Alle Ports wurden durch eine Firewall auf Ihrem PC oder dem Internet Gateway (z.B. Router) blockiert
  2. Ein Antiviren-Programm hat den ausgehenden Verkehr blockiert
  3. Ein temporärer Netzwerk- oder Serverfehler ist auch möglich. Deshalb empfehlen wir, dass Sie Ihre Internetverbindung überprüfen und den Test in ein paar Minuten wiederholen.

Wenn das Problem weiter besteht, konfigurieren Sie bitte Ihre Firewall- und Antiviren-Anwendungen so, dass SendBlaster der Zugang zum Internet und das Senden von Emails gestattet wird (fragen Sie evtl. Ihren Netzwerkadministrator nach Unterstützung) und stellen Sie sicher, dass Ihr Internet-Anbieter nicht entsprechende Funktionen verbietet oder Ports blockiert. Weitere Informationen können Sie hier finden: Netzwerkprobleme lösen. Nachdem das Problem gelöst wurde, starten Sie den Assistenten bitte erneut indem Sie auf den "SMTP.com Assistenten"-Button auf der "Senden" Oberfläche klicken.
