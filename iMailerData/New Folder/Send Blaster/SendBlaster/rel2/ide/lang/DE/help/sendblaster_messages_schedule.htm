Zeitplanung

SendBlaster ermöglicht es den Versand von Nachrichten genau zu planen, so dass Mailings zu einer vorgegebenen Zeit abgeschickt werden können. Mailings können nach der Fertigstellung auch erst einmal "geparkt" werden. Der Versand wird somit nicht automatisch gestartet, sondern vom Benutzer zu einem späteren Zeitpunkt manuell ausgeführt.

Achtung: Der geplante Versand der Mailings wird nur ausgeführt, wenn sich SendBlaster im "Zeitplanmodus" befindet. - soll hei§en, wenn sich das Programm minimiert in der Systemablage befindet. Es werden keine Mailings versendet, so lange sich das Programm im "normalen" Modus befindet. Auch dann nicht, wenn der Versandtermin bereits fällig wurde.. Klicken Sie auf den Button Zeitplanmodus ausführen wenn Sie möchten, dass SendBlaster in die Systemablage minimiert wird und bereits geplante Mailings trotzdem verschickt werden. (Sie können durch Klicken auf das Icon in der Systemablage in den normalen Programmmodus zurückkehren.)

Das Zeitplanfenster zeigt eine Werkzeugleiste und eine Übersicht der vorgesehenen Postversand-Aufgaben.

Jede geplante Versandaktion zeigt, ob die vorgesehene Aktion aktiv ist oder nicht, Ausführungsdatum und -zeit, Titel, die Zielliste und den Betreff, der für den Versand geplanten Nachricht. Die Größe der Spalten kann angepasst werden.

Die geplanten Versandaktionen können durch einfaches Klicken auf die Überschriften der Spalten sortiert werden.

Oberhalb der Adressliste finden Sie die Werkzeugleiste. Diese Befehle erreichen Sie auch über das Kontextmenü, welches angezeigt wird, sobald Sie mit der rechten Maustaste auf eine der Listen klicken.

Sie können gegenwärtige Versandaktionen planen (Button Neu), den Termin der geplanten Versandaktion ändern (Datum ändern), eine Vorschau der geplanten Nachricht ansehen (E-mail Vorschau), die geplante Versandaktion starten (Jetzt senden), oder eine oder mehrere Versandaktionen löschen (Löschen).
