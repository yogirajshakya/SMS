TrackReport

Mit dieser Navigation bewegen Sie sich ganz einfach durch die TrackReport Website, ohne ein zusätzliches Browserfenster zu öffnen. Das Navigationsmenü wird standardmäßig nicht angezeigt, Sie können es sichtbar machen, in dem Sie auf das Navigationsmenü klicken; die bekannten Buttons Zurück, Vorwärts, Startseite, Neu laden und Stopp werden nun angezeigt.

TrackReport ist ein Online-Dienst, welcher Ihnen das Verfolgen von Kampagnen ermöglicht (zum Bsp.: von wievielen Empfängern wurde Ihr Newsletter geöffnet, auf welchen Ihrer Links wurde von wem wie oft geklickt, etc.?) Die gesammelten Daten werden in detaillierten Charts angezeigt. Um mit der Verfolgung Ihres Mailings zu beginnen, klicken Sie einfach auf den TrackReport Button in der Steuerung des Nachrichteneditors von SendBlaster:

  1. Erstellen Sie Ihre Nachricht und fügen Sie die gewünschten Links ein.
  2. Sobald Sie die Erstellung Ihrer Nachricht abgeschlossen haben, (sollten Sie noch nicht Online sein) verbinden Sie Ihren PC mit dem Internet und klicken dann auf den TrackReport Button.
  3. Melden Sie sich in Ihrem TrackReport-Konto an (Falls Sie noch kein Konto angemeldet haben, müssen Sie sich zuerst registrieren).
  4. Geben Sie einen neuen, eindeutigen Namen für Ihr Mailing ein oder wählen Sie aus den bereits vorhandenen aus (Sollten Sie Hilfe benötigen, besuchen Sie bitte die TrackReport Webseite).

Alle in der Nachricht enthaltenen Links werden automatisch aktualisiert, so dass sie die notwendigen TrackReport Codes und Nachverfolgungssysteme beinhalten; ein "Tracking-Bild" wird der Nachricht ebenfalls hinzugefügt. Da der ursprüngliche HTML-Code dabei modifiziert wird, sollte das Hinzufügen des TrackReport Codes erst nach Abschluss der Nachrichtenerstellung erfolgen. (Mit anderen Worten: das Einsetzen des TrackReport Codes ist der letzte Schritt bei der Fertigstellung von Nachrichten)*.

Da diese Funktion den Zugang zur TrackReport Online API voraussetzt, müssen Sie, um Ihrer Nachricht den TrackReport Code erfolgreich hinzufügen zu können, mit dem Internet verbunden sein.

Da sich TrackReport zur Zeit in der Beta-Phase befindet, wird dieser Service den Kunden von SendBlaster momentan kostenlos zur Verfügung gestellt. Bitte beachten Sie, dass SendBlaster diesen Service zwar vollständig integriert hat, TrackReports selbst aber kein Teil der SendBlaster (Software) ist und in Zukunft eventuell auch als bezahlter Service angeboten werden kann; das momentane Angebot ist ohne Garantie.

Warnhinweis: TrackReport verwendet einen Weiterleitungsmechanismus (ähnlich dem der Kurz-URL-Dienste); sollte der Server auf dem TrackReport gehostet ist nicht verfügbar sein, sind auch die getrackten Links nicht mehr erreichbar. (Ihre Empfänger können die Links in dieser Zeit somit nicht öffnen.) Wir empfehlen Ihnen deshalb, TrackReport oder andere Nachverfolgungssysteme (Kurz-URL-Dienste), bei zeitkritischer Kommunikation, nicht anzuwenden.

*Es sei denn, Sie möchten einige Links einfügen, welche nicht verfolgt werden sollen. In diesem Fall fügen Sie dann die jeweiligen Links erst nach dem Hinzufügen des TrackReport Codes ein. Falls Sie beide Systeme nutzen möchten, Google Analytics als auch TrackReport, so fügen Sie Ihrer Nachricht zuerst den Google Analytics Code und dann den TrackReport Code hinzu.
