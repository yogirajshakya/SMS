Zeitweises aktivieren oder deaktivieren eines oder mehrerer Kontakte erlaubt es Ihnen eine Nachricht an eine Untergruppe der Adressen einer Verteilerliste zu senden. Inden Sie diese Funktion zusammen mit den Sortier- und Gruppierfunktionen verwenden, sind Sie in der Lage leistungsstarke Filterfunktionen zu erzeugen.\r\rUm schnell Filter zu erzeugen, deaktivieren Sie alle Adressen, gruppieren Sie sie und aktivieren anschliessend nur die, die Sie interessieren.
Eine der leistungsfähigen Funktionen in SendBlaster ist das mehrfache Gruppieren. Schauen Sie sich diese einzigartige Funktion an und experimentieren Sie: Ziehen Sie mehrere Spaltenköpfe in den Bereich oberhalb der Liste und sehen Sie was passiert.
Sie müssen sich niemals über Dupletten ärgern: SendBlaster kümmert sich automatisch darum. Wenn die selbe Adresse öfter als einmal in einer Versendeaktion vorkommt, wird die Nachricht nur an eine dieser Adressen gesendet - Duplikate werden beim Senden automatisch herausgefiltert.
Wenn Sie die kompletten Daten Ihrer Kunden erfassen, und nicht nur ihre Email-Adressen, können Sie Marker verwenden um personalisierte Nachrichten zu versenden: "Lieber Klaus" klingt viel besser als ein neutrales "Lieber Benutzer". Sie brauchen nichts zu aktivieren oder zu konfigurieren. Wählen Sie nur einen passenden Marker aus der Liste im "Nachrichten erstellen"-Formular und es wird an der aktuellen Position im Text eingefügt.
Die eingebaute "Vorschau"-Funktion benutzt Ihren Standard-Email-Client. Wenn Sie eine höhere Flexibilität benötigen, erzeugen Sie eine "Test"-Liste die einige Ihrer eigenen Email-Adressen enthält (füllen Sie auch ein paar der anderen Felder aus) und senden Sie Ihre Nachricht an diese Testliste bevor Sie sie an Ihre Kontakte senden. Über diesen Weg sind Sie in der Lage Ihre Nachricht in verschiedenen Email-Anwendungen, inklusive Online-Diensten wenn Sie Ihre GMail/Hotmail (oder ähnliche) Adressen hinzufügen, zu überprüfen.
Sie können von Zeit zu Zeit Ihre Listen bereinigen indem Sie zurückgekommene Nachrichten herunterladen und automatisch abarbeiten lassen.\r\rDie betroffenen Adressen werden aus der Liste gelöscht, was die nächsten Sendevorgänge beschleunigt, da keine Zeit für ungültige Adressen verschwendet wird.
Sie sind nicht daran gebunden Ihre Email-Vorlagen zu verwenden. Jede .html oder .eml-Datei kann ganz einfach, in nur zwei Schritten, zu einer Vorlage werden: Zuerst importieren und dann als neue Vorlage speichern.
Denken Sie bei Vorlagen nicht nur an grafische Layouts: Wenn Sie einen Text erstellen, den Sie in Zukunft wiederverwenden möchten, speichern Sie ihn als Vorlage. Dadurch müssen Sie den Text beim nächsten mal nicht mehr komplett neu schreiben.
Sie können Ihre Kontakte aus jeder Quelle importieren, die in der Lage ist, Daten als formatierten Text zu exportieren.\rFormatierter Text (auch bekannt als .csv - engl. für "Komma separierte Werte") ist ein sehr populäres Format zum Datenaustausch zwischen verschiedenen Programmen.\r\rZ.B. können Sie Ihre Kontakte aus Outlook oder aus einer Excel-Datei exportieren um sie anschliessend in SendBlaster zu importieren.
Der einfachste Weg einen Anmeldelink Ihrer Webseite hinzuzufügen ist, einen Link auf eine Ihrer Email-Adressen zu platzieren, die Sie für die Verwaltung Ihrer Verteilerlisten nutzen und "Anmelden" als Betreff verwenden. Dies erfordert keinen serverseitigen Code auf Ihrer Webseite.\rWenn Ihre Webseite PHP oder ASP unterstützt, können Sie auch ein komplettes Anmeldeformular bereitstellen, welches bereits alle gewünschten Daten Ihrer Kontakte sammelt. Sie können kostenlose Beispielformulare und Skripte von unserer Webseiter herunterladen.
Es ist besser eine separate Email-Adresse zum Verwalten Ihrer Listen zu verwenden (z.B. "anmeldung@meineseite.de"), anstatt der Adresse über die Sie versenden (z.B. "nachrichten@meineseite.de"). Dadurch werden Antworten Ihrer Nutzer nicht mit An- und Abmeldungen vermischt, was es Ihnen erlaubt sie schneller zu bearbeiten.
Wenn Sie SendBlaster dazu konfigurieren Anmeldungen von der selben Email-Adresse herunterzuladen die Sie auch in anderen Email-Anwendungen verwenden, aktivieren Sie die "Eine Kopie der Nachrichten auf dem Server belassen"-Checkbox unter "Listen verwalten" und "Rückläufer verwalten".\rDadurch ist es möglich die Nachrichten zusätzlich auch auf Ihren Email-Client herunterzuladen.
Zurückliegenden Sendeaktionen werden im Abschnitt "Verlauf" aufgezeichnet. Sie können eine bereits gesendete Nachricht wiederverwenden, auch wenn sie nicht gespeichert wurde. Wählen Sie dazu nur "Öffnen / Öffne Email aus dem Verlauf" im "Nachrichten erstellen"-Formular.
Anstatt Anhänge Ihrer Nachricht hinzuzufügen, können Sie die Dateien auch auf Ihrer Webseite ablegen und Links zu diesen Dateien in der Nachricht platzieren: Die Zustellung der Nachricht wird viel schneller erfolgen und die Empfänger werden sich über kleinere Nachrichten freuen.
Um die Zustellrate zu erhöhen, vermeiden Sie es, Bilder zu Ihrer Nachricht hinzuzufügen: Moderne Antispamfilter verdächtigen Nachrichten die viele Grafiken beinhalten und könnten sie in den Junk-Mail Ordner verschieben oder sie sogar löschen bevor sie den Empfänger erreichen.
Verschiedene Email-Programme stellen HTML verschieden dar. Versuchen Sie komplexe Formatierungen zu vermeiden: Halten Sie sie einfach sodass die Nachrichten auf dem meisten Platformen lesbar sein werden (und ihr Layout konsistent, wenn nicht identisch, in den meisten EMail-Clients ist).
Um die Zustellrate zu erhöhen und eine falsche Spameinstufung von Antispamfiltern zu vermeiden, fügen Sie immer einen "reinen Text" Teil in Ihre Nachricht ein. SendBlaster kann diesen Text automatisch aus HTML erzeugen wenn Sie auf den "Text aus HTML"-Button klicken. Dies sollte die letzte Aktion sein, bevor Sie eine Nachricht versenden.
Wenn Sie einen Newsletter versenden, fügen Sie immer einen Abmeldelink am Ende der Nachricht ein: Platzieren Sie einfach einen Link auf die Email-Adresse, mit dem Betreff "Abmelden", die Sie zur Verwaltung Ihrer Liste verwenden. Wenn Sie mit Links nicht vertraut sind finden Sie im Handbuch einige Beispiele.
SendBlaster sendet jede Nachricht als eigene Email. Es wird kein "Cc" oder "Bcc" verwendet: Empfänger können nicht die Email-Adressen der anderen Empfänger sehen an die die Nachricht verschickt wurde.
Wenn Sie einen speziellen Zeichensatz benötigen (unterschiedlich vom Standard), wählen Sie ihn bitte bevor Sie beginnen Ihre Nachricht zu verfassen: Die Auswahl des "Zeichensatzes" sollte die erste Aktion sein, sodass die eingegebenen Zeichen in der Voransicht korrekt dargestellt werden.
Wenn Sie eine reine Text-Nachricht versenden möchten, die keinen HTML Teil enthält, erzeugen Sie eine neue Nachricht und bearbeiten Sie nur den "reinen Text" Teil und lassen Sie den Bereich des visuellen Editors leer: SendBlaster wird automatisch erkennen dass der HTML Teil eine leere Seite enthält und nur den reinen Text als Email versenden.\r\rBevor Sie die Nachricht senden, kontrollieren Sie immer die Vorschau um sicherzustellen, dass nicht aus Versehen etwas im HTML Teil eingegeben wurde.
SendBlasters Geschwindigkeit hängt stark von Ihrer Internet-Verbindung und der Leistungsfähigkeit des SMTP Servers ab: Wenn Sie nicht sicher sind, wie schnell Nachrichten gesendet werden können, beginnen Sie mit langsamen Einstellungen (wählen Sie ein langsames Profil im "Senden"-Formular), dann erhöhen Sie stufenweise die Geschwindigkeit (wählen Sie ein schnelleres Profil, oder erniedrigen Sie die Pausen und erhöhen die Anzahl der Verbindungen).
Anstatt nur SMTP oder direktes Senden zu verwenden, versuchen Sie eine Mischung aus den zwei Methoden: Wählen Sie SMTP als Ihre primäre Sendemethode und aktivieren Sie die "Wiederhole im Fehlerfall mit direktem Senden"-Checkbox.
Verwenden Sie die "Quote druckbar"-Kodierung im "Senden"-Formular, falls Ihre Nachricht vorwiegend aus Text besteht. Wählen Sie "Base64" nur dann, wenn sie viele Bilder oder Anhänge enthält (die Nachrichtengröße wird kleiner, aber einige Antispamfilter mögen es evtl. nicht).
Kontaktlisten können viele Felder enthalten die Sie evtl. nicht alle benötigen: Unter "Listen verwalten" benutzen Sie den "Sichtbare Felder"-Button um Felder auszublenden die Sie nicht benötigen.
Das Abschliessen einer Versendeaktion über den "Verlauf" ist eine gute Möglichkeit die Einschränkungen eines SMTP Servers zu umgehen, welche es sonst nicht erlauben würden viele Nachrichten in einer kurzen Zeitspanne zu versenden. Beginnen Sie eine Sendeaktion, dann stoppen Sie sie nach einer Weile. Sie werden sie dann später zuendeführen - SendBlaster wird dort fortfahren wo die Sendeaktion unterbrochen wurde (es werden keine Duplikate versendet).
"Importieren" bedeutet nicht nur neue Kontakte zu einer Liste hinzuzufügen. Sie können mittels externen Daten mehr leisten: Z.B. wenn Sie eine Datei haben, die "ungültige" Adressen enthält, können Sie diese Datei importieren und so diese Adressen aus der Liste entfernen.
Die Verwendung von Sperrlisten ist wichtig um schnell und sicher zu verhindern das Nachrichten an jemanden gesendet werden oder um vor ungewollten Anmeldungen zu schützen: Wenn eine Email-Adresse oder Domäne in eine Sperrliste aufgenommen wurde, wird sie niemals eine Nachricht empfangen - egal welchen Status sie in den Versendelisten führt.
SendBlaster erlaubt es Ihnen die Datenbank durch ein Passwort zu schützen. Um ein Passwort zu entfernen, geben Sie das alte Passwort einfach in das zugehörige Feld ein und lassen Sie die Felder "Neues Passwort" und "Passwort bestätigen" leer.
