Felder und Erscheinungsbild

In diesem Fenster können Sie einige programmbezogene Optionen einstellen.

Feldnamen ändern: SendBlaster ermöglicht es Ihnen, die Namen der 15 verfügbaren Felder für jede Adresse zu bearbeiten (das Email-Feld wird in Rot markiert, denn unabhängig von seiner Bezeichnung, ist dies das Feld, mit dem Nachrichten versendet werden. Es muss immer die Email-Adresse des Empfängers beinhalten.). Geben Sie einfach die neuen Namen in den zugehörigen Boxen ein und klicken Sie auf den Speichern-Button.

Achtung: Durch das Ändern der Feldnamen werden auch die Tags für die Erstellung von Nachrichten geändert. Deshalb müssen eventuell die Inhalte von Nachrichten ebenfalls überarbeitet werden.

Erscheinungsbild: Die Bereichsübersicht an der Seite kann zwischen einer Ansicht im Windows XP-Stil und einer im Outlook-Stil umgeschaltet werden.

Standardzeichensatz für neue Nachrichten: Diese Option ermöglicht es, den Standardzeichensatz für das Erstellen einer Nachricht zu verändern.
