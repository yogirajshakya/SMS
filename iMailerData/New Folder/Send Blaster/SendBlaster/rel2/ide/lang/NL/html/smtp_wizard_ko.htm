Er kon geen verbinding gemaakt worden met de server

Het spijt ons, we konden geen verbinding met de server maken en een test bericht versturen. Dit kan meerdere oorzaken heeft:

  1. Alle poorten zijn geblokkerd door een firewall die aanstaat op uw PC of op uw internet gateway
  2. Een antivirus software blokkeerd uitgaand verkeer
  3. Een tijdelijk netwerk of server probleem is ook mogelijk, daarom raden we aan dat u uw internet verbinding over een paar minuten weer controleerd

Als het probleem zich voor blijft doen, moet u uw firewall en antivirus zo instellen dat SendBlaster verbinding met het internet mag maken en e-mail mag versturen (het zou kunnen dat u hulp moet vragen van uw netwerk administrator) en zorg ervoor dat uw Internet Service Provider deze functie niet blokkeerd of bepaalde poorten blokkeerd. U kan extra informatie hier vinden: Netwerk problemen opsporen. Nadat het probleem is opgelost, kan u de Wizard opnieuw opstarten door op "SMTP.com Wizard" te klikken in het 'Verzend' formulier van SendBlaster.
