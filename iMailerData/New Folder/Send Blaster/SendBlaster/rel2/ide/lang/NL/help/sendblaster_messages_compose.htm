Bericht samenstellen

Het bericht samenstellen scherm laat u een bericht maken dat verstuurd kan worden.

Het samenstellen wordt gedaan net als in elke andere standaard e-mail client: typ het Onderwerp, voeg als het nodig is Bijlages toe*, typ de tekst voor in het bericht. Als de tekst een speciaal alfabet nodig heeft (niet op latijns gebaseerd), moet u eerst de Encoding instellen (standaard encoding kan aangepast worden in het Velden en uiterlijk scherm).

Waarschuwing: De encoding voor de berichten moet ingsteld worden voordat u het bericht typt om de goede resultaten te krijgen. Als u de encoding niet van te voren instelt kan het zijn dat de tekst niet te lezen is.

Er zijn twee gebieden op de pagina:

  • In het hoofdgebied, waar de lange toolbar staat, kan u tekst invoeren om een HTML bericht te maken. Ook kan u afbeeldingen en links toevoegen, en u kan de HTML code controleren (en aanpassen) wanneer u maar wilt door op de Laat code zien knop te klikken in de bovenste balk .
     
  • In het Gewone tekst gebied, kan u een bericht typen in het gewone tekst formaat. Sommige ontvanger kunnen misschien geen HTML bekijken, daarom is het belangrijk om een alternatief aan te bieden, zonder afbeeldingen en/of links. Gewone tekst kan automatisch gegenereerd worden van HTML tekst door op de Tekst van HTML knop te kliken als u klaar bent met het typen van uw bericht.

Om aangepaste berichten te versturen (in plaats van algemene berichten), kan u in het onderwerp en gewone bericht (zowel in de gewone tekst als in de HTML) Labels invoegen. Elk label is eigenlijk een referentie naar een veld: als het bericht verzonden wordt elk label vervangen met een waarde die uitgelezen wordt uit de info van de ontvanger in het specifieke veld. Bijvoorbeeld, als de lijst een Bedrijfsnaam veld en een Adres veld heeft (veld namen kunnen aangepast worden), door tekst te typen in het bericht zoals hieronder:

Messrs. #Bedrijfsnaam#
#adress#

de ontvangers zullen een aangepast bericht ontvangen zoals(voorbeeld):

Messrs. The Coca Cola Company
Atlanta, GA

Aangezien veldnamen aangespast kunnen worden, is het ook mogelijk om 'fix' labels te gebruiken (labels die onafhankelijk zijn van de veldnaam) die naar een verld nummer verwijzen (#1# #2# enz. tot #10#). Veld nummer kunnen gevonden worden in het Velden en uiterlijk scherm. Drie speciale labels zijn er gegeben: #lijstnaam# die vervangen wordt door de naam van de lijst waar het bericht bij hoort; #managingemail# wat vervangen wordt door het e-mailadres waar de In/Uitschrijvingen heen worden gestuurd (dit laat ontvanger makkelijk hun adressen verwijderen van de lijst); en #systeemdatum# wat vervangen wordt door de huidige datum.

     

Hoe maak je een Uitschrijf link

SendBlaster kan distibutie lijsten onderhouden door berichten te downloaden van een e-mail adres dat speciaal daarvoor is, wat anders is voor elke lijst anders is (Onderhoud inschrijvingen). Daarom is het makkelijk om labels toe te voegen aan de HTML code met een uitschrijf link:

<a href="mailto:#managingemail#?Subject=Uitschrijven">Verwijder</a>

Deze HTML code kan worden toegevoegd aan het HTML bericht door op Show Code te klikken, of het kan worden toegevoegd door de Link knop te gebruiken in de formaat toolbar (in dat geval moet u de waarde mailto:#managingemail#?Subject=Uitschrijven specificeren in het link scherm en, als de link is aangemaakt, de tekst 'Verwijderen' (of iets soort gelijks) typen om de tekst die er staat te vervangen).

      

Het gewijzigde bericht kan opgeslagen worden in het .elm formaat (kan gebruikt worden door de meeste e-mail clients) door op de Opslaan knop te klikken.

Om nieuwe berichten vanaf het begin te maken (met de Nieuw of Kies template knoppen) of, gebruik de Open knop, om een eerder opgeslagen bericht te openen:

  • Importeer een .eml bestand(bijvoorbeeld gemaakt met Outlook) met Open / Open .eml bestand
  • Laad een eerder verstuurd bericht met Open / Open e-mail van Geschiedenis
  • Importeer tekst van een HTML pagina van de lokale harde schijf, samen met afbeeldingen, met Open / Importeer HTML van bestand
  • Importeer gewone tekst met Open / Importeer gewone tekst van extern bestand

De knop Tekst van HTML laat u gewone tekst van HTML maken. Het is daarom aan te raden om het te gebruiken nadat u het bericht hebt samen gesteld.

Voordat u het bericht verstuurd, is het mogelijk (en aangeraden) om te controleren hoe het bericht eruit ziet in de e-mail client, met de E-mail preview knop.

*Bijlage functies zijn beschikbaar in de Pro versie.
