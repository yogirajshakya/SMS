FAQ

Als is een bericht naar meerdere mensen verstuur kunnen ze dan elkaar's e-mailadres zien?

Nee, SendBlaster verstuurd elk bericht apart, de privacy van de klant wordt bewaard. SendBlaster gebruiken om meerdere zendingen te onderhouden is daarom handig, om te voorkomen dat ontvangers per ongeluk elkaar's e-mail adres zien, wat niet mag volgens de privacy wetten.
 

Hoe kan ik een lijst filteren?

Door de sorteer/groepeer functies te gebruiken in de Onderhoud lijsten sectie, en tijdelijk in/uitschakelen van een groep adressen laat u een bericht naar een subgroep van de adressen in de lijst sturen.

Hoe kan ik een uitschrijf link invoegen in het bericht?

SendBlaster kan zjn distributie lijsten onderhouden door berichten te downloaden van een e-mailadres dat alleen daarvoor is, die anders is voor elke lijst.(zie Onderhoud inschrijvingen). Daarom kan u door gebruik van labels makkelijk een uitschrijf link toevoegen:

<a href="mailto:#managingemail#?Subject=Uitschrijven">Uitschrijven</a>

De HTML code moet toegevoegd worden in de HTML sectie van het bericht door te klikken op Code laten zien, anders kan het als link worden toegevoegd met de Link knop in de bericht formaat balk(in dat geval moet de tekst mailto:#managingemail#?Subject=Uitschrijven ingevoerd worden in het link scherm, en als de link is gemaakt kan de tekst verandert worden naar "Uitschrijven" of iets dergelijks).

Als ik een bericht stuur naar een mailing lijst, worden sommige berichten niet afgeleverd en komen ze terug met een error.

Als u naar veel adressen verstuurd is het normaal dat niet alle berichten erdoor komen: sommige adressen kunnen inactief zijn of tijdelijk geblokkeerd, e-mail server kunnen te druk zijn, de connectie kan tijdelijk onderbroken of slecht zijn.

Als het percentage van verzonden berichten erg laag is, of er worden zelfs geen berichten verstuurd, is het nodig om naar de Verzend instellingen te kijken.

Eigenlijk, zou het ook nodig kunnen zijn om de verzend methode te veranderen (direct of via SMTP) en als uw SMTP server antispam filters gebruikt, het aantal verbindingen verlagen en de pauze tussen de berichten vergroten. De SMTP server zou meer restricties kunnen hebben die u verhinderen meer als een bepaalde hoeveelheid berichten te versturen binnen een bepaalde tijd (een server kan bijvoorbeeld zo ingesteld zijn dat er maximaal 50 berichten per uur verstuurd mogen worden); in dat geval, moet na de eerste verzending, de mailing afgemaakt worden op een later tijdstip met de Geschiedenis functies.

Als de antispam instellingen van de SMTP server niet verandert kunnen worden en mailingen alleen met erg langzame instellingen verzonden kunnen worden, kan u het proberen met direct versturen of door een `tolerantere` SMTP server te gebruiken (u kan op het internet inschrijven voor zeer efficiënte SMTP services zelfs voor een lage prijs).

Als het probleem zich voordoet terwijl u de direct versturen methode gebruikt, kan het nodig zijn om handmatig het IP adres van de DNS server in te stellen.
 

Ik ben het wachtwoord kwijt geraakt en kan niet meer bij het programma. Wat moet ik doen?

U moet weten dat het niet mogelijk is om verloren data terug te halen. Door middel van de volgende procedure kan u weer toegang krijgen tot het programma.

1. Verwijder het programma vanuit het Windows Configuratie Scherm.

2. Verwijder de installatie map handmatig (bijv. C.\Program files\SendBlaster), met alle submappen; om veiligheidsredenen, als het programma wordt verwijderd wordt de database niet verwijderd (daarom moet dit handmatig gedaan worden).

3. Installeer het programma opnieuw.
