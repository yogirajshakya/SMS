Segurança

O acesso ao software pode ser protegido por senha*.

  • Para definir uma nova senha (caso o acesso ao software não tenha sido anteriormente protegido) digite-a no campo Nova senha e digite-o novamente no campo Confirmar senha e, finalmente, clique no botão Guardar.
     
  • Para alterar a senha, digite a senha actual no campo Senha antiga, depois digite a nova senha no campo Nova senha e digite também no campo Confirmar a nova senha. Para terminar, clique no botão Guardar.
     
  • Para apagar uma senha (tornando o software acessível a qualquer um) digite a senha actual no campo Senha antiga, deixe os campos Nova senha e Confirmar nova senha vazios e depois clique no botão Guardar.

As novas configurações ficarão disponíveis da próxima vez que o software for executado.

Aviso: Se decidir proteger o acesso ao software, cuide bem da sua senha! Não são fornecidas funcionalidades especiais para recuperar a senha e caso esta se perca você perderá o acesso ao software. Para voltar a ter acesso ao programa será necessário desinstalá-lo, remover manualmente todos os ficheiros e directorias do local de instalação (por motivos de segurança os ficheiros da base de dados não são desinstalados) e, finalmente, instalar o software novamente. Neste caso, todos os seus dados serão perdidos!

*A protecção de senha está disponível na versão profissional
