FAQ

Quando envio uma mensagem para vários destinatários eles conseguem ver os endereços dos outros destinatários?

Não, o SendBlaster envia cada mensagem separadamente: a privacidade dos destinatários é preservada. Utilizar o SendBlaster gerir vários envios é útil para prevenir que os endereços dos outros destinatários sejam acidentalmente enviados, o que viola a regras de privacidade.
 

Como filtro uma lista?

Se utilizar as funções de ordenamento/agrupamento da secção Gerir listas e activar/desactivar temporariamente um grupo de endereços, irá permitir enviar uma mensagem a um subgrupo dos endereços da lista actual. Pode também usar o botão Filtro.

Como posso inserir no corpo da mensagem um link para cancelar a subscrição?

O SendBlaster pode gerir as suas listas de distribuição descarregando as mensagens a partir de um endereço de e-mail, diferente para cada lista (ver Gerir subscrições). Portanto, ao utilizar tags, é simples inserir um link de cancelamento no código de HTML:

<a href="mailto:#managingemail#?Subject=Unsubscribe">Unsubscribe</a>

O código HTML será inserido na secção HTML da mensagem, e poderá revê-lo/editá-lo clicando em Exibir código.

Quando eu envio uma mensagem para uma lista de distribuição, algumas das mensagens não são entregues e são devolvidas com erro.

Ao enviar para vários de endereços é normal que nem todas as mensagens consigam chegar ao destino: alguns dos endereços poderão ter ficado inactivos ou temporariamente indisponíveis, servidores de e-mail sobrecarregados, a ligação estar momentaneamente interrompida ou lenda.

Se a percentagem de mensagens enviadas for muito baixa ou mesmo nenhuma mensagem for enviada, será necessário verificar as Opções de envio.

Na verdade poderá ser preciso alterar o modo de envio (directo ou através de SMTP) e, caso o seu servidor de SMTP utilize filtros antispam, baixar o número de ligações e aumentar a pausa entre as mensagens. O servidor SMTP poderá ter restrições que impeçam o envio de mais do que um determinado número de mensagens num intervalo de tempo (por exemplo, um determinado servidor poderá impedir o envio de mais de 50 e-mails por hora); nesse caso, após o primeiro envio, a operação será concluída mais tarde através das funções do Histórico.

Se as restrições antispam do servidor SMTP não puderem ser contornadas e as operações de envio apenas puderem ser enviadas através de configurações lentas, você poderá tentar o envio directo ou utilizar um servidor de SMTP "mais tolerante" (é possível subscrever serviços SMTP na internet, mesmo por preços reduzidos).

Se o problema se mantiver com o envio directo, poderá ser necessário especificar manualmente o endereço IP do servidor de DNS.
 

Perdi a senha e não consigo mais aceder ao programa. O que posso fazer?

Infelizmente não será possível recuperar os seus dados. O procedimento seguinte apenas permitirá voltar a ter acesso ao software.

1. Desinstalar o software através do painel de controle do Windows.

2. Remova manualmente a pasta de instalação (i.e. C.\Programas\SendBlaster), incluindo todas as subpastas; por motivos de segurança, no caso do software ser desinstalado, a base de dados não será automaticamente removida (é por isso que terá de a apagar manualmente).

3. Instale o software novamente.
