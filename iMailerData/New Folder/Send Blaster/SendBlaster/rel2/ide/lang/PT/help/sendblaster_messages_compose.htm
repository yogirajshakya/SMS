Escrever mensagem

A janela de escrita de mensagens permite criar a mensagem a ser enviada para a lista de distribuição.

A escrita das mensagens no SendBlaster é semelhante à escrita de mensagens em qualquer cliente de e-mail standard: digite o Assunto da mensagem, insira os Anexos* necessários, insira o email do destinatário (apenas para efeitos e pré-visualização) e digite o texto da mensagem. Caso o texto precise de alfabetos especiais (não latinos), você terá primeiro de escolher a Codificação (a codificação padrão pode ser alterada na janela Campos e aparência).

Tags podem ser inseridas e usadas por pressionar o botão (#) ao lado dos campos Assunto e Para de forma a enviar emails com cabeçalho personalido.

Aviso: A codificação da mensagem deverá ser especificada antes de digitar a mensagem, de modo a obter resultados correctos. Se não o fizer, poderá obter texto ilegível.

A página exibe duas áreas de texto:

  • Na área principal, onde se encontra a barra de ferramentas longa, você poderá inserir texto formatado e gerar uma mensagem no estilo HTML. Poderá também inserir imagens e links, assim como verificar (e modificar) o código HTML gerado automaticamente, clicando, a qualquer momento, no botão Exibir código da barra de cima.
     
  • Na área Texto simples poderá digitar o texto da mensagem no formato de texto simples. Alguns destinatários não conseguem visualizar mensagens no formato HTML e, por isso, é importar fornecer um formato alternativo para os conteúdos, sem quaisquer imagens e/ou links. O texto simples pode ser automaticamente gerado a partir do texto formatado, clicando no botão Texto do HTML assim que terminar a digitação da mensagem.

Para enviar mensagens personalizadas (em oposição às mensagens genéricas), a linha de assunto e o corpo da mensagem (tanto na secção HTML como na secção de texto simples) você poderá inserir Tags. Cada tag é, na verdade, uma referência a um campo: quando a mensagem é enviada a tag é substituída por um valor obtido a partir do texto do respectivo campo dos dados do destinatário. Por exemplo, se a lista incluir um campo "NomeDaEmpresa" e um campo "Endereço" (os nomes dos campos podem ser alterados), ao digitar o seguinte texto no corpo da mensagem:

Para: #NomeDaEmpresa#
#endereco#

os destinatários receberão uma mensagem personalizada, como por exemplo:

Para: The Coca Cola Company
Atlanta, GA

Uma vez que os nomes dos campos podem ser alterados, também é possível utilizar tags "fixas" (tags independentes do nome do campo) que se referem ao número do campo (#1#, #2#, etc... até #10#). Os números de campo podem ser encontrados na janela Campos e aparência. São disponibilizadas três tags especiais: #listname# que é substituída pelo nome da lista para onde a mensagem é enviada; #managingemail# substituída pelo endereço de e-mail para o qual as subscrições/cancelamentos são enviados (isto irá permitir que os destinatários consigam remover o seu endereço da lista); e #systemdate# que é substituída pela data actual, formatada de acordo com as configurações do Windows.

     

Como criar um link de cancelamento de subscrição

O SendBlaster pode gerir listas de distribuição descarregando mensagens de um endereço de e-mail dedicado, diferente para cada lista (Gerir subscrições). Portanto, é simples incluir no código HTML um link de cancelamento de subscrição, através de tags:

<a href="mailto:#managingemail#?Subject=Unsubscribe">Remove</a>

Este código HTML pode ser incluído na mensagem de HTML, clicando em Inserir link de cancelamento de subscrição através do botão Link da barra de formatação que também permite especificar o texto associado à remoção.

      

A mensagem editada poder ser guardada no formato .eml (compatível com a maioria dos clientes de e-mail), clicando no botão Guardar.

Você pode criar uma mensagem nova a partir do zero (através dos botões Novo ou Escolher modelo) ou utilizando o botão Abrir para abrir uma mensagem guardada anteriormente:

  • Importar um ficheiro .eml (por exemplo, criado no Outlook) através de Abrir / Abrir ficheiro .eml
  • Abrir uma mensagem enviada anteriormente através de Abrir / Abrir e-mail a partir do histórico
  • Importar texto formatado a partir de uma página HTML do disco local, incluindo imagens, através de Abrir / Importar HTML de um ficheiro externo
  • Importar texto simples através de Abrir / Importar texto simples a partir de um ficheiro externo

O botão Texto do HTML permite extrair texto simples a partir de texto com formatação HTML. Aconselhamos que apenas execute esta função depois de terminar de escrever a mensagem.

As funcionalidades de monitoramento de emails podem ser adicionadas ao email por usar os botões Google Analytics e/ou o TrackReports no páinel HTML; os links na mensagem serão automaticamente modificados de acordo com o sistema de monitoramento escolhido, bem como o códig necessário é inserido na mensagem. A adição de código do Google Analytics ou do TrackReports deverá ser sempre a última operação a ser efectuada na mensagem - ambos os métodos modificam o código, e portanto os utilizadores deverão primeiro terminar a edição do email. O monitoramento de mensagens usando ambos os sistemas é possível; neste caso, o utilizador deverá primeiro adicionar o sistema Google Analytics.

Antes de enviar a mensagem, é possível (e altamente recomendado) que verifique como a mensagem irá ser exibida nos clientes de email, através do botão Pré-visualizar Email. Na pré-visualização, será usada a informação de exemplo do remetente definida na secção de Meus dados pessoais no ecrã Campos.

É possível realizar um teste de nível spam enquanto edita a mensagem através do botão Spam check. Este teste de nível de spam é baseado no SpamAssassin engine, um dos filtros de spam mais comuns nos servidores de email. Através da utilização desta funcionalidade avançada do SendBlaster, irá evitar que a sua mensagem seja considerada spam por engano.

*As funcionalidades de anexação estão disponíveis na edição profissional.
