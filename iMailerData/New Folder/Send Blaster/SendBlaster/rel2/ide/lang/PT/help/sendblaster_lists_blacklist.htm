Lista negra

A lista negra é um filtro que impede que determinados endereços de e-mail ou domínios recebam e-mails, independentemente do seu estado de subscrição/activação.

Quando você envia uma mensagem para uma determinada lista, antes de cada e-mail ser enviado o Sendblaster verifica se o endereço do destinatário ou o seu domínio está incluído na lista negra: se estive, o e-mail não é enviado (aparece uma mensagem "<endereço>: em lista negra" no painel de registo).

O Sendblaster suporta dois tipos de listas negras:

  • Lista negra de endereços: introduza aqui os endereços de e-mail para os quais não quer enviar mensagens.
     
  • Lista negra de domínios *: em vez de endereços individuais, você pode indicar domínios inteiros (um domínio de e-mail é a parte que vem depois do sinal "@" de um endereço de e-mail: cuidado para não incluir "www." ou outro tipo de nome de domínio de terceiro nível). Quando um domínio é colocado em lista negra, nenhuma mensagem será enviada para nenhum endereço pertencente a ele.

Você pode introduzir itens como texto livre; cada linha tem de conter um endereço / domínio (e nenhuns caracteres extra: não é possível utilizar caracteres joker). Você pode copiar para / colar da área de transferência, de modo a manipular listas negras com outro software; ou pressionar o botão Editar com bloco de notas para abrir uma lista negra com o editor de texto do Windows (a execução é interrompida até você terminar a edição e fechar o bloco de notas).

Note que as listas negras tem um limite para o seu tamanho, e não servem como um substituto das tarefas "próprias" de cancelamento de subscrições. Como regra geral, devem ser usadas como uma "emergência" ou apenas em tarefas especiais (i.e. quando alguém pede explicitamente que não quer receber mensagens de ninguém, nem depois da subscrição; ou quando quer assegurar que os seus concorrentes não estão a receber as suas newsletters).
