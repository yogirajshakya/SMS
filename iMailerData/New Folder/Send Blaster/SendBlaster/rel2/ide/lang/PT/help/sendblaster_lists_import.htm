Importar

Você pode importar dados a partir de uma fonte externa para a lista que se encontra seleccionada. Há quatro tipos de fontes aceites:

  • Você pode importar dados a partir de um ficheiro de texto formatado (.csv, .txt) que se encontre no seu computador ou rede local, seleccionando a opção Importar a partir de ficheiro de texto externo. O “texto formatado” é um formato de intercâmbio muito usado por software de folha de cálculo e de bases de dados: a maioria do software que se enquadra nesta categoria consegue exportar dados para este formato.
     
  • Você pode importar dados a partir de um web site escolhendo a opção Importar a partir de um site via http://. O processo é muito semelhante ao método de texto formatado, mas com a diferença de o ficheiro a ser importado não estar acessível localmente, mas sim através do protocolo HTTP. Os dados a serem importados também podem ser gerados dinamicamente por um (por exemplo ASP ou PHP). Este método permitirá importar os conteúdos de uma base de dados on-line que possua um script de exportação. O web link será especificado como sendo a fonte dos dados.
     
  • Você pode importar dados a partir da área de transferência do Windows (depois de ter colado dados a provenientes de outro software), seleccionando a opção Importar a partir da área de transferência.
     
  • Você pode importar dados a partir do livro de endereços do Outlook Express, seleccionando a opção Importar a partir do Outlook Express.

Os dados obtidos a partir de fontes externas podem ser usados para uma das seguintes operações:

  • Pode ser adicionada à lista seleccionada, escolhendo a opção Adicionar à lista e subscrever novos endereços.
     
  • Podem ser removidos da lista seleccionada, através da opção Remover os novos endereços da lista. Todos os endereços incluídos na fonte de dados serão removidos.
     
  • Podem ser cancelados da lista (sem os remover e, portanto, preservando os seus dados), escolhendo a opção Cancelar a subscrição dos novos endereços. Esta operação é semelhante à descrita acima, mas os endereços não são removidos da lista e apenas são assinalados como cancelados. Este método permite manter os dados na lista, evitando a inclusão dos mesmos endereços em futuras operações de mailing.

Depois de ter seleccionado a lista, a fonte de dados e o tipo de operação, clique no botão Importar. Aparecerá um assistente que permitirá especificar algumas opções necessárias para concluir a operação de importação:

  • Caminho do ficheiro a importar ou URL do ficheiro a importar no caso de uma importação web (a importação a partir da área de transferência ou do Outlook Express não requer nenhum caminho).
     
  • O Formato dos dados (não necessário para importar a partir do Outlook Express): se você escolher Delimitado terá também de especificar o Delimitador de campo e se A primeira linha contém nomes de campo. Se escolher o formato Livre apenas serão importados endereços de e-mail (automaticamente detectados) e não será necessário indicar o tipo de formato dos dados.
     
  • Se estiver a efectuar uma operação de importação típica (isto é, adicionando dados ou a operação Adicionar à lista e subscrever novos endereços), duas opções extra estarão disponíveis, indicando o que fazer quando um endereço externo (presente nos dados a serem importados) já se encontrar incluído na lista de destino: se escolher a opção Não adicionar endereços quando forem duplicados ou já existirem na lista, estes endereços serão ignorados (caso contrário seriam adicionados e ficariam duplicados); se escolher a opção Subscrever endereços já existentes na lista, se um endereço já se encontrar a lista de destino, mas não estiver subscrito (ver Gerir listas), os novos dados não serão importados, mas o endereço será assinalado como subscrito.
     
  • No passo seguinte terá de estabelecer a correspondência entre os campos da fonte de dados e os campos do SendBlaster e, ao clicar no botão Seguinte, a operação de importação terá início.

Os snapshots são particularmente úteis aquando da importação: se importou com sucesso um ficheiro, tire um snapshot da sua configuração de forma a poder guardar as configurações que usou; assim poderá reutilizá-las mais tarde com um par de cliques.

Aviso: a indicação correcta de qual o campo da fonte de dados externa que contém os endereços de e-mail é particularmente importante, porque:

  1. todos os registos externos que não contenham, no campo seleccionado, um endereço de e-mail válido serão ignorados. Portanto, se for indicado um campo errado, nenhum registo possuirá um endereço de e-mail válido e todos os dados externos serão ignorados.
  2. a operação de comparação entre os dados externos e os dados presentes na lista, para verificar se um endereço já se encontra na lista ou não, é efectuada comparando os endereços de e-mail presentes com campo especificado.
