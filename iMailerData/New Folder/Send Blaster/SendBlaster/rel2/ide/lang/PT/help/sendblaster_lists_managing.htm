Gerir listas

O SendBlaster pode gerir várias listas de distribuição*.

A lista actual é seleccionada através de um menu suspenso que se encontra na parte superior da janela. Ao lado, poderá encontrar 3 botões que permitem criar uma nova lista (botão Nova), renomear a lista actual (Renomear), apagá-la (Apagar) ou enviá-la para a impressora (Imprimir). Estas tarefas também poder ser efectuadas clicando no menu suspenso com o botão direito do rato.

A área restante da janela é dedicada à lista de endereços da lista de distribuição seleccionada.

Cada endereço mostra: o estado da subscrição (assinalado com os ícones e ), o estado da activação (assinalado com uma caixa de verificação), os conteúdos dos 15 campos personalizados (o primeiro é sempre o endereço de e-mail), a presença de comentários (assinalada através do ícone , clicando nele abre os comentários). A ordem das colunas pode ser alterada, bastando que as mova (arrastando o cabeçalho de coluna). As colunas também podem ser redimensionadas.

Durante uma operação de envio, os endereços assinalados como cancelados ou inactivos serão ignorados. A operação de envio irá portanto afectar apenas os endereços que estiverem subscritos e activos. Poderá detectar facilmente os endereços que não serão incluídos nos mailings, porque estes são exibidos a cinza:

  No exemplo da esquerda, a operação de envio apenas afectará o primeiro endereço, uma vez que é o único que está subscrito e activo.

A lista de endereços pode ser ordenada livremente, clicando nos cabeçalhos de coluna. Se arrastar um ou mais cabeçalhos de coluna para a área acima da lista (Arrastar um cabeçalho de coluna para esta área para agrupar os dados por essa coluna) permitirá agrupar endereços com base no conteúdo dos campos seleccionados. A caixa de verificação Total também será exibida, permitindo (quando assinalada) saber o número de endereços incluídos em cada grupo.

     

Como filtrar endereços

A activação/desactivação temporária de um ou mais itens (sem ter de os subscrever) irá permitir enviar uma mensagem para um subgrupo de endereços da lista. Ao combinar esta funcionalidade com as funcionalidades de ordenamento/agrupamento permitirá criar filtragens poderosas a serem aplicadas aos processos de mailing.

Por exemplo, se o campo "Cidade" se encontrar definido, você poderá enviar uma mensagem apenas para as pessoas que vivem numa determinada cidade, desactivando temporariamente todos os outros endereços. Se estiver definido um campo "Profissão" você poderá enviar uma mensagem apenas para determinadas categorias profissionais (também será possível combinar os dois filtros, utilizando a funcionalidade de agrupamento: os endereços serão primeiro agrupados por "Cidades" e depois por "Profissão").

O Sendblaster fornece um poderoso filtro de dados que permite aos utilizadores ajustar a sua lista por permitir seleccionar e activar ou desactivar registos que satisfazem determinadas condições. O SendBlaster suporta diferentes tipos de condições e operadores complexos para que possa especificar uma interrogação SQL poderosa sem escrever código. Este construtor de filtros pode ser acedido através do botão Filter.

O desactivar de um ou mais endereços irá permitir criar listas de exclusão (endereços para os quais nunca nenhuma mensagem será enviada, nem mesmo se o utilizador tiver subscrito a lista por vontade própria).

      

Acima da lista de endereços poderá encontrar a barra de gestão de listas. Também pode aceder a esses comendo através do menu contextual que é exibido quando se clica com o botão direito do rato na lista.

Você pode inserir novos endereços (botão Novo), modificar os endereços seleccionados (Modificar) ou apagar um ou mais endereços (Apagar). Pode seleccionar vários endereços através do teclado ou do rato: clicar num endereço enquanto se mantém a tecla Ctrl pressionada irá adicionar o endereço à selecção e clicar num endereço enquanto se mantém a tecla Shift pressionada irá adicionar um grupo de endereços.

Com os botões Recortar / Copiar / Colar você poderá apagar endereços ou copiá-los para a área de transferência (se precisar de seleccionar todos os registos, clique com o botão direito do rato em qualquer registo e depois escolha a opção Seleccionar tudo).

Os botões Subscrever endereços seleccionados e Cancelar subscrição dos endereços seleccionados irá permitir gerir as subscrições manualmente na lista seleccionada (o subscrever de uma lista irá causar a alteração da data de subscrição). Os botões Activar endereços seleccionados e Desactivar endereços seleccionados permite activar/desactivar um endereço, independentemente do seu estado de subscrição.

O botão Filtro permite aos utilizadores filtrar registos usando até 5 regras, tal como uma interrogação SQL mas sem complexidade; também permite especificar uma acção a efectuar sobre todos os contactos que satisfazem as regras: Activá-los/Desactivá-los, ou Subscrever/Cancelar Subscrição. Pode também escolher Desactivar os contactos que não satisfazem as regras, ou apenas deixá-los tal como estão.

O botão Histórico exibe a lista das mensagens anteriormente enviadas para o endereço seleccionado.

Se clicar no botão Notas aparecerá uma nova janela na qual poderá inserir texto livre associado ao endereço actual. Os endereços quem têm uma nota associada são assinalados com o ícone na última coluna (se clicar neste ícone abrirá a janela de notas).

O botão Localizar** exibirá uma janela que permite procurar texto num determinado campo ou em vários campos (incluindo notas). Se seleccionar Palavras exactas a operação de pesquisa irá apresentar todos os endereços que correspondem exactamente ao valor indicado. Caso contrário, a operação de pesquisa irá exibir todos os endereços que contêm o valor, mesmo que não corresponda exactamente ao valor indicado. Para se mover dentro da lista de resultados utilize os botões Encontrar primeira, Encontrar seguinte e Encontrar anterior da janela de pesquisa.

O botão Encontrar Duplicados irá abrir uma janela que lhe permite gerir os emails duplicados nas várias listas. Pode procurar por duplicados nas listas seleccionadas, e remover ou editar os contactos duplicados. Note que poderá ter emails duplicados nas suas listas e pode escolher se quer enviar emails para os duplicados na janela de Configurações Avançadas.

O botão Comutar vista permite alternar entre "vista normal" e a "vista de cartão". Na "vista de cartão" cada endereço será exibido num cartão separado com uma largura configurável. A navegação é feita através da barra de rolamento.

Você pode apenas querer utilizar um subconjunto dos 15 campos disponíveis; para poupar espaço no ecrã, ao clicar em Campos visíveis poderá escolher quais os campos a serem exibidos.

*A edição grátis pode gerir até 2 listas de distribuição.
**A pesquisa apenas está disponível na versão profissional
