Exportar

O SendBlaster permite exportar dados a partir de uma lista de distribuição* para um ficheiro no formato .csv. Depois de seleccionar a lista a ser exportada poderá escolher um dos seguintes métodos de exportação:

  1. Exportar todos os endereços: todos os endereços da lista serão exportados.
     
  2. Exportar apenas os endereços subscritos: apenas os endereços subscritos serão exportados da lista seleccionada (endereços com o símbolo  na janela Gerir listas).
     
  3. Exportar apenas os endereços que cancelaram a subscrição: apenas os endereços que cancelaram a subscrição serão exportados a partir da lista seleccionada (os endereços com o símbolo  na janela Gerir listas).
     
  4. Exportar apenas os endereços activos: apenas os endereços que cancelaram a subscrição serão exportados. Através das opções de filtragem que se encontram na janela Gerir listas este método permite definir exportações selectivas e específicas.

Ao clicar no botão Exportar aparece uma janela na qual se define o nome do ficheiro para o qual se vai exportar. A opção Delimitador de campo terá também de ser especificada (o carácter separador de coluna) e você poderá especificar se A primeira linha contém nomes de campos. Para terminar, você poderá seleccionar os campos a serem exportados e, depois de clicar no botão Seguinte, o processo de exportação começará.

* As funções de exportação apenas estão disponíveis na versão profissional.
