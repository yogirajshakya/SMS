Configurações Avançadas

Nesta janela pode alterar algumas opções avançadas do programa.

Interface:

  • Aparência: permite especificar se o painel lateral deverá ser mostrado no estilo de uma Barra XP (opcionalmente com Ícones Pequenos para poupar espaço), ou no estilo de uma Barra Outlook.
     
  • Mostrar link do TrackReports no painel: permite definir se mostra/esconde o link do TrackReports no painel principal; se não planeia usar o TrackReports, poderá optar por esconder este link de forma a poupar espaço na sua barra de ícones.
     
  • Mostrar link do Google Analytics no painel: permite definir se mostra/esconde o link do Google Analytics no painel principal; se não planeia usar o Google Analytics, poderá optar por esconder este link de forma a poupar espaço na sua barra de ícones.
     
  • Mostar dicas ao iniciar programa: activa a janela de dicas que é mostrada quando o SendBlaster é iniciado. É boa ideia manter esta opção activa até que fique familiarizado com o SendBlaster.
     
  • Sincronizar selecção de lista através dos painéis: esta é uma opção bastante poderosa que lhe permite continuar a trabalhar na mesma lista entre os diferentes painéis, até que decida mudar de lista. Por exemplo, se seleccionar uma lista na janela de Enviar, depois mudar para a secção de Gerir listas para adicionar um novo endereço, a lista que seleccionou na janela de Envio já estará seleccionada na secção Gerir listas. A desvantagem desta opção é que se tiver listas longas e um computador lento, esta opção poderá fazer com que o programa fique também lento: se verificar algum atraso indesejado na resposta por parte da interface, desactive esta opção.
     
  • Use o botão Idioma para alterar o idioma usado na interface do SendBlaster. O idioma seleccionado será usado da próxima vez que o SendBlaster for iniciado.

Mensagens:

  • Codificação padrão para as novas mensagens: Esta opção permite alterar a codificação de caracteres padrão usada para a composição de novas mensagens.

Subscrição de Emails:

  • Esta secção permite especificar o assunto que deverá ser usada para as mensagens de subscrição e cancelamento de subscrição. Pode também escolher se as mensagens devem corresponder exactamente com o assunto definido, a fim de serem tratadas como subscrições/cancelamentos, ou apenas incluir as palavras indicadas. Por razões de compatibilidade com versões anteriores, os assuntos "Subscrições" e "Cancelamento de Subscrições" funcionarão sempre também (além dos assuntos que definir aqui).

Formatos personalizados dos Emails devolvidos:

  • Usando o botão Editar formatos, pode adicionar ou editar formatos de emails devolvidos. Uma vez que os formatos dos emails devolvidos foram originalmente criados para serem lidos por humanos, e não existe regra, o SendBlaster consegue processar a maioria dos formatos de emails devolvidos usando a sua base de dados interna. Mas pode acontecer que encontre alguns formatos que não estão a ser reconhecidos pelo software. Se isto acontecer, pode adicioná-los manualmente à base de dados dos emails devolvidos para que SendBlaster possa processar estes novos formatos de email devolvidos também.

Agenda:

  • Esta secção permite configurar o SendBlaster para iniciar em modo de agendamento (minimizado na Área de Notificação) quando o Windows inicia; desta forma as operações de mailing agendadas podem ser automaticamente iniciadas no seu devido tempo.

Envio:

  • Esta secção permite especificar se deseja que o SendBlaster envie emails duplicados para os seus contactos quando estes existem nas listas mais do que uma vez. Normalmente não precisa enviar mais do que uma cópia de cada mensagem para os seus contactos. Assim, recomendamos que não escolha esta opção, a não ser que tenha uma razão muito específica e particular para enviar emails duplicados para contactos duplicados.

Plugins:

  • Esta scção permite que active ou desactive os Plugins do SendBlaster actualmente instalados (sem de facto os desinstalar, o que pode fazer recorrendo ao Painel de Controlo do Windows ou ao desinstalador de cada plugin). Pode também descarregar mais plugins do SendBlaster usando o botão Mais Plugins.

Base de dados:

  • Usando o botão Pasta da Base de Dados, irá abrir a pasta do Windows onde os dados do utilizador estão guardados; isto é útil para manutenção, cópias de segurança e outras operações sobre a base de dados.

Informação e licença:

  • Aqui pode ler informação sobre o lançamento do software e sobre a sua licença. Não há nada a configurar aqui (se, por alguma razão, precisa de mudar a chave da sua licença, pode fazê-lo a partir do janela inicial do SendBlaster).
