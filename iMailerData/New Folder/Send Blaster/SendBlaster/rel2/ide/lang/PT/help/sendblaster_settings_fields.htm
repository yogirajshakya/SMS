Campos e aparência

Nesta janela pode configurar algumas das opções do programa.

Definir nomes de campos: O SendBlaster permite personalizar os nomes dos 15 campos disponíveis para cada endereço (o campo E-mail é realçado com vermelho uma vez que, independentemente do seu nome, é o campo utilizado para enviar várias mensagens e deverá sempre conter o endereço de e-mail do destinatário). Digite os nomes novos nas caixas correspondentes e clique no botão Guardar.

Aviso: Depois de alterar os nomes de campo, as tags de escrita de mensagem também mudarão; por isso, o texto da mensagem também terá de ser modificado.

Os meus dados pessoais: Nesta secção pode adicionar os seus dados pessoais que serão usados para conjugar com a funcionalidade de pré-visualizar a mensagem na janela Compor Mensagem. Esta funcionalidade permite-lhe ter uma ideia mais precisa da mensagem que irá ser enviada.
