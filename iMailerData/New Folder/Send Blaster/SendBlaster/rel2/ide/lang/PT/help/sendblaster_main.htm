SendBlaster

O Sendblaster está desenhado para enviar rápida e automaticamente várias mensagens de e-mail para várias listas de distribuição. O motor de dados sofisticado e a capacidade de enviar mensagens personalizadas (“Caro Sr. André Rodrigues”, em vez da expressão anónima “Caro cliente”) permitem criar newsletters automatizadas e campanhas publicitárias. Tudo através de um software que corre no seu desktop.

Graças ao motor SmartSend é possível enviar várias mensagens em simultâneo, mesmo sem um servidor SMTP, tornando o envio de mensagens mais seguro e mais fácil do que através de um cliente de e-mail normal. Cada mensagem é enviada separadamente. A privacidade dos destinatários é protegida. Ao utilizar o SendBlaster para gerir vários envios também é útil para impedir que os endereços das outras pessoas sejam revelados.

As mensagens podem ser escritas directamente no software, com um conjunto completo de ferramentas de formatação ou importadas a partir de ficheiros externos (páginas HTML previamente guardas como mensagens .eml – por exemplo a partir do Outlook). As funções de edição e de importação permitem inserir imagens directamente no corpo da mensagem, evitando assim links para ficheiros externos (os clientes de e-mail mais recentes não as mostram ou exibe um aviso de segurança).

As listas de distribuição são altamente optimizadas: com uns poucos cliques poderá criar novas listas, modificar as listas existentes e importar endereços a partir de fontes externas. É possível importar dados de:

  • Ficheiros de texto formatado (.csv)
  • Outlook Express
  • Área de transferência
  • De uma página web

Também é possível receber mensagens de subscrição/cancelamento num endereço de e-mail dedicado, diferente para cada lista de distribuição: para subscrever uma lista terá de enviar um e-mail para este endereço de e-mail com “Subscrever” no assunto (ou “Cancelar subscrição” para ser removido). O software descarrega as mensagens e actualiza automaticamente as listas de distribuição correspondentes.

A gestão das subscrições através de e-mails e a possibilidade de importar dados a partir de uma fonte da web permitem facilmente adicionar funcionalidades de mailing list ao seu web site, permitindo que o SendBlaster trate de actualizar as listas e de enviar as mensagens para os destinatários correctos (evitando assim o envio de mensagens para pessoas que não as deviam receber).

A gestão dos e-mails das listas de distribuição é sofisticada: os endereços podem ser activados ou desactivados através de um simples clique e é possível filtrar os dados de uma forma muito avançada, baseada no conteúdo de um ou mais campos. O combinar destas duas funcionalidades permite enviar mensagens para um subgrupo muito específico de endereços de uma determinada lista (por exemplo, apenas as pessoas que vivem numa determinada área ou os contactos pertencentes a um ramo de actividade específico).

O software também reconhece as novas subscrições e, por isso, é possível enviar uma mensagem apenas para as pessoas que ainda não a tenham recebido. Os campos podem ser personalizados e existem várias opções de pesquisa. Cada endereço pode ser associado com notas do tipo “PostIt”, cujos conteúdos também são pesquisáveis.

Os primeiros passos

Como escrever uma mensagem e a enviar em poucos simples passos:

1. Escrever uma mensagem: no painel Mensagens clique em Escrever mensagem; depois escreva a mensagem (adicione imagens, caso deseje) ou importe-a a partir de um ficheiro externo (por exemplo, a partir de um e-mail anteriormente guardado no Outlook Express). Nesse momento pode ser útil explorar a utilização de marcadores que serão substituídos pelos dados reais do destinatário: Caro #name# passará a ser Caro André Rodrigues.

2. Definir uma lista de endereços: clique em Gerir listas no painel Listas e endereços. Insira os novos endereços na lista padrão ou crie uma nova lista, por exemplo através da função Importar a partir do Outlook Express; experimente as funções de agrupamento, arrastando o cabeçalho de coluna para a área especial acima do cabeçalho de coluna e activar e desactivar endereços.

3. Activar a mensagem: no painel Mensagens clique em Enviar; escolha a lista destinatários e especifique os parâmetros da conexão do servidor SMTP ou experimento o envio directo (podendo neste caso experimentar as opções de envio rápido).
