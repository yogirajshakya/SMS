Snapshots

Os snapshots são uma nova funcionalidade que permite ao utilizador guardar todas as configurações para um dado painel e reutilizá-las com um único clique - é tão fácil como tirar uma fotografia! Quer esteja a configurar as suas opções de entrega, ou a configurar a forma como um ficheiro deverá ser importado, nunca os reconfigure a partir do nada - guarde um snapshot, volte a carregá-los mais tarde, e está feito.

Os snapshots estendem o conceito de "perfis" para um novo nível: quase todos os grupos de configurações relacionadas no SendBlaster podem ser guardadas como um snapshot. Olhe para os botões Tirar um snapshot e Carregar snapshot no canto superior direito da maioria dos paineis.
