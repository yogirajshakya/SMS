Bem-vindo ao SendBlaster, o software simples e poderoso para gerir as suas mailing lists a partir do seu PC:

  • Gerir várias listas
     
  • Criar newsletters com anexos e mensagens embebidas
       Os ambientes de operação mais recentes não permitem a exibição de mensagens remotas: por isso, as imagens são embebidas no corpo da mensagem, evitando todos os problemas de leitura das mensagens
     
  • Importar páginas HTML e mensagens de e-mail criadas pelo cliente (formato .eml)
       As imagens externas referenciadas no código HTML podem ser embebidas na mensagem e todos os anexos que se encontram no ficheiro .eml são preservadas.
     
  • Enviar mensagens personalizadas
       Cada destinatário pode receber uma mensagem personalizada: as mensagens de e-mail podem conter tags que são substituídas pelos dados do destinatário
     
  • Permite o envio directo de mensagens ser servidor SMTP
       Através do modo de envio directo, se você utilizar várias conexões de internet, não precisará de alterar quaisquer configurações para se ligar ao serviço SMTP do destinatário: as mensagens serão entregues directamente aos servidores dos destinatários
     
  • Enviar muitas mensagens rapidamente
       Graças ao poderoso motor SmartSend podem ser enviadas várias mensagens em simultâneo, reduzindo drasticamente a velocidade de envio
     
  • Gerir subscrições e cancelamentos a partir de um endereço de e-mail dedicado
       Os utilizadores podem subscrever ou cancelar a sua subscrição enviando uma mensagem para um endereço de e-mail com a mensagem "Subscrever" ou "Cancelar subscrição" na linha de assunto. O SendBlaster descarrega as mensagens por POP3 e actualiza as listas
     
  • Importar do livro de endereços do Outlook Express
     
  • Importar listas a partir de ficheiros .csv e de recursos da internet
       Se o seu site já estiver equipado com um sistema de subscrições/cancelamentos e com o script de exportação, o SendBlaster pode obter dados a partir do seu site e depois actualizá-los na sua base de dados local
     
  • Fácil de integrar com qualquer site
       Ao adicionar funcionalidades de subscrição/cancelamento ao seu web site é fácil: não são precisos scripts; basta adicionar um simples link
