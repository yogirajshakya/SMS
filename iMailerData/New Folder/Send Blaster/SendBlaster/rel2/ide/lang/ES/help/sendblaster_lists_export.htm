Exportar

SendBlaster permite exportar información desde una lista de distribución* a un archivo con formato .csv. Una vez seleccionada la lista a ser exportada, usted puede seleccionar uno de los métodos de exportación disponibles:

  1. Exportar todas las direcciones: todas las direcciones de la lista serán exportadas.
     
  2. Exportar sólo direcciones suscriptas: sólo las direcciones actualmente suscriptas serán exportadas desde la lista seleccionada (direcciones marcadas con el símbolo   en la sección de Administrar listas).
     
  3. Exportar sólo direcciones desuscriptas: sólo las direcciones desuscriptas serán exportadas desde la lista seleccionada (direcciones marcadas con el símbolo   en la sección de Administrar listas).
     
  4. Exportar sólo direcciones habilitadas: sólo las direcciones habilitadas serán exportadas. Utilizando las opciones de filtros avanzadas en la sección de Administrar listas, podrá definir grupos específicos a ser exportados.

Presionando el botón Exportar, observará una ventana la cuál le permitirá establecer el nombre del archivo a exportar. El delimitador de campos (separador de columnas) deberá ser especificado, y usted también podrá especificar si la Primera fila contiene nombres de campos. Finalmente podrá seleccionar los campos a ser exportados, luego de presionar el botón Siguiente iniciará el proceso de exportación.

* Las funciones de exportación sólo se encuentran disponibles en la edición Pro.
