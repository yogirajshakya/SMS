Importación

Usted puede importar información desde fuentes de datos externas en la lista seleccionada. Estas son las cuatro fuentes de datos aceptadas:

  • Usted puede importar información desde un archivo de texto con formato (.csv, .txt), existente en su equipo local, o en la red local, seleccionando Importar desde un archivo de texto externo. El “texto formateado” es un formato de intercambio ampliamente utilizado por los productos de software de planillas de cálculo y bases de datos: la mayoría de los productos de software pertenecientes a estas categorías son capaces de exportar información en este formato.
     
  • Usted puede importar información desde un sitio web seleccionando Importar desde un sitio web vía http://. El proceso es muy similar al método de texto formateado, además del hecho de que el archivo a ser importado no reside en el equipo local, o la red local, sino que es alcanzable mediante el protocolo HTTP. La información a ser importada también puede ser generada dinámicamente mediante un script (por ejemplo ASP o PHP). Este método le permitirá importar los contenidos de una base de datos on-line que soporte un script de exportación. El vínculo web será especificado como la fuente de datos.
     
  • Usted puede importar información desde el Portapapeles de Windows (luego de haber pegado información desde otro software), seleccionando Importar desde el portapapeles.
     
  • Puede importar información dese la Libreta de Direcciones de Outlook Express (también conocida como Libreta de Direcciones de Windows), seleccionando Importar desde Outlook Express.

La información leída desde fuentes externas puede ser utilizada para una de las siguientes operaciones:

  • Puede ser agregada a la lista seleccionada eligiendo la opción Agregar a la lista y suscribir nuevas direcciones.
     
  • Puede ser removida de la lista seleccionada eligiendo la opción Eliminar direcciones nuevas de la lista. Todas las direcciones incluidas en la fuente de datos serán removidas, en caso de encontrarse presentes.
     
  • Puede ser desuscripta de la lista (sin ser eliminada, preservando de esta forma la información) eligiendo la opción Desuscribir nuevas direcciones. Esta operación es similar a la descrita en el punto anterior, pero las direcciones no son removidas de la lista, simplemente son marcadas como desuscriptas. Este método le permitirá mantener la información en la lista, al mismo tiempo que evita incluir a las direcciones en las operaciones de envío.

Luego de haber seleccionado la lista, la fuente de información, y la operación deseada, presione el botón Importar. Podrá observar una ventana de tipo asistente que le permitirá especificar algunas opciones necesarias para completar la operación de importación:

  • La Ruta de archivo a importar, o la URL de archivo a importar en caso de una importación web (en el caso de importar desde el Portapapeles o desde Outlook Express, ninguna ruta es requerida).
     
  • El Formato de archivo (no requerido para importar desde Outlook Express): si selecciona Delimitado también deberá especificar el Delimitador y si La primera fila contiene nombres de campos. Al seleccionar el formato Libre sólo las direcciones de e-mail serán importadas (detectadas automáticamente) y la especificación del formato de la información no será requerido.
     
  • Si se encuentra realizando una operación de importación típica (es decir agregando información, utilizando la opción Agregar a la lista y suscribir nuevas direcciones), deberá indicar dos opciones extras, la acción a tomar cuando las direcciones externas (presentes en la información a importar) se encuentren ya incluidas en la lista de destino: seleccionando la opción No agregar direcciones duplicadas o presentes en la lista, estas direcciones serán ignoradas (en caso contrario serán agregadas, creando direcciones duplicadas); seleccionando la opción Suscribir direcciones ya existentes en la lista, si una dirección ya existe en la lista de destino, pero no se encuentra suscripta (ver Administrar listas), la nueva información no será importada, pero las direcciones serán marcadas como suscriptas.
     
  • En el siguiente paso, finalmente deberá indicar la correspondencia entre los campos de la fuente de datos externa y los campos de SendBlaster, al presionar el botón Siguiente, la operación de importación iniciará.

Las Imágenes son particularmente útiles al importar: si usted ha importado exitosamente un archivo, puede tomar una imagen de su configuración para almacenar las configuraciones utilizadas; podrá reutilizar estas configuraciones en el futuro con sólo un par de clicks.

Advertencia: es necesario indicar correctamente el campo de la fuente de datos externa que posee las direcciones de e-mail, debido a que:

  1. todos los registros externos que no contienen, en el campo seleccionado, una dirección de e-mail con formato válido serán ignorados. Por ello, si el campo de e-mail es especificado en forma incorrecta, ningún registro incluirá una dirección de e-mail válida y toda la información externa será ignorada.
  2. la comparación entre la información externa y la información presente en la lista, para verificar si las direcciones deben ser incluidas, es realizada comparando las direcciones de e-mail presentes en el campo especificado.
