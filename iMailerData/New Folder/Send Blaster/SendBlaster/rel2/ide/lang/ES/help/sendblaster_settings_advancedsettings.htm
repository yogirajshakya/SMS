Configuración Avanzada

En esta ventana usted puede establecer opciones avanzadas del programa.

Interfaz:

  • Apariencia: permite especificar si el panel lateral debe mostrarse con un estilo de Barra de Exploración (opcionalmente con Iconos pequeños para ahorrar espacio), o estilo de Barra de Herramientas.
     
  • Mostrar acceso a TrackReports en el panel: permite mostrar/ocultar el vínculo a TrackReports en el panel principal; si usted no planea utilizar TrackReports, puede ocultar este vínculo para ahorrar espacio en la barra.
     
  • Mostrar acceso a Google Analitycs en el panel: permite mostrar/ocultar el vínculo a Google Analytics en el panel principal; si usted no planea utilizar Google Analytics, puede ocultar este vínculo para ahorrar espacio en la barra.
     
  • Mostrar Consejos al Iniciar: activa la ventana de consejos que será mostrada al iniciar SendBlaster. Es buena idea mantener esta opción activa hasta que se encuentre familiarizado con el uso de SendBlaster.
     
  • Sincronizar selección de listas en los paneles: esta es una opción poderosa, que le permite mantener la misma lista de trabajo sincronizada en todos los paneles hasta que usted la cambie. Por ejemplo, si selecciona una lista en la sección de Envío, luego ingresa a la sección de Administrar listas para agregar nuevas direcciones, la lista seleccionada en la sección de Envío, se encontrará seleccionada en la sección de Administrar listas. La desventaja radica en que si usted posee grandes listas, en una pc lenta, esta opción puede reducir la velocidad de ejecución de SendBlaster: si experimenta cualquier demora no deseada en la respuesta de la interfaz de usuario, simplemete desactive esta opción.
     
  • Utilice el botón de Idioma para cambiar el idioma de la interfaz de SendBlaster, el idioma seleccionado será utilizado la próxima vez que inicie SendBlaster.

Mensajes:

  • Codificación por defecto para nuevos mensajes: Esta opción le permite establecer la codificación por defecto a utilizar en la redacción de nuevos mensajes.

Suscripciones:

  • Esta sección le permite especificar las líneas de asunto que deberán ser utilizadas en mensajes de suscripciones y desuscripciones. Usted también puede decidir si los mensajes deberán coincidir exactamente, para ser tratados como suscripciones/cancelaciones, o simplemente necesitan incluir las palabras especificadas. Por asuntos de compatibilidad, los asuntos de mensaje, "Subscribe" y "Unsubscribe" también funcionarán (en forma adicional a los asuntos que usted defina aquí).

Formatos personalizados de devoluciones:

  • Utilizando el botón Editar formatos, usted puede agregar o editar formatos de devoluciones personalizados. Debido a que los formatos de devoluciones de e-mail han sido originalmente diseñados para ser leídos por humanos, no existe un estándar, SendBlaster puede procesar los formatos de devoluciones más comunes utilizando su base de datos de formatos interna, pero usted puede encontrar algunos formatos que no sean reconocidos por el software. Si esto sucede, usted puede agregar estos formatos manualmente a la base de datos de formatos de SendBlaster, con el objetivo de permitir a SendBlaster procesar estos nuevos formatos de devoluciones.

Programación:

  • Esta sección le permite configurar SendBlaster para que inicie en modo programación (minimizado en el área de bandeja de sistema) al iniciar Windows; de esta forma las operaciones de envío programadas serán lanzadas automáticamente en su debido momento.

Envío:

  • Esta sección le permite especificar si desea que SendBlaster envíe e-mails duplicados a sus contactos cuando estos se encuentren en las listas más de una vez. Generalmente, usted no necesita enviar más de una copia a cada dirección de e-mail, por ello recomendamos que no active esta opción, excepto en el caso en que usted posea una razón particular y específica para enviar e-mails duplicados a las direcciones duplicadas.

Plugins:

  • Esta sección le permite activar o desactivar los complementos de SendBlaster actualmente instalados (sin desinstalarlos - usted puede hacer desde el Panel de Control de Windows, utilizando el desinstalador de cada complemento). También puede descargar más complementos utilizando el botón Más Plugins.

Base de Datos:

  • Presionando el botón de Carpeta de Datos, usted podrá acceder a la carpeta de Windows en la cual se encuentra almacenada la información de usuario; esto es útil para propósitos de mantenimiento, backup, y otras operaciones de base de datos.
