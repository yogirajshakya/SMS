Programación

SendBlaster permite programar envíos de mensajes, para que puedan ser enviados a la hora especificada. Los mensajes también pueden ser "aparcados": son marcados como listos para ser enviados, pero no son enviados automáticamente - el usuario lanzará el envío de una operación aparcada manualmente más tarde.

Importante: Las operaciones de envío programadas sólo son lanzadas cuando SendBlaster se encuentra en "Modo programación" - en otras palabras, cuando se encuentra minimizado en el área de bandeja de iconos. Ninguna operación de envío, aún si la hora de programación ha ocurrido, será iniciada automáticamente si el software se encuentra en modo "normal". Presione el botón Ejecutar en modo programación cuando desee que SendBlaster sea minimizado al área de iconos y procese los envíos programados (usted puede regresar al modo normal presionando su icono en la bandeja de iconos).

La ventana de programación muestra una barra de herramientas y la lista de las operaciones de envío programadas.

Cada operación de envío programada muestra: si la operación de envío programada se encuentra activa o no, la fecha y hora en que la operación de envío iniciará, el título de la operación de envío, la lista de destino para la operación de envío y el asunto del mensaje de e-mail programado. Las columnas pueden ser redimensionadas.

Las operaciones de envío programadas pueden ser ordenadas libremente presionando las cabeceras de columnas.

En la parte superior de las direcciones, usted puede encontrar la barra de herramientas de programaciones. Puede acceder a estos comandos también mediante el menú contextual que es mostrado al realizar click derecho con el mouse sobre la lista.

Puede programar su operación de envío actual (botón Nueva), cambiar la fecha de programación para la operación de envío seleccionada (Editar fecha), realizar una vista previa del mensaje programado (Vista Previa), iniciar la operación de envío seleccionada (Enviar ahora), o eliminar una o más operaciones de envío (Eliminar).
