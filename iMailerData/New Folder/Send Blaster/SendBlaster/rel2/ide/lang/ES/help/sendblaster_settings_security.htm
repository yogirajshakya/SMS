Seguridad

El acceso al software puede ser protegido por contraseña*.

  • Para establecer una nueva contraseña (en caso de que el acceso al software no se encontrara ya protegido), escriba la contraseña en el campo Nueva contraseña y escríbala nuevamente en el campo Confirmar contraseña, finalmente presione el botón Guardar.
     
  • Para modificar una contraseña, escriba la contraseña actual en el campo Contraseña anterior, luego escriba la nueva contraseña en el campo Nueva contraseña y escríbala nuevamente en el campo Confirmar contraseña, finalmente presione el botón Guardar.
     
  • Para eliminar una contraseña (haciendo el software accesible a cualquier usuario) escriba la contraseña actual en el campo Contraseña anterior, y deje los campos Nueva contraseña y Confirmar contraseña vacios, luego presione el botón Guardar.

La nueva configuración tendrá efecto la próxima vez que inicie SendBlaster.

Advertencia: Si decide proteger el acceso al software, debe cuidar su contraseña! No existe ningún procedimiento especial para obtener acceso a la contraseña si la misma es perdida u olvidada, en cuyo caso usted no podrá acceder al software. Para poder acceder al software nuevamente, usted necesitará desinstalarlo, eliminar manualmente todos los archivos y carpetas del directorio de instalación (por razones de seguridad, los archivos de base de datos no son desinstalados), y finalmente instalar SendBlaster nuevamente. En este caso, toda su información se perderá!

*La protección por contraseña se encuentra disponible en la edición Pro
