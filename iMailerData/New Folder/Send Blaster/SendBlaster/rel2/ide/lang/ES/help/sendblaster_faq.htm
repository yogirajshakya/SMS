Preguntas Frecuentes

Al enviar un mensaje a varios destinatarios, ellos pueden ver las direcciones de los otros?

No, SendBlaster envía cada mensaje en forma separada, preservando de esta forma la privacidad de los lectores. Por ello, utilizando SendBlaster para administrar envíos múltiples usted evita que un lector pueda ver en forma accidental los detalles de las cuentas de e-mail de otros lectores, lo cual constituiría una violación a las reglas de privacidad.
 

Cómo puedo filtrar una lista?

Utilizando las funciones de orden/grupos en la sección de Administrar listas, habilitando/deshabilitando en forma temporal grupos de direcciones, usted podrá enviar mensajes a un subgrupo de direcciones de la lista actual. También puede utilizar el botón de Filtro.

Cómo puedo insertar un vínculo de desuscripción a la lista de correos en el cuerpo del mensaje?

SendBlaster puede administrar sus listas de distribución descargando mensajes desde una dirección de e-mail dedicada, diferente para cada lista (vea Administrar subscripciones). Utilizando el botón Insertar vínculo de desuscripción, durante la redacción de un mensaje, usted insertará un vínculo como el siguiente:

<a href="mailto:##nombredelista##?Subject=Remover">Remover</a>

El código HTML será insertado en la sección HTML del mensaje, usted puede editarlo presionando el botón Mostrar código.

Al enviar un mensaje a una lista de correos, algunos mensajes no son entregados y son retornados con un error.

Al enviar correos a muchas direcciones es normal que no todos los mensajes sean entregados: algunas direcciones pueden haber sido desactivadas o pueden encontrarse con problemas, los servidores de e-mail pueden encontrarse sobrecargados, la conexión puede encontrarse temporalmente interrumpida o presentar otros problemas.

Si el porcentaje de mensajes entregados es muy bajo, o aún si ningún mensaje puede ser entregado, es necesario verificar la Configuración de Envío.

En realidad, podría ser necesario cambiar el modo de envío (Directo o mediante servidor SMTP) y, si su servidor SMTP utiliza filtros antispam, disminuir la cantidad de conexiones e incrementar la pausa entre mensajes. El servidor SMTP puede poseer restricciones que prevengan el envío superior a una cierta cantidad de mensajes en un período de tiempo determinado (por ejemplo, un servidor puede no permitir enviar más de 50 mensajes por hora); en tal caso, luego del primer envío, la operación será completada más tarde utilizando las funciones del Historial.

Si las restricciones antispam del servidor SMTP no pueden ser superadas y las opciones de envío sólo son posibles mediante una configuración muy lenta, usted puede intentar el envío directo, o utilizando un servidor SMTP más tolerante (en la Internet es posible suscribirse, aún por un precio bajo, a muchos servidores SMTP eficientes).

Si el problema surge al utilizar el modo de envío directo, puede ser necesario especificar manualmente la dirección IP del servidor DNS.
 

He perdido mi contraseña y no puedo acceder al programa nuevamente. Qué puedo hacer?

Debe tener en cuenta que no será posible recuperar su información perdida. El siguiente procedimiento sólo le permitirá obtener acceso al software nuevamente.

1. Desinstale el software desde el Panel de Control de Windows.

2. Elimine manualmente la carpeta de instalación (por ejemplo C:\Archivos de Programa\SendBlaster), incluyendo todas las subcarpetas; por razones de seguridad, en caso de que el software sea desinstalado, las bases de datos no son removidas automáticamente (es por ello que usted necesita eliminarlas manualmente).

3. Instale el software nuevamente.
