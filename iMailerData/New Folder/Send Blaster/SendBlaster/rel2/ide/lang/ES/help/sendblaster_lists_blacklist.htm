Listas Negras

Una lista negra es un filtro que previene que ciertas direcciones de e-mail o dominios reciban correos, independientemente de su estado de suscripción/activación.

Cuando usted envía un mensaje a una lista determinada, antes de que cada correo sea enviado realmente, SendBlaster verifica si la dirección del destinatario o su dominio se encuentran incluidos en la lista negra: en caso afirmativo, la dirección de correo es saltada (un mensaje de "dirección bloqueada" será mostrado en el panel de Log).

SendBlaster soporta dos tipos diferentes de listas negras:

  • Lista negra de direcciones: ingrese aquí las direcciones de e-mail a las cuales no desea que los mensajes sean enviados.
     
  • Lista negra de dominios*: en lugar de direcciones únicas, usted puede especificar dominios completos (un dominio de e-mail es la parte que sigue al signo "@" en las direcciones de e-mail: tenga cuidado de no incluir "www." o cualquier otro nombre de dominio de tercer nivel). Cuando un dominio se encuentre en una lista negra, ninguna dirección perteneciente al dominio recibirá un mensaje.

Puede ingresar los ítems como texto libre; cada fila debe contener una dirección / dominio (sin caracteres extra: no es posible utilizar comodines). Puede copiar al portapapeles, o pegar desde el, para manipular las listas negras con software de terceros; o presionar el botón Editar con Block de Notas para abrir una lista negra con el editor de texto por defecto de Windows (La ejecución del programa se detiene hasta que usted finalice la edición y cierre el Block de Notas).

Por favor tenga en cuenta que las listas negras poseen un límite, y no pretenden reemplazar a las tareas propias de las desuscripciones. Como regla general, deberían ser utilizadas sólo en caso de "emergencia" o en casos especiales (por ejemplo, cuando alguien solicita explícitamente no recibir un mensaje en el futuro, o cuando usted desea asegurarse de que sus competidores no reciban newsletters suyas).

 

*Sólo disponible en la Edición Pro.
