Redigera meddelanden

Här skapar du meddelanden som skickas till alla på en adresslista.

Medelanden skrivs precis som i ett vanligt mailprogram med rubrik, meddelandetext och eventuella bilagor.

Taggar kan läggas till genom att trycka på (#) bredvid Ämne- och Till fälten för att skapa personlig meddelandens.

Varning: Om du skriver på något annat språk än svenska kan du behöva ändra fältet Kodning. Det måste i så fall göras innan du börjar skriva texten i meddelandet.

Det finns två textrutor:

  • I huvudrutan överst kan du skriva formaterad text med HTML-kod. Det går att sätta in bilder och klickbara länkar. Du kan se och ändra HTML-koden genom att klicka på Visa kod knappen längst till vänster på verktygsraden.
     
  • I Text-rutan nedanför kan du skriva in vanlig oformaterad text. En del mottagare kan kanske inte läsa mail i HTML-format utan ser då denna text i stället, så det är viktigt att den finns. För att slippa skriva in allt kan den skapas med knappen Text från HTML när du skrivit färdigt meddelandet.

SendBlaster kan också skapa personliga meil genom att man sätter in referenser till fält i adresslistan. När meddelandet sänds erätts dessa referenser med fältets innehåll. Tex om man har fälten Förnamn och Ort (fältnamnen kan ändras), kan man skriva:

Hej #Förnamn# i #Ort#

Hos mottaharen blir det då :

Hej Anders i Stockholm

Eftersom fältets namn kan ändras kan man i stället använda fältets nummer för att referera till ett fält (#1# #2#, osv till #10#). Fältens nummer hittar du under Inställningar, Fält och utseende. Det finns också speciella referenser som tex: #listname# som byts ut mot listans namn när meddelandet sänds; #managingemail# byts ut mot mailadressen som används för prenumeration/avbeställning och #systemdate# som ersätts med dagens datum med samma format som Windows använder.

     

Hur man lägger till en länk för avbeställning av prenumeration ( Unsubscribe )

SendBlaster kan hantera adresslistor genom att hämta mail från en speciell adress, olika för varje adresslista (Hantera prenumerationer). Här är ett exempel på en sådan länk som du kan lägga in i ditt meddelande:

<ahref="mailto:#managingemail#?Subject=Unsubscribe">Remove</a>

För att lägga in denna HTML-kod i meddelandet klickar du på Visa kod, eller använder knappen Länk på verktygsraden och skriv mailto:#managingemail#?Subject=Unsubscribe i rutan.

      

Meddelanden kan sparas i .eml format (används av de flesta mailprogram) genom att klicka på Spara.

För att skapa ett nytt meddelande kan man använda Välj mall eller knappen Nytt. Ett annat sätt är att Öppna ett tidigare sparat meddelande och ändra detta.

  • Import an .eml file (for instance, created with Outlook) with Open / Open .eml file
  • Load a message previously sent with Open / Open e-mail from History
  • Import a formatted text from an HTML page from the local hard disk, including images, with Open / Import HTML from file
  • Import plain text with Open / Import plain text from external file

Knappen Text från HTML används för att skapa textdelen från HTML-texten. Om mottagarens mailprogram inte kan visa HTML-format är det denna text som visas. Kom ihåg att klicka här igen för att uppdatera om du ändrat något i HTML-fönstret.

Trackingfunktioner kan läggas till i meddelandet med Google Analytics och/eller TrackReports från HTML-panelen; länkarna i medelandet modifieras automatiskt till det valda trackingsystemet, och den valda koden läggs till. Att lägga till Google Analytics eller TrackReports -kod är det sista du gör på ett meddelande - båda metoderna modifierar orginalkoden, och därför måste meddelandet redigeras klart först. Spårning av meddelandet med båda systemet är möjligt; i detta fall ska Google Analytics läggas till först.

Använd gärna Förhandsvisning för att kontrollera hur det ser ut innan du startar utskicket.

Det är möjligt att utföra spampoängskontroller medan du redigerar meddelandet med knappen Spamkontroll , och då räknas spampoängen ut baserat på SpamAssassin engine , en av de vanligaste antispam-filtren på epost-servrar. Genom att använda den här avancerade funktionen kan du undvika att ses som spam av misstag.

*Bilagor kan bara hanteras i PRO versionen.
