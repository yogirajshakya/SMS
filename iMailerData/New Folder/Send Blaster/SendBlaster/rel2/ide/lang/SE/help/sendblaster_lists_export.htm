Export

SendBlaster tillåter dig att exportera data från en adresslista* till .csv fil. När du valt adresslista som skall exporteras kan du välja en av följande metoder:

  1. Exportera alla adresser: alla adresser i listan kommer att exporteras.
     
  2. Exportera bara adresser med prenumerationer: Endast adresser med aktiva prenumeratione kommer att exporteras från adresslistan (adresser markerade med  symbolen i Hantera listor ).
     
  3. Exportera bara adresser med inaktiva prenumerationer: Bara adresser med icke aktiva prenumerationer kommer att exporteras från adresslistan (adresser markerade med  symbolen i Hantera listor ).
     
  4. Exportera bara aktiva adresser: Endast aktiva adresser kommer att exporteras. Genom att använda de avancerade filterfunktionerna som finns i Hantera listor , ger denna metod möjligheter till selektiva och specifika urval.

När du klickar på Exportera visas ett fönster där du anger namn för exportfilen. Fältavgränsare måste också anges (det tecken som avgränsar kolumnerna) och du kan också ange Första raden innehåller fältnamn. Slutligen väljer du vilka fält som skall exporteras och klickar på Nästa för att starta exporten.

* Exportfunktioner finns bara i Pro versionen (PRO Edition).
