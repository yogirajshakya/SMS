Import

Du kan importera data från en extern källa till den aktuella adresslistan. Det finns fyra olika metoder:

  • Du kan importera data från en formaterad textfil (.csv, .txt), som finns i din egen dator genom att använda Importera från extern textfil.
    “Formaterad textfil är ett format som ofta används i kalkylprogram och databaser.
     
  • Du kan importera data från en websia med Importera från websida via http://. Denna metod liknar mycket metoden med formaterad textfil, med den skillnade att indata hämtas via http. Det kan också vara data som skapats dynamiskt (tex .ASP eller .PHP). Denna metod tillåter import från vissa databaser. Ange webadressen som datakälla.
     
  • Du kan importera data från Windows Clipboard genom att klistra in (efter att du använt kopiera i något annat program), genom att välja Importera från clipboard.
     
  • Du kan importera data från Outlook Express Adressbok (även kallad Windows Adressbok), genom att välja Importera från Outlook Express.

Data som hämtas från externa källor enligt ovan kan användas till följande:

  • Adderas till en adresslista med Addera till listan och aktivera prenumeration för nya adresser.
     
  • Tas bort från en adresslista med Ta bort nya adresser från listan. Alla adresser som importerades kommer då att tas bort från listan.
     
  • Man kan avbeställa prenumeration (utan att ta bort från listan) med Avbeställ prenumeration för nya adresser. Denna operation liknar ovanstående men adresserna markeras bara som avbeställda. Med den här metoden kan du behålla adressinformationen men inte inkludera dessa i utskicket.

När du valt adresslista och datakälla klicka på knappen Importera. Ett fönster visas då där du kan göra några inställningar för att genomföra importen:

  • Importfilens sökväg, eller Importfilens URL om det gäller import från websida (import från Windows Clipboard eller Outlook Express kräver ingen sökväg).
     
  • Data Format (behövs inte vid inport från Outlook Express): om du väljed Avgränsad behöver du också anje vilket tecken som används som avgränsare i Fältavgänsare och om Första raden innehåller fältnamn. Genom att välja fritt format kommer bara mailadresser (dessa hittas automatiskt) att importeras och något dataformat behöver inte anges.
     
  • Om du utför en typisk import (som Addera till listan och aktivera prenumeration för nya adresser ), finns möjligheten att adressen redan ingår i den aktuella listan: Genom att välja Addera inte adersser som redan finns i listan, kommer de importerade adresserna att ignoreras och du undviker dubbletter. Du kan ocksåt välja Aktivera prenumeration för adresser som redan finns i listan (se Hantera adresslistor),
     
  • I nästa steg får du möjlighet att koppla fälten i importerade data och SendBlasters fält, och genom att klicka på Nästa startas importen.

Varning: Det är extra viktigt att ange rätt fält för mail-adressen därför att:

  1. alla externa rader som inte innehåller formellt riktig mailadress i detta fält kommer att ignoreras. Om felaktigt fält anges kommer alla data att ignoreras.
  2. jämförelsen mellan externa data och data i adresslistan sker genom att jämföra mailadresserna i det angivna fältet.
