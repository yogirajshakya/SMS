Menyer

Menyn är indelad i grupper som leder dig till alla programmets funktioner:

  • Meeddelanden

    Panelen Meddelanden samlar alla funktioner för  skapande, schemaläggande, sändning och spärning, inklusive historik.

    *Spårfunktioner finns bara med hjälp via TrackReports  och Google Analytics

  • Listor och adresser

    Gruppen Listor och adresser ger dig möjlighet att hantera adresslistor och adresser som tillhör varje lista, importera och exportera data*, hantera prenumerationer via e-mail.

  • Inställningar

    Gruppen Inställningar ger dig möjlighet att ändra fältnamn och utseende och säkerhetsfunktionen som gör att du kan skydda data med lösenord**. I Advancerade inställningar kan du ändra utseende plugins och många fler avancerad funktioner.

  • Detaljer

    Här visas detaljer som namn på aktuell adresslista, antal adresser och vald adress. Det visas bara om du valt utseendet Windows XP stil annars om Outlook stil är vald syns inte denna grupp men en del information visas i statuslisten i nedre kanten av fönstret.

*The exporting functions are available in the Pro edition
**Password protection is available in the Pro edition
