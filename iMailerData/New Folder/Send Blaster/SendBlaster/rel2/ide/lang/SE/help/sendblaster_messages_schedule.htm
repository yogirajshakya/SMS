Schemalägg

SendBlaster tillåter dig att schemalägga meddelanden så att de kan skickas senare. Utskick kan också "parkeras": de är redo att skickas, men kommer inte automatiskt att skickas - användaren skickar manuellt ett parkerat utskick senare.

Viktigt: Schemalagda utskick körs bara när SendBlaster är i "Schemaläge" - med andra ord, när det är minimerat till verktygsfältet. Inga utskick, även om den schemalagda tiden har passerats, kommer att köras automatiskt om mjukvaran är i "normal"-läget. Klicka på  Kör i schemaläge -knappen när du vill att SendBlaster ska minimeras till verktygsfältet och köra schemalagda utskick (du kan gå tillbaka till normalläget genom att klicka på ikonen i verktygsfältet).

Schemaskärmen visar ett verktygsfält och listan över schemalagda utskick.

Varje schemalagt utskick visar: om det schemalagda utskicket är aktivt eller inte, datum och tiden för start av utskicket, titel, destinationslisten för utskicket och ämnet på det schemalagda meddelandet. Du kan ändra storlek på kolumnerna.

De schemalagda utskicken kan sorteras genom att klicka på kolumnhuvudena.

Högst upp i adresslistan finns schemaverktygsfältet. Du kan även komma åt de här kommandona genom kontextmenyn som visas när du högerklickar med musen på listan.

Du kan schemalägga ett utskick (med knappen ny  Ny ), ändra datum för det valda utskicket (Redigera datum ), förhandsgranska det valda meddelandet (Förhandsgranskning av meddelande ), starta utskicket (Skicka nu), eller radera en eller flera utskick (Radera).
