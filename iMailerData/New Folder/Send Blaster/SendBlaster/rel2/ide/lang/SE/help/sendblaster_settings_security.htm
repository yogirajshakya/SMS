Säkerhet

Användningen av SendBlaster kan skyddas av ett lösenord*.

  • För att sätta ett nytt lösenord (om inget var satt tidigare) skriv in det Nya lösenordet och bekräfta genom att skriva det igen i fältet Bekräfta lösenord och klicka på Spara.
     
  • För att ändra ett befintligt lösenord skriv in detta i fältet Gammalt lösenord, och skriv in det Nya lösenordet och bekräfta genom att skriva det igen i fältet Bekräfta lösenord och klicka på Spara.
     
  • För att ta bort ett lösenord (och göra programmet tillgängligt för alla som använder datorn) skriver du in gamla lösenordet, och lämnar fälten Nya lösenordet och Bekräfta lösenord tomma. Klicka sedan på Spara.

Nya inställningarna gäller nästa gång programmet startas.

Varning: Om du väljer att skydda programmet med ett lösenord, se till att du kommer ihåg det! Det finns inget sätt att få tillgång till programmet om du förlorar lösenordet. Det enda sättet att lösa detta är att avinstallera programmet och radera alla data, och sedan installera programmet igen. All information som adresslistor och tidigare utskick försvinner!

*Lösenords-skydd finns bara i Pro versionen
