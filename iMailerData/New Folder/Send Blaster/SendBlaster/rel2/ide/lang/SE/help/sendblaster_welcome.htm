Välkommen till SendBlaster, det enkla och kraftfulla verktyget för att hantera mail-listor från din PC:

  • Hantera flera listor
     
  • Skapa nyhetsbrev med bilagor och bilder
       En del system tillåter inte länkar till externa bilder i mail så därför inkluderas bilderna direkt i mailet för att undvika sådana problem
     
  • Importera HTML-filer och mail-meddelanden (.eml format)
       Externa bilder i HTML-koden bäddas in i meddelandet och alla bilagor i eml-filen bevaras.
     
  • Skicka personliga meddelanden
       Varje mottagare kan få ett personligt mail med tex sitt namn.
     
  • Tillåter även att du sänder meddelanden direkt från din PC utan SMTP-server
       Att använda sändning direkt från PC kan vara särskilt användbart om du använder olika internetanslutningar, då behöver du inte ändra SMTP-inställningar. Medddelandena skickas direkt till mottagarens mailserver.
     
  • Sänd meddelanden snabbt
       Tack vare den kraftfulla SmartSend funktionen kan många meddelanden skickas samtidigt, vilket dramatiskt ökar hastigheten.
     
  • Hantera prenumerationer och avbeställningar via en speciell mailadress
       Användare kan prenumerera eller avbeställa prenumerationen genom att skicka ett mail till en viss adress med rubriken "Subscribe" eller "Unsubscribe". SendBlaster hämtar dessa mail via POP3 och uppdaterar adresslistorna
     
  • Importera från Outlook Express adressbok
     
  • Importera listor från .csv filer och websidor
       Om din webplats redan innehåller ett system för prenumerationer med export-script, kan SendBlaster direkt hämta och uppdatera den lokala databasen
     
  • Lätt att integrera med alla webplatser
       Att lägga till funktioner för att prenumerera och avbeställa prenumerationer till din webplats är enkelt. Det behövs bara en enkel länk.
