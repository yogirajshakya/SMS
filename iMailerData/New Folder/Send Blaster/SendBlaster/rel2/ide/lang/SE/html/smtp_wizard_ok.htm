Ett test meddelande skickades utan problem och ditt system är redo för automatisk konfigurering.

Guiden lyckades hitta de bästa inställningarna för din PC. Du kommer att kunna sända mail via ditt gratis testkonto hos SMTP.com som följde med SendBlaster och tillåter upp till 100 meddelanden via en eller flera adresslistor.

Notera att om du fortsätter kommer dina tidigare inställningar att ändras.
