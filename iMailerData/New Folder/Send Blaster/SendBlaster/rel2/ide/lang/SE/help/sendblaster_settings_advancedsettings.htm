Advancerade inställningar

I det här fönstret kan du ställa in några avancerade inställningar.

Gränssnitt:

  • Utseende: låter dig ställa in om sidopanelen ska visas med som ett  Explorerfält (alternativt med  Små ikoner för att spara plats) eller Verktygsfält .
     
  • Visa TrackReports-länk i panelen: låter dig visa/dölja TrackReports-länken i huvudpanelen; om du inte planerar att använda TrackReports kan du gömma den för att spara utrymme i verktygsfältet.
     
  • Visa Google Analytics-länk i panelen : låter dig visa/dölja Google Analytics-länken i huvudpanelen; om du inte tänker använda Google Analytics kan du gömma den för at spara plats på verktygsfältet.
     
  • Visa tips vid uppstart : aktiverar tipsfönstret när SendBlaster startas. De kan vara bra att ha dessa på fram tills att du blir familjär med SendBlaster.
     
  • Synkronisera val av listor över paneler: detta är ett väldigt kraftfullt alternativ, vilket låter dig arbeta med samma lista över alla paneler fram till att du ändrar den. Till exempel, om du väljer en lista på Skicka-skärmen så kan du gå till sektionen  Hantera listor för att lägga till en ny adress. Listan som du valde på skärmen Skicka kommer att vara vald i avdelningen Hantera listor . Nackdelen med detta är att om du har väldigt stora listor och en lite sämre PC så kan det här alternativet sakta ner programmet: om du märker av några fördröjningar, avaktivera det.
     
  • Använd knappen  Språk för att ändra SendBlaster's språkgränssnitt. Det valda språket kommer att användas nästa gång SendBlaster startar.

Meddelanden:

  • Standardteckenkodning för nya meddelanden : Alternativet låter dig ställa in standardteckenkodningen för nya meddelanden.

Epostprenumerationer:

  • Den här sektionen låter dig specificera ämnesraderna som ska användas för prenumerations- och uppsägningsmeddelanden. Du kan också välja om meddelandet ska matcha exakt för att behandlas eller bara innehålla vissa ord. För bakåtkompabilitetens skull kommer "Subscribe" och "Unsubscribe" alltid att fungera (plus det som du väljer här).

Anpassade format för bounce-backs:

  • Genom att använda knappen  Redigera format kan du lägga till eller redigera format för bounce-backs. Format för Bounce-backs var först designade för att kunna läsas av människan, inte av maskiner, men ändå finns det inga standarder eller regler. SendBlaster kan hantera de flesta formaten för bounce-backs med hjälp av sin interna databas, men du kanske stöter på något format som inte känns igen av mjukvaran. Om det händer, så kan du manuellt lägga till ett nytt anpassat format till SendBlaster's databas för bounce-backs.

Schemaläge:

  • Den här sektionen låter dig konfigurera SendBlaster att starta i schemaläge (minimerat till verktygsfältet) när Windows startar; då kan schemalagda utskick automatiskt starta när de ska.

Skicka:

  • Den här delen låter dig konfigurera om du vill att SendBlaster ska skicka dubbla meddelanden till kontakter som finns mer än en gång i listan. Vanligtvis behöver du inte skicka mer än en kopia till varje epost, så vi råder dig att låte den här rutan vara urkryssad om du inte har ett väldigt specifikt själ att sända dubbla meddelanden till dubbla kontakter.

Insticksprogram:

  • Den här delen låter dig aktivera eller avaktivera SendBlaster Plugins (utan att faktiskt avinstallera dem, vilket du kan göra från Windows kontrollpanel eller genom att använda insticksprogrammets avinstallerare). Du kan också ladda ner mer insticksprogram till SendBlaster genom att använda knappen  Fler tillägg .

Databas:

  • Genom att använda knappen Databasmapp öppnas mappen där datan lagras; detta är användbart för underhåll, backup och andra databasoperationer.

Information och licens:

  • Här kan du läsa information om mjukvarureleaser och din licens. Det finns inget att konfigurera här (om du av någon anledning ska ändra licensnyckel, kan du göra det från SendBlaster's startskärm).
