Testkontot har förbrukats

   Tyvärr har du nått gränsen för max antal tillåtna meddelanden hos SMTP.com för det gratiskonto som medföljde SendBlaster. Om du var nöjd med denna tjänst kan du fortsätta genom att Registrera ett konto hos SMTP.com och fortsätta använda denna SMTP-server.

Om du använder SendBlaster Free (gratisversionen), observera att uppgradering till Pro Edition inte betyder att du får ett obegränsat konto hos SMTP.com eftersom detta är en separat tilläggstjänst (vi påminner om att du kan använda SendBlaster även med din vanliga SMTP-server hos din internetleverantör).
