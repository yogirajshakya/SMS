Kunde inte ansluta till servern

Tyvärr gick det inte att ansluta till servern, och det gick inte att skicka testmeddelandet. Det kan bero på flera olika saker:

  1. Alla portar är blockerade av någon brandvägg i din PC eller Internet-anslutning
  2. Ett antivirusprogram blockerar utgående trafik.
  3. Det kan också vara ett tillfälligt fel så försök igen om några minuter

Om problemet kvarstår kontrollera din brandvägg och antivirusprogram så att SendBlaster kan ansluta till Internet för att sända email och kontrollera att din internetleverantör inte blockerar några portar. (Det är tex vanligt att internetleverantörer i Sverige blockerar port 25, så att "direkt sändning från PC" inte fungerar).

Mer information finns här: Felsökning nätverksproblem. När problemet är löst kan du köra guiden igen genom att klicka på "SMTP.com Wizard" i SendBlaster under Skicka.
