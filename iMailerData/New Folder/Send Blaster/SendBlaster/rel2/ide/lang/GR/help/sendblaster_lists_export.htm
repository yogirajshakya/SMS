Εξαγωγή

Το SendBlaster σας επιτρέπει να εξάγετε στοιχεία απο μια λίστα διανομής* σε αρχεία .csv . Αφού επιλέξετε την λίστα που θέλετε να εξάγετε, μπορείτε να επιλέξετε μια απο τις διαθέσιμες μεθόδους εξαγωγής:

  1. Εξαγωγή όλων των διευθύνσεων: θα γίνει εξαγωγή σε όλες τις διευθύνσεις της λίστας.
     
  2. Εξαγωγή μόνο των διευθύνσεων σε συνδρομή: θα γίνει εξαγωγή μόνο στις τρέχουσες διευθύνσεις σε συνδρομή απο την επιλεγμένη λίστα (οι διευθύνσεις που έχουν σημειωθεί με το σύμβολο στο παράθυρο Διαχείριση λίστας ).
     
  3. Εξαγωγή μόνο των διευθύνσεων που έχουν ακυρώθει : θα γίνει εξαγωγή μόνο στις τρέχουσες διευθύνσεις που έχουν ακυρωθεί απο την επιλεγμένη λίστα (οι διευθύνσεις που έχουν σημειωθεί με την εικόνα στο παράθυρο Διαχείριση λίστας ).
     
  4. Εξαγωγή μόνο των ενεργοποιημένων διευθύνσεων: θα γίνει εξαγωγή μόνο στις τρέχουσες ενεργοποιημένες διευθύνσεις. Χρησιμοποιώντας τις εξελιγμένες επιλογές φιλτραρίσματος που θα βρείτε στο παράθυρο Διαχείριση λίστας . Αυτή η μέθοδος, σας επιτρέπει να κάνετε επιλεγμένες και συγκεκριμένες εξαγωγές.

Πατώντας το κουμπί Εξαγωγή θα εμφανιστεί ένα παράθυρο στο οποίο θα πρέπει να εισάγετε το όνομα του φακέλου που θα εξαχθεί. Το Πεδίο οριοθέτηση θα πρέπει επίσης να προσδιοριστεί (ο χαρακτήρας διαχωριστών των στηλών) και θα μπορείτε επίσης να διευκρινίσετε αν στην Πρώτη γραμμή θα περιέχονται τα ονόματα των πεδίων. Τελικά, θα’ έχετε την δυνατότητα να επιλέξετε τα πεδία που θα εξαχθούν, και αφού πατήσετε το κουμπί Επόμενο ,η διαδικασία εξαγωγής θα ξεκινήσει.

* Οι λειτουργίες εξαγωγής είναι διαθέσιμες μόνο στην έκδοση Pro.
