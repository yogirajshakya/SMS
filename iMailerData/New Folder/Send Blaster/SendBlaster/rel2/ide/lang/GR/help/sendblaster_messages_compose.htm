Σύνθεση μηνύματος

Το παράθυρο σύνθεσης μηνύματος, σας επιτρέπει να συνθέσετε το μήνυμα που θέλετε να στείλετε σε μια λίστα διανομής.

Η σύνθεση μηνύματος γίνεται όπως και στα κανονικά ηλεκτρονικά μηνύματα: πληκτρολογείστε το μήνυμα Θέμα, εισάγετε οτι χρειάζεται Επισύναψη*, πληκτρολογήστε το κείμενο του μηνύματος. Σε περίπτωση που το κείμενο χρειάζεται κάποια ειδική αλφάβητο (όχι Λατινικής βάσης), θα πρέπει πρώτα να ρυθμίσετε την Κωδικοποίηση (η προκαθορισμένη κωδικοποίηση μπορεί αλλάξει απο το παράθυρο Πεδία και εμφάνιση).

Προειδοποίηση: Η κωδικοποίηση του μηνύματος πρέπει να προσδιοριστεί πριν την πληκτρολόγηση του μηνύματος για να υπάρξουν τα σωστά αποτελέσματα. Αν αποτύχετε να ρυθμίσετε την κωδικοποίηση πριν απο την πληκτρολόγηση του κειμένου τότε μπορεί να έχετε ως αποτέλεσμα ένα κείμενο που δεν μπορεί να διαβαστεί.

Αυτή η σελίδα δείχνει δύο περιοχές κειμένων:

  • Στην κύρια περιοχή, όπου υπάρχει η μεγάλη γραμμή εργαλείων, μπορείτε εισάγετε μορφοποιημένα κείμενα για να δημιουργήσετε μηνύματα τύπου HTML. Μπορείτε επίσης να εισάγετε εικόνες και συνδέσμους, και μπορείτε να ελέγξετε (και να τροποποιήσετε) τον κωδικό HTML τον οποίο μπορείτε να δείτε ανα πάσα στιγμή, κάνοντας κλικ στο κουμπί Εμφάνιση κωδικού στην πάνω μπάρα.
     
  • Στην περιοχή Απλό κείμενο , μπορείτε να πληκτρολογήσετε το κείμενο του μηνύματος σε μορφή απλού κειμένου. Κάποιοι αποδέκτες μπορεί να μην έχουν την δυνατότητα να δουν μηνύματα σε μορφή HTML, άρα είναι σημαντικό να έχετε μια ενλλακτική μορφή περιεχομένων, χωρίς εικόνες και/ή συνδέσμους. Το Απλό κείμενο μπορεί να δημιουργείται αυτόματα αρχής γενομένης από το διαμορφωμένο κείμενο, κάνοντας κλικ στο κουμπί Κείμενο απο HTML όταν ολοκληρώσετε την πληκτρολόγηση του μηνύματος.

Για να στείλετε προεπιλεγμένα μηνύματα (σε αντίθεση με τα γενικευμένα μηνύματα), μπορείτε να εισάγετε στην γραμμή θέματος του μηνύματος και στο κύριο μήνυμα (και στον HTML τομέα αλλά και στον τομέα απλού κειμένου) Ετικέτες. Η κάθε ετικέτα είναι στην πραγματικότητα μια αναφορά σε κάποιο πεδίο: όταν το μήνυμα αποστέλλεται, η ετικέτα αντικαθίσταται απο την αξία που διαβάζεται απο το κείμενο του παραλήπτη στον συγκεκριμένο πεδίο. Για παράδειγμα, αν η λίστα συμπεριλαμβάνει έναν πεδίο "Όνομα εταιρίας" και ένα πεδίο "Διεύθυνση" (τα ονόματα των πεδίων μπορούν να τροποποιηθούν), πληκτρολογώντας μέσα στο μήνυμα ένα κείμενο όπως:

Messrs. #όνομα εταιρίας#
#διεύθυνση#

ο παραλήπτης θα λάβει ένα προεπιλεγμένο μήνυμα (παράδειγμα):

Messrs. Εταιρία Coca Cola
Ατλάντα, GA

Αφού τα ονόματα των πεδίων μπορούν να τροποποιηθούν, υπάρχει επίσης η δυνατότητα να χρησιμοποιηθούν "προσδιορισμοί" ετικετών (ανεξάρτητες ετικέτες ονομάτων πεδίων) που να ανφέρονται στους αριθμούς πεδίων (#1# #2#, κτλ. ως το #10#). Οι αριθμοί πεδίων μπορούν να βρεθούν στο παράθυρο Πεδία και εμφάνιση. Παρέχονται επίσης τρείς ακόμη ειδικές ετικέτες: #όνομα λίστας# η οποία αντικαθίσταται απο το όνομα της λίστας στην οποία αποστέλλεται το μήνυμα, #διαχείριση ηλεκτρονικής διεύθυνσης# η οποία αντικαθίσταται απο την ηλεκτρονική διεύθυνση στην οποία στέλνονται οι Εγγραφές/Διαγραφές (αυτό επιτρέπει στους αποδέκτες να αποσύρουν εύκολα την διεύθυνσή τους απο την λίστα), και #ημερομηνία συστήματος# το οποίο αντικαθίσταται απο την τρέχουσα ημερομηνία, διαμορφωμένη ανάλογα με τις ρυθμίσεις των Windows.

     

Πως να δημιουργήσετε έναν σύνδεσμο Διαγραφής

Το SendBlaster μπορεί να διαχειριστεί λίστες διανομής λαμβάνοντας μηνύματα απο μια συγκεκριμέη ηλεκτρονική διεύθυνση, διαφορετική για κάθε λίστα (Διαχείριση εγγραφών). Άρα χρησιμοποιώντας ετικέτες, είναι απλό να συμπεριλάβετε ένα σύνδεσμο διαγραφής στον κωδικό HTML:

<a href="mailto:#διαχείριση ηλεκτρονικής διεύθυνσης#?Θέμα=Διαγραφή">Απόσυρση</a>

Αυτός ο HTML κώδικός μπορεί να συμπεριληφθεί στο HTML μήνυμα πατώντας στο Εμφάνιση Κωδικού, ή μπορεί να εισαχθεί ως σύνδεσμος χρησιμοποιώντας το κουμπί Σύνδεσμος στην γραμμή εργαλείων μορφοποίησης (όπου η αξία mailto:#διαχείριση ηλεκτρονικής διεύθυνσης#?Θέμα=Διαγραφή θα πρέπει να προσδιοριστεί στο παράθυρο συνδέσμου και μόλις δημιουργηθεί ο σύνδεσμος, το κείμενο "Απόσυρση" (ή κάποιο παρόμοιο) μπορεί να πληκτρολογηθεί για να αντικαταστήσει το κείμενο που εμφανίζεται).

      

Το διορθωμένο μήνυμα μπορεί να αποθηκευτεί σε μορφή .eml (συμβατή με τους περισσότερους πελάτες ηλεκτρονικών ταχυδρομείων) πατώντας το κουμπί Αποθήκευση .

Για να δημιουργήσετε ένα νέο μήνυμα, μπορείτε να ξεκινήσετε απο την αφετηρία (με το κουμπί Νέο ή Επιλογή προτύπου) ή, χρησιμοποιώντας το κουμπί Άνοιγμα , και ανοίγοντας ένα μήνυμα που έχει αποθηκευτεί στο παρελθόν:

  • Εισάγετ ένα .eml αρχείο (για παράδειγμα, κάποιο που έχει δημιουργηθεί στο Outlook) με το Άνοιγμα / Άνοιγμα .eml αρχείου
  • Φορτώστε ένα μήνυμα που έχετε στείλει στο παρελθόν με το Άνοιγμα / Άνοιγμα ηλεκτρονικού μηνύματος απο Ιστορία
  • Εισάγετε ένα μορφοποιημένο κείμενο απο μια σελίδα HTML απο τον τοπικό σκληρό δίσκο, συμπεριλαμβανομένων και εικόνων, με το Άνοιγμα / Εισαγωγή HTML απο αρχείο
  • Εισάγετε απλό κείμενο με το Άνοιγμα / Εισαγωγή απλού κειμένου απο αρχείο

Το κουμπί Κείμενο απο HTML σας επιτρέπει να αποσπάσετε απλά κείμενα απο το μορφοποιημένο HTML κείμενο. Άρα σας συμβουλεύουμε να το χρησιμοποιήσετε αφου έχετε τελειώσει με την σύνθεση του μηνύματος.

Πριν στείλετε το μήνυμα, μπορείτε (και σας το συνιστούμε) να επιβεβαιωθείτε για το πως θα φαίνεται το μήνυμα στους πελάτες ηλεκτρονικού ταχυδρομείου, χρησιμοποιώντας το κουμπί Προβολή ηλεκτρονικού μηνύματος.

*Οι λειτουργίες επισύναψης είναι διαθέσιμες στην έκδοση Pro.
