Διαχείριση λιστών

Το SendBlaster μπορεί να διαχειριστεί πολλαπλή διανομή λιστών*.

Η τρέχουσα λίστα επιλέγεται απο το πτυσσόμενο κουτάκι το οποίο βρίσκεται στο πάνω μέρος του παραθύρου. Δίπλα απο αυτό, μπορείτε να δείτε 3 κουμπιά που θα σας βοηθήσουν να δημιουργήσετε μια νέα λίστα (κουμπί Νέο), να μετονομάσετε την τρέχουσα λίστα (Μετονομασία), να την διαγράψετε (Διαγραφή) ή να την στείλετε για εκτύπωση (Εκτύπωση). Αυτές οι λειτουργίες μπορούν επίσης να συμβούν πατώντας στο πτυσσόμενο κουτάκι με το δεξί κουμπί του ποντικιού σας.

Ο υπόλοιπος χώρος του παραθύρου είναι αφιερωμένος στην λίστα διευθύνσεων της επιλεγμένης λίστας διανομής.

Η κάθε διευθυνση δείχνει: το επίπεδο εγγραφών (σημειωμένο με και εικονίδια), την κατάσταση ενεργοποίησης (σημειωμένη σε ένα κουτάκι), το περιεχόμενο των 15 προσαρμοσμένων πεδίων (το πρώτο είναι πάντα η ηλεκτρονική διεύθυνση), η παρουσία σχολίων (σημειώνεται με το εικονίδιο , και πατώντας πάνω σε αυτό μπορείτε να ανοίξετε τα σχόλια). Η σειρά των στηλών μπορεί να τροποποιηθεί μετακινώντας τις στήλες (απλώς σύρετε την στήλη της κεφαλίδας). Οι στήλες μπορούν επίσης να αλλάξουν μέγεθος.

Κατά την διάρκεια μιας λειτουργίας αποστολής, οι διεύθύνσεις που είναι σημειωμένες ως διαγραμμένες ή ως ανενεργές θα παραληφθούν. Η λειτουργία αποστολής λοιπόν, θα επηρεάσει μόνο τις διευθύνσεις που είναι και εγγεγραμμένες και ενεργές. Μπορείτε εύκολα να εντοπίσετε τις διευθύνσεις που δεν θα περιληφθούν στην αποστολή γιατί θα εμφανίζονται με γκρι χρώμα:

  Στο παράδειγμα αριστερά, η λειτουργία αποστολής, θα επηρεάσει μόνο την πρώτη διεύθυνση αφού είναι η μοναδική που είναι και εγγεγραμμένη και ενεργή.

Η λίστα των διευθύνσεων μπορεί να ταξινομηθεί ελεύθερα με το να πατήσετε στην κεφαλίδα της στήλης. Σύροντας μία ή περισσότερες κεφαλίδες στον χώρο πάνω απο την λίστα (Σύρετε την κεφαλίδα μιας στήλης σε αυτό τον χώρο για να ομαδοποιήσετε τα δεδομένα) θα σας επιτραπεί να ομαδοποιήσετε τις διευθύνσεις βάση των περιεχομένων των επιλεγμένων πεδίων. Το Συνολικό κουτάκι θα εμφανιστεί, και θα σας δώσει την δυνατότητα (αν επιλεχθεί) να μάθετε τον αριθμό των διευθύνσεων που συμπεριλαμβάνονται σε κάθε ομάδα.

     

Πως να φιλτράρετε διευθύνσεις

Το να ενεργοποιείτε/απενεργοποιείτε προσωρινά κάποιες λειτουργίες (χωρίς να τις διαγράφετε) θα σας επιτρέψει να στέλνετε μηνύματα σε κάποια υποομάδα διευθύνσεων στην λίστα. Συνδυάζοντας αυτή την λειτουργία και τις λειτουργίες διαλογή/ομαδοποίηση θα έχετε την δυνατότητα να δημιουργήσετε πιο ισχυρό φιλτράρισμα στην διαδικασία αλληλογραφίας.

Για παράδειγμα, αν το πεδίο "Πόλη" έχει προσδιοριστεί, μπορείτε να στείλετε μήνυμα μόνο σε αυτούς που ζούν σε αυτή την συγκεκριμένη πόλη, απενεργοποιώντας προσωρινά όλες τις άλλες διευθύνσεις. Αν το πεδίο "Επάγγελμα" έχει προσδιοριστεί, μπορείτε να στέλνετε μηνύματα μόνο σε συγκεκριμένες επαγγελματικές κατηγορίες (επίσης είναι δυνατό να συνδυάσετε αυτά τα 2 φίλτρα χρησιμοποιώντας τη λειτουργία ομαδοποίησης: οι διευθύνσεις θα ομαδοποιούνται βάση "Πόλης" πρώτα και μετά βάση "Επαγγέλματος").

Για να δημιουργήσετε φίλτρα γρήγορα, πρώτα απενεργοποιήστε όλες τις διευθύνσεις, μετά ομαδοποιήστε τις και τέλος ενεργοποιήστε μόνο αυτές που σας ενδιαφέρουν.

Η απενεργοποίηση μίας ή πολλών διευθύνσεων θα σας δώσει επίσης την δυνατότητα να δημιουργήσετε αποκλεισμό απο λίστες (διευθύνσεις στις οποίες δεν θα σταλούν ποτέ μηνύματα, ούτε ακόμη κι αν ο χρήστης αυθόρμητα εγγραφεί στη λίστα.

      

Πάνω απο την λίστα διευθύνσεων, μπορείτε να βρείτε την γραμμή εργαλείων διαχείρισης της λίστας. Μπορείτε να έχετε πρόσβαση σε αυτές τις εντολές και απο το ευρύτερο μενού, το οποίο εμφανίζεται όταν κάνετε δεξί κλικ, με το ποντίκι σας, στην λίστα.

Μπορείτε να εισάγετε μια νέα διεύθυνση (κουμπί Νέο), να τροποποιήσετε μια επιλεγμένη διεύθυνση (Τροποποίηση) ή να διαγράψετε μια ή περισσότερες διευθύνσεις (Διαγραφή). Μπορείτε να επιλέξετε πολλαπλές διευθύνσεις χρησιμοποιώντας και το πληκτρολόγιο και το ποντίκι σας: πατώντας πάνω σε μια διεύθυνση ενώ κρατάτε πατημένο το πλήκτρο Ctrl θα προσθέσει αυτή την διεύθυνση στην επιλογή σας, πατώντας πάνω σε μια διεύθυνση ενώ κρατάτε πατημένο το πλήκτρο Shift θα προσθέσει μια ομάδα διευθύνσεων.

Με τα κουμπιά Αποκοπή / Αντιγραφή / Επικόλληση μπορείτε να διαγράψετε τις διευθύνσεις ή να τις αντιγράψετε σε/απο το clipboard (αν θέλετε να επιλέξετε όλες τις καταχώρησεις, κάντε δεξί κλικ σε οποιαδήποτε καταχώρηση και διαλέξτε Επιλογή όλων).

Τα κουμπιά Εγγραφή επιλεγμένης διεύθυνσης και Διαγραφή επιλεγμένης διεύθυνσης σας επιτρέπουν να διαχειριστείτε με το χέρι τις εγγραφές σας στην επιλεγμένη λίστα (η εγγραφή μιας διεύθυνσης θα τροποποιήσει την ημερομηνία εγγραφής). Τα κουμπιά Ενεργοποίηση επιλεγμένης διεύθυνσης και Απενεργοποίηση επιλεγμένης διεύθυνσης σας δίνουν την δυνατότητα να ενεργοποιήσετε/απενεργοποιήσετε μια διεύθυνση ανεξάρτητα απο την κατάσταση εγγραφής της.

Το κουμπί Ιστορία θα εμφανίσει μια λίστα απο μηνύματα που έχουν σταλεί στην επιλεγμένη διεύθυνση.

Πατώντας στο κουμπί Σημειώσεις ένα νέο παράθυρο θα εμφανιστεί στο οποίο θα πρέπει να εισάγετε ένα ελεύθερο κείμενο που θα συνδέεται με την τρέχουσα διεύθυνση. Οι διευθύνσεις που είναι συνδεδεμένες με κάποια σημείωση, είναι σημειωμένες με το εικονίδιο στην τελευταία στήλη (πατώντας πάνω σε αυτό το εικονίδιο θα οδηγηθείτε επίσης στο παράθυρο Σημειώσεις).

Το κουμπί Αναζήτηση** θα εμφανίσει ένα παράθυρο που θα σας επιτρέψει την αναζήτηση ενός κειμένου σε κάποιο συγκεκριμένο πεδίο ή σε πολλαπλά πεδία (συμπεριλαμβανομένων και των σημειώσεων). Επιλέγοντας Αντιστοιχία πλήρους λέξης η λειτουργία αναζήτησης θα εμφανίσει όλες τις διευθύνσεις που αντιστοιχούν ακριβώς στην προσδιορισμένη αξία. Αλλιώς η λειτουργία αναζήτησης θα εμφανίσει όλες τις διευθύνσεις που περιέχουν την αξία ακόμη κι αν δεν αντιστοιχεί ακριβώς με την προσδιορισμένη αξία. Για να μετακινηθείτε στη λίστα που προκύπτει χρησιμοποιήστε τα κουμπιά Εύρεση πρώτα, Επόμενο και Προηγούμενο στο παράθυρο αναζήτησης.

Το κουμπί οπτικού πεδίου Toggle σας δίνει την δυνατότητα να κάνετε εναλλαγή μεταξύ της κανονικής προβολής και της "Προβολής κάρτας" (και πίσω). Στην "Προβολή κάρτας" η κάθε διεύθυνση εμφανίζεται σαν ξεχωριστή κάρτα με προκαθορισμένο πλάτος. Η πλοήγηση γίνεται δυνατή από την οριζόντια γραμμή κύλισης.

Μπορεί να θέλετε να χρησιμοποιήσετε μόνο ένα υποσύνολο από τα 15 διαθέσιμα πεδία, προκειμένου να εξοικονομήσετε χώρο στην οθόνη, πατώντας στο κουμπί Εμφανή πεδία μπορείτε να επιλέξετε ποια πεδία θα εμφανιστούν.

*Η δωρέαν έκδοση μπορεί να διαχειριστεί μέχρι 2 λίστες διανομής.
**Η Αναζήτηση ειναι διαθέσιμη μόνο στην έκδοση Pro
