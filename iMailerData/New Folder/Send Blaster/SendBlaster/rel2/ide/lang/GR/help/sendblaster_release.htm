Τι καινούριο υπάρχει στο SendBlaster 2

  • Η βελτιωμένη και επανασχεδιασμένη διάταξη
     του SendBlaster είναι ακόμη πιο εύκολη στη χρήση απο πριν. Οι χρήστες μπορούν να επιλέξουν μεταξύ του κλασσικού πλαισίου στα αριστερά, που έχει επανασχεδιαστεί για να φαίνεται πιο ελκυστικό οπτικά, και τώρα έχει και μεγάλα ή μικρά εικονίδια, και μια νέα οριζόντια σειρά εργαλείων που πιάνει λιγότερο χώρο στην οθόνη σας. Όλα τα πλαίσια είναι τώρα ανακύλισης απο τον τροχό στο ποντίκι σας, η επιλογή λίστας μπορεί να συγχρονιστεί με όλα τα πλαίσια και εμφανίζεται με αλφαβητική σειρά. Όλα τα εικονίδια έχουν ανασχεδιαστεί, και πολλές άλλες μικρές βελτιωμένες λεπτομέρειες έχουν προστεθεί στη διάταξη.
     
  • Προγραμματισμένη αλληλογραφία
    Τώρα οι χρήστες μπορούν να δημιουργήσουν ένα ταχυδρομείο και να αφήσουν το SendBlaster 2 να το στείλει αυτόματα αργότερα, σε συγκεκριμένη ημέρα και ώρα. Ή οι χρήστες μπορούν ακόμη και να "σταθμέυσουν" το ταχυδρομείο τους για να μπορούν να το ξεκινήσουν χειροκίνητα αργότερα. Τώρα το SendBlaster μπορεί να τρέξει σε ελαχιστοποιημένο τρόπο προγράμματος και να ξεκινάει όταν ξεκινούν τα Windows.
     
  • Παρακολούθηση αλληλογραφίας
    Το SendBlaster 2 έχει ενσωματωμένη υποστήριξη, wizards και πλαίσια για το Google Analytics, δωρεάν υπηρεσία παρακολούθησης καμπάνιας του Google , και του TrackReports, υπάρχει η online υπηρεσία εξελιγμένης παρακολούθησης και αναφορών.
     
  • Νέα, δυνατά φίλτρα δεδομένων
    Ένα πολύ δυνατό φίλτρο έχει προστεθεί, το οποίο επιτρέπει στους χρήστες να τελειοποιήσουν την αλληλογραφία τους επιλέγοντας την ενεργοποίηση ή απενεργοποίηση των εγγρααφών που πληρούν ορισμένες προϋποθέσεις. Υποστηρίζει μεγάλο αριθμό συνθηκών και περίπλοκων λειτουργιών, ώστε να μπορεί να οικοδομηθεί μια ισχυρή SQL διατύπωση χωρίς να γράφεται ο κωδικος. Οι διπλές εγγραφές μπορούν τώρα να φιλτραριστούν αυτόματα, όπως στην έκδοση 1, ή διαχειριζόμενες απο τον χρήστη μέσω μιας νέας, δυνατής μηχανής αναζήτησης.
     
  • Περισσότερες δυνατότητες αποστολής
    Οι χρήστες μπορούν τώρα να προσθέσουν προκαθορισμένες κεφαλίδες, να ζητήσουν αποδείξεις ανάγνωσης, να θέσουν υψηλή προτεραιότητα για τα μηνύματά τους, να προσδιορίσουν ένα προκαθορισμένο Σε:, κεφαλίδα που να υποστηρίζει τη συγχώνευση ετικετών, ενεργοποίηση ή απενεργοποίηση του αυτόματου φιλτραρίσματος των διπλών διευθύνσεων όταν αποστέλονται.
     
  • Περισσότερες δυνατότητες διόρθωσης μηνυμάτων
    Πολλά καινούρια κουμπιά στο πλαίσιο σύνθεσης δίνουν την δυνατότητα στους χρήστες να προσθέσουν εύκολα συνδέσμους διαγραφής στο μήνυμά τους, να προσθέσουν ειδικούς συνδέσμους στο Google Analytics και στο TrackReports, και να προσθέσουν συγχωνευμένες ετικέτες στο θέμα και στο πεδίο Σε: . Τώρα τα κειμενικά μηνύματα (σε αντίθεση με τα HTML) μπορούν να δημιουργηθούν πιο γρήγορα απλώς επιλέγοντας ένα κουτάκι.
     
  • Έλεγχος Spam
    Τώρα υπάρχει η δυνατότητα να κάνετε έλεγχο spam καθώς διορθώνετε ένα μήνυμα. Ο έλεγχος spam βασίζετε στην δημοφιλή μηχανή SpamAssassin, με ένα απο τα πιο κοινά φίλτρα antispam των διακομιστών ηλεκτρονικών ταχυδρομείων.
     
  • Προεπισκόπιση με συγχωνευμένες ετικέτες
    Η προεπισκόπηση τώρα υποστηρίζει την υποκατάσταση συγχωνευμένων ετικετών επίσης, για να μπορούν οι χρήστες να έχουν ακριβή εικόνα του τι θα αποστείλουν.
     
  • Περισσότερη ταχύτητα
    Το SendBlaster τώρα υποστηρίζει μέχρι 50 συνδέσεις - κάτι το οποίο μπορεί να βελτώσει πάρα πολύ την ταχύτητα της αλληλογραφίας όταν χρησιμοποιείτε εναν γρήγορο SMTP διακομιστή. Επιπλέον, το SendBlaster δίνει στους χρήστες την επιλογή του να μην διακόψουν την αλληλογραφία σε περίπτωση που η σύνδεση με το Δίκτυο διακοπεί προσωρινά, το οποίο μπορεί να βοηθήσει με την ανεπιτήρητη αλληλογραφία.
     
  • Εγγραφές που προεπιλέγοντα και μορφές bounce-back
    Οι χρήστες μπορούν τώρα να προεπιλέξουν την εγγραφή και τα θέματα ακύρωσης και να προσθέσουν προεπιλεγμένες μορφές bounce-back για πιο εύκολη, και αυτόματη επεξεργασία των ανεπίδοτων μηνυμάτων.
     
  • Προφίλ των SMTP και POP3
    Χρησιμοποιώντας πολλαπλές ρυθμίσεις SMTP και POP3 είναι όλα πιο εύκολα απο πριν - οι πληροφορίες του λογαριασμού μπορούν να αποθηκευτούν σε πολλά προφίλ, έτσι ώστε όλες οι ρυθμίσεις του SMTP και/ή του POP3 να μπορούν να αλλάξουν με λίγα μόνο κλικ. Αυτό είναι εξαιρετικά χρήσιμο όταν στέλνετε μυνήματα εκ μέρους κάποιου άλλου.
     
  • Στιγμιότυπα
    Αποθηκεύστε τις ρυθμίσεις για οποιοδήποτε πλαίσιο και ξαναχρησιμοποιήστε τις αργότερα με ένα μόνο κλικ - είναι τόσο εύκολο όσο το να τραβάτε μια φωτογραφία. Τα Στιγμιότυπα επεκτείνουν την έννοια των "προφίλ" σε ένα καινούριο επίπεδο - σχεδόν κάθε ομάδα με σχετικές ρυθμίσεις στο SendBlaster μπορεί να αποθηκευτεί ως στιγμιότυπο.
     
  • Υποστήριξη για εξωτερικά πρόσθετα εξαρτήματα
    Οι χρήστες μπορούν να 'κατεβάσουν' δωρεάν πρόσθετα εξαρτήματα (μικρά εργαλεία που εκτελούν ειδικές δουλειές, όπως το να κάνουν εισαγωγές απο διάφορους πελάτες ηλεκτρονικών διευθύνσεων, κτλ) απο την ιστοσελίδα μας, και στο μέλλον θα προστεθούν πολλά ακόμη. Τα πρόσθετα εξαρτήματα API είναι κοινά και οι κατασκευαστές ενθαρρύνονται να δημιουργήσουν νέα, χρήσιμα πρόσθετα εξαρτήματα.
     
  • Νέα πρότυπα ευκολότερα στην διόρθωση
    Το SendBlaster 2 έχει περισσότερα απο 100 νέα επαγγελματικά σχεδιασμένα πρότυπα, όλα τους χρησιμοποιούν ευέλικτη διάταξη που μπορεί εύκολα να προσαρμοστεί.
     
  • Ευκολότερη πρόσβαση στα αρχεία δεδομένων
    Άμεση πρόσβαση στην βάση δεδομένων των αρχείων, για εφεδρικούς λόγους, είναι τώρα πιο εύκολη, αφού τώρα τα δεδομένα αποθηκεύονται μέσα στον φάκελο Έγγραφα    .
     
  • Δεν απαιτείται κανένα προνόμιο διοίκησης
    Το SendBlaster 2 μπορεί να τρέξει απο κανονικούς χρήστες σε οποιδήποτε σύστημα Windows
     
  • Έτοιμο για τα Windows 7
     
  • Και πολλά περισσότερα !
    Το SendBlaster 2 έχει πολλές ακόμη μικρές βελτιώσεις που επιτρέπουν στους χρήστες να δουλεύουν καλύτερα και γρηγορότερα.
