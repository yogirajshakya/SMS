Εισαγωγή

Μπορείτε να εισάγετε απο μια εξωτερική πηγή στην τρέχουσα επιλεγμένη λίστα. Υπάρχουν τέσσερις αποδεκτές πηγές:

  • Μπορείτε να εισάγετε στοιχεία απο ένα μορφοποιημένο αρχείο (.csv, .txt), απο τον τοπικό υπολογιστή ή το τοπικό δίκτυο, επιλέγοντας Εισαγωγή απο εξωτερικό αρχείο. “Το Μορφοποιημένο κείμενο” είναι ένα είδος ανταλλαγής που χρησιμοποιείται ευρέως απο λογιστικά φύλλα και λογισμικά βάσεων δεδομένων: τα περισσότερα λογισμικά αυτής της κατηγορίας έχουν την δυνατότητα να εξάγουν δεδομένα με αυτή την μορφή.
     
  • Μπορείτε να εισάγετε δεδομένα απο κάποια ιστοσελίδα επιλέγοντας Εισαγωγή απο την ιστοσελίδα μέσω http://. Η διαδικασία είναι παρόμοια με με την διαδικασία μορφοποιημένων κειμένων, εκτός απο το γεγονός οτι το αρχείο που θα εισαχθεί δεν βρίσκεται στον τοπικό υπολογιστή ή στο τοπικό δίκτυο αλλά μπορεί να βρεθεί μέσω του HTTP πρωτόκολλου. Τα δεδομένα που θα εισαχθούν μπορούν επίσης να παραχθούν απο ένα script (για παράδειγμα ASP ή PHP). Αυτός ο τρόπος θα σας επιτρέψει να εισάγετε τα περιεχόμενα μιας on-line βάσης δεδομένων που διαθέτει εξαγωγέα script. Ο σύνδεσμος παραπομπής θα προσδιοριστεί ως η πηγή των δεδομένων.
     
  • Μπορείτε να εισάγετε δεδομένα απο το Clipboard των Windows (αφου θα έχετε επικολλήσει δεδομένα απο κάποιο άλλο λογισμικό), επιλέγοντας Εισαγωγή απο το clipboard.
     
  • Μπορείτε να εισάγετε δεδομένα απο το Outlook Express Address Book (γνωστό και ως Windows Address Book), επιλέγοντας Εισαγωγή απο το Outlook Express.

Τα δεδομένα που θα φορτωθουν απο εξωτερικές πήγες μπορούν να χρησιμοποιηθούν για οποιαδήποτε απο τις ακόλουθες λειτουργίες:

  • Μπορούν να προστεθούν σε επιλεγμένη λίστα επιλέγοντας το Προσθήκη στην λίστα και εγγραφή νέας διεύθυνσης.
     
  • Μπορούν να μεταφερθούν απο επιλεγμένη λίστα διαλέγοντας την επιλογή Μεταφορά νέας διεύθυνσης απο την λίστα . Όλες οι παρούσες διευθύνσεις συμπεριλαμβανομένων και των πηγών δεδομένων θα μεταφερθούν.
     
  • Μπορούν να διαγραφούν απο την λίστα (χωρίς να μεταφερθούν και άρα διατηρώντας τα δεδομένα) διαλέγοντας την επιλογή Διαγραφή νέων διευθύνσεων . Αυτή η λειτουργία είναι παρόμοια με αυτήν που περιγράφηκε παραπάνω, αλλά οι διευθύνσεις δεν μεταφέρονται απο την λίστα, απλά σημειώνονται ως διαγραμμένες. Με αυτό τον τρόπο θα κρατήσετε τα δεδομένα στην λίστα ενώ θα αποφύγετε το να συμπεριλαμβάνετε τις διευθύνσεις σε λειτουργίες αποστολής ηλεκτρονικών μηνυμάτων.

Αφού θα έχετε επιλέξει την λίστα, την πηγή δεδομένων και την λειτουργία, πατήστε το κουμπί Εισαγωγή . Ένα παράθυρο-οδηγός θα εμφανιστεί και θα σας δώσει την δυνατότητα να διευκρινίσετε ορισμένες επιλογές που χρειάζονται για να ολοκληρωθεί η λειτουργία εισαγωγής:

  • Το μονοπάτι Εισαγωγής αρχείων, ή το URL εισαγωγής αρχείων στην περίπτωση εισαγωγής απο το δίκτυο (η εισαγωγή απο το Clipboard ή απο το Outlook Express δεν απαιτούν κανένα μονοπάτι).
     
  • Τα δεδομένα Μορφοποίησης (δεν χρειάζονται στην εισαγωγή απο το Outlook Express): αν επιλέξετε Οριοθετημένο θα πρέπει επίσης να διευκρινίσετε το Πεδίο οριοθέτησης και εαν η πρώτη σειρά θα περιέχει τα ονόματα των πεδίων. Επιλέγοντας το Ελεύθερη μορφοποίηση μόνο οι διευθύνσεις ηλεκτρονικού ταχυδρομείου θα εισαχθούν (εντοπίζονται αυτόματα) και οι διευκρινίσεις μορφοποίησης των δεδομένων δεν θα χρειαστούν.
     
  • Αν ακολουθείτε μια τυπική λειτουργία εισαγωγής (όπως προσθήκη δεδομένων, ή Προσθήκη στην λίστα και λειτουργία εγγραφής νέων διευθύνσεων), δύο ακόμη επιλογές θα είναι διαθέσιμες για να προσδιοριστεί η ενέργεια που πρέπει να γίνει όταν μια εξωτερική διεύθυνση (παρούσα στα δεδομένα που θα εισαχθούν) συμπεριλαμβάνεται ήδη στην λίστα προορισμού: διαλέγοντας την επιλογή Να μην προστίθενται διευθύνσεις όταν υπάρχουν ήδη στην λίστα , τότε αυτές οι διευθύνσεις θα αγνοούνται (αλλιώς θα προσθέτονται και θα υπάρχουν διπλά). Διαλέγοντας την επιλογή Εγγραφή διευθύνσεων που υπάρχουν ήδη στην λίστα , αν κάποια διεύθυνση υπάρχει ήδη στην λίστα προορισμού, αλλά δεν είναι εγγεγραμμένη (δείτε Διαχείριση λιστών), τα νέα δεδομένα δεν θα εισάγονται, αλλά οι διευθύνσεις θα σημειώνονται ως εγγεγραμμένες.
     
  • Στο ακόλουθο βήμα θα σας ζητηθεί τελικά να ρυθμίσετε την αλληλογραφία μεταξύ των εξωτερικών πεδίων πηγών των δεδομένων και των πεδίων του SendBlaster και, πατώντας το κουμπί Επόμενο , η λειτουργία εισαγωγής θα ξεκινήσει.

Προειδοποίηση: η σωστή ένδειξη του πεδίου της πηγής εξωτερικών δεδομένων που περιέχει διευθύνσεις ηλεκτρονικού ταχυδρομείου είναι πολλή σημαντική γιατί:

  1. Όλα τα εξωτερικά αρχεία που δεν περιέχουν, στο επιλεγμένο πεδίο, μια έγκυρη διεύθυνση ηλεκτρονικού ταχυδρομείου, θα αγνοούνται. Άρα αν προσδιοριστεί λάθος πεδίο, κανένα αρχείο δεν θα περιέχει έγκυρη ηλεκτρονική διεύθυνση και όλα τα εξωτερικά δεδομένα θα αγνοηθούν.
  2. Η λειτουργία σύγκρισης ανάμεσα στα εξωτερικά δεδομένα και στα δεδομένα της λίστας, για να εξακριβώθει αν μια διεύθυνση περιλαμβάνεται ήδη, γίνεται στην πραγματικότητα με το να συγκρίνονται οι ηλεκτρονικές διευθύνσεις που βρίσκονται στο συγκεκριμένο πεδίο.
