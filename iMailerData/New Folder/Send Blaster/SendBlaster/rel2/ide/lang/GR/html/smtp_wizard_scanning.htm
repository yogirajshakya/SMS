Αναζήτηση

Ο οδηγός ρυθμίσεων θα αναζητήσει μια διαθέσιμη πόρτα στο PC σας, προκειμένου να συνδεθεί στον εξυπηρετητή SMTP και να στείλει ένα δοκιμαστικό μήνυμα. Η διαδικασία μπορεί να διαρκέσει μερικά λεπτά, ανάλογα με το πόσο γρήγορα θα βρεθεί η διαθέσιμη πόρτα.

Αν χρησιμοποιείται λογισμικό ασφάλειας (firewall / antivirus), μπορεί να σας ζητηθεί να επιτρέψετε στο SendBlaster να συνδεθεί με το Internet. Εφόσον σας ζητηθεί, παρακαλούμε απαντήστε "ναι" προκειμένου το SendBlaster να αποκτήσει πλήρη πρόσβαση στο Internet. Επίσης, συνιστούμε να απενεργοποιήσετε προσωρινά στο antivirus τη λειτουργία ελέγχου εξερχόμενων e-mail.

Στο μεταξύ, μπορείτε να ρίξτε μια ματιά στις υπηρεσίες της SMTP.com που απευθύνονται αποκλειστικά σε χρήστες του SendBlaster.
