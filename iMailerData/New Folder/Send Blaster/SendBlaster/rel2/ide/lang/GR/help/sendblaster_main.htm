SendBlaster

Το Sendblaster είναι σχεδιασμένο να στέλνει με γρήγορο και αυτόματο τρόπο, μεγάλο αριθμό ηλεκτρονικών μηνυμάτων σε διάφορες λίστες διανομής. Η εξελιγμένη αυτή μηχανή δεδομένων, και η δυνατότητα να αποστέλονται προσωποποιημένα μηνύματα (“Αγαπητέ Κε. Τζών Σμίθ”, αντί του απρόσωπου και ανώνυμου “Αγαπητέ πελάτη”) θα σας δώσει την δυνατότητα να δημιουργείτε αυτόματα ενημερωτικά δελτία και διαφημιστικές καμπάνιες. Όλα μέσα απο ένα λογισμικό που θα τρέχει στην επιφάνεια εργασίας του υπολογιστή σας.

Χάρη στην μηχανή SmartSend υπάρχει η δυνατότητα να στέλνετε πολλά μηνύματα ταυτόχρονα, ακόμη και χωρίς SMTP διακομιστή, κάνοντας την αποστολή μηνυμάτων ασφαλή και γρηγορότερη απ'οτι μέσω ενός κανονικού πελάτη ηλεκτρονικού ταχυδρομείου. Το κάθε μήνυμα αποστέλλεται ξεχωριστά. Οι ιδιωτικότητα των παραληπτών προστατεύεται. Το να χρησιμοποιείτε το SendBlaster για να διαχειριστείτε πολλαπλές αποστολές είναι επίσης χρήσιμο γιατί σας αποτρέπει απο το να αποκαλύψετε κατά λάθος τις ηλεκτρονικές διευθύνσεις άλλων ανθρώπων.

Τα μηνύματα μπορούν να συνθεθούν απευθείας στο λογισμικό, με μια πλήρη σειρά εργαλείων σχηματισμού, ή να είσαχθούν απο εξωτερικά αρχεία (σελίδες HTML , που έχουν αποθηκευτεί ως .eml μηνύματα – για παράδειγμα απο το Outlook). Οι λειτουργίες διόρθωσης και εισαγωγής θα σας επιτρέψουν να εισάγετε εικόνες απεύθείας στο κεντρικό μήνυμα, κι έτσι να αποφύγετε την σύνδεση με εξωτερικά αρχεία (κάτι το οποίο οι τελευταίοι πελάτες ηλεκτρονικού ταχυδρομείου δεν επιτρέπουν ή είναι κάτι που τους οδηγεί σε προειδοποιήσεις ασφαλείας).

Η διαχείριση των λιστών διανομής είναι ιδιαίτερα βελτιωμένη: με λίγα μόνο κλικ θα’μπορείτε να δημιουργήσετε καινούριες λίστες, να τροποποιήσετε τις υπάρχουσες και να εισάγετε διευθύνσεις απο εξωτερικές πηγές. Μπορείτε να εισάγετε στοιχεία απο:

  • Ένα μορφοποιημένο φάκελο κειμένου (.csv)
  • Outlook Express
  • Το clipboard
  • Μια ιστοσελίδα

Υπάρχει επίσης η δυνατότητα να λαμβάνετε μηνύματα εγγραφής/διαγραφής σε μια συγκεκριμένη ηλεκτρονική διεύθυνση, διαφορετική για κάθε λίστα διανομής: για να εγγραφείτε σε μια λίστα, το μόνο που χρειάζεται είναι να στείλετε ένα μήνυμα σε αυτή την ηλεκτρονική διεύθυνση με την λέξη “Εγγραφή” στο πεδίο αντικείμενο (ή “Διαγραφή” για να διαγραφείτε). Το λογισμικό λαμβάνει τα μηνύματα και αυτόματα ενημερώνει τις αντίστοιχες λίστες διανομής.

Η διαχείριση των εγγραφών μέσω ηλεκτρονικών μηνυμάτων και η δυνατότητα να εισάγετε στοιχεία απο το δίκτυο θα σας επιτρέψει να προσθέσετε εύκολα λειτουργίες των λιστών διευθύνσεων στην ιστοσελίδα σας, αφήνοντας το SendBlaster να αναλάβει την ενημέρωση των λιστών και να στέλνει τα μηνύματα στους σωστούς παραλήπτες (άρα θα αποφύγετε το να στέλνετε μηνύματα σε ανθρώπους που δεν θα έπρεπε να τα λάβουν).

Η διαχέιριση ηλεκτρονικών διευθύνσεων των λιστών διανομής είναι πολύ εξελιγμένη: οι διευθύνσεις μπορούν να ενεργοποιηθούν ή να απενεργοποιηθούν με ένα απλό κλικ, και υπάρχει η δυνατότητα να φιλτράρετε τα στοιχεία με έναν πολύ εξελιγμένο τρόπο, που βασίζεται στο περιεχόμενο ενός ή πολλών πεδίων. Συνδυάζοντας αυτές τις δύο λειτουργίες, θα μπορείτε να στέλνετε μηνύματα σε πολύ συγκεκριμένες υποομάδες διευθύνσεων μιας συγκεκριμένης λίστας (για παράδειγμα, μόνο οι άνθρωποι που ζουν σε μια συγκεκριμένη περιοχή, ή οι επαφές που σχετίζονται με μια συγκεκριμένη επιχείρηση).

Το λογισμικό, αναγνωρίζει επίσης τις νέες εγγραφές και άρα μπορεί να στείλει μηνύματα σε ανθρώπους που δεν τα έχουν λάβει στο παρελθόν. Τα πεδία μπορούν να επιλεχθούν και υπάρχουν πολλές επιλογές αναζήτησης. Η κάθε διεύθυνση μπορεί να συνδεθεί με “PostIt” σημειώσεις, των οποίων το περιεχόμενο είναι επίσης εξερευνήσιμο.

Οι εκδόσεις

Κάποιες απο τις λειτουργίες που περιγράφονται στα αρχεία βοήθειας είναι διαθέσιμες μόνο στην έκδοση Pro. Εδώ θα βρείτε έναν σύντομο συγκριτικό πίνακα:
     

Πρώτα βήματα

Εδώ μπορείτε να δείτε πως θα συνθέσετε και θα στείλετε ένα μήνυμα με μερικά απλά βήματα:

1. Συνθέστε το μήνυμα: στον πίνακα Μηνύματα πατήστε το Σύνθεση μηνύματος, και συνθέστε (προσθέτοντας και εικόνες αν θέλετε) ή εισάγετε το μήνυμα απο ένα εξωτερικό αρχείο (για παράδειγμα μια ηλεκτρονική διεύθυνση που έχετε αποθηκεύσει στο παρελθόν στο Outlook Express). Σε αυτό το σημείο, θα ήταν χρήσιμο να διερευνήσετε την χρήση των δεικτών, που θα υποκατασταθούν από τα πραγματικά στοιχεία των αποδεκτών: Αγαπητέ #όνομα# θα γίνει Αγαπητέ Τζων Σμιθ.

2. Καθορίστε μια λίστα διευθύνσεων: πατήστε στο Διαχείριση λίστας στο πλαίσιο Λίστες και διευθύνσεις . Εισάγετε μια νέα διεύθυνση στην προκαθορισμένη λίστα ή δημιουργήστε μια νέα λίστα, για παράδειγμα χρησιμοποιώντας την λειτουργία Εισαγωγή δεδομένων απο το Outlook Express, δοκιμάστε τις λειτουργίες ομαδοποίησης, σύροντας την κεφαλίδα μιας στήλης στον ειδικό χώρο πάνω απο την κεφαλίδα της στήλης και ενεργοποιώντας/απενεργοποιώντας τις διευθύνσεις.

3. Στείλτε το μήνυμα: στο πλαίσιο Μηνύματα πατήστε την Αποστολή, επιλέξτε την λίστα παραλήπτη και διευκρινήστε τις παραμέτρους της SMTP σύνδεσης του διακομιστή ή δοκιμάστε την απευθείας αποστολή (στην οποία υπάρχουν πολύ γρήγορες επιλογές αποστολής).
