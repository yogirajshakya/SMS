Εξελιγμένες ρυθμίσεις

Σε αυτό το παράθυρο μπορείτε να ρυθμίσετε μερικές επιλογές εξελιγμένων ρυθμίσεων προγράμματος.

Διάταξη:

  • Εμφάνιση: σας επιτρέπει να προσδιορίσετε αν το πλαϊνό πλαίσιο θα πρέπει να φαίνεται με στυλ Explorer Bar (προαιρετικά με Μικρό Εικονίδιο για οικονομία χώρου), ή με στυλ Σειρά Εργαλείων .
     
  • Εμφάνιση του συνδέσμου TrackReports στο πλαίσιο: σας επιτρέπει να εμφανίσετε/κρύψετε τους συνδέσμους TrackReports στο βασικό πλαίσιο. Αν δεν χρησιμοποιείτε το TrackReports, μπορείτε να κρύψετε τον σύνδεσμο για να εξοικονομήσετε χώρο.
     
  • Εμφάνιση του συνδέσμου του Google Analytics στο πλαίσιο: σας επιτρέπει να εμφανίσετε/κρύψετε τον σύνδεσμο του Google Analytics στο βασικό πλαίσιο. Αν δεν έχετε σκοπό να χρησιμοποιείτετο Google Analytics, μπορείτε να κρύψετε τον σύνδεσμο για να εξοικονομήσετε χώρο.
     
  • Εμφάνιση συμβουλών κατά την εκκίνηση: ενεργοποιήστε το παράθυρο των συμβουλών, να εμφανίζεται όταν ξεκινάει το SendBlaster. Είναι καλή ιδέα να αφήσετε αυτή την επιλογή ενεργοποιημένη μέχρι να εξοικιωθείτε με το SendBlaster.
     
  • Συγχρονισμός επιλεγμένων λιστών ανάμεσα στα πλαίσια: αυτή είναι μια πολύ δυνατή επιλογή, που σας επιτρέπει να συνεχίσετε να δουλεύετε στην ίδια λίστα κατά μήκος όλων των πλαισίων μέχρι να το αλλάξετε αυτό. Για παράδειγμα, αν επιλέξετε μια λίστα στην οθόνη Αποστολή , και μετά πάτε στον τομέα Διαχείριση λιστών για να προσθέσετε μια νέα διεύθυνση, η λίστα που επιλέξατε στην οθόνη Αποστολή , θα έχει ήδη επιλεγεί στον τομέα Διαχείριση λιστών . Το μειονέκτημα είναι οτι, αν έχετε πολύ μεγάλες λίστες και αργό ηλεκτρονικό υπολογιστή, αυτή η επιλογή μπορεί να κάνει το πρόγραμμα λίγο πιο αργό: αν δείτε κάποια ανεξήγητη καθυστέρηση στην ανταπόκριση της διάταξης, απλά απενεργοποιήστε την.
     
  • Χρησιμοποιήστε το κουμπί Γλώσσα για να αλλάξετε την διάταξη γλώσσας του χρήστη του SendBlaster, η επιλεγμένη γλώσσα θα χρησιμοποιηθεί την επόμενη φορά που θα ξεκινήσει το SendBlaster.

Μηνύματα:

  • Προεπιλεγμένη κωδικοποίηση για νέα μηνύματα: Αυτή η επιλογή σας επιτρέπει να ρυθμίσετε ένα προκαθορισμένο χαρακτήρα κωδικοποίησης που θα χρησιμοποιηθεί για την σύνθεση νέων μηνυμάτων.

Εγγραφές ηλεκτρονικών διευθύνσεων:

  • Αυτός ο τομέας σας επιτρέπει να προσδιορίσετε τις γραμμές θέματος που θα πρέπει να χρησιμοποιηθούν για μηνύματα εγγραφής και διαγραφής. Θα πρέπει επίσης να επιλέξετε αν τα μηνύματα θα πρέπει να είναι απολύτως σε αντιστοιχία, για να επεξεργαστούν ως εγγραφές/ακυρώσεις, ή αν απλά θα πρέπει να περιέχουν τις προσδιορισμένες λέξεις. Για λόγους συμβατότητας, τα θέματα "Εγγραφή" και "Διαγραφή" θα δουλεύουν πάντα (παράλληλα με τα θέματα που θα προσδιορίσετε εδώ).

Προσαρμοσμένες μορφές Bounce-backs:

  • Χρησιμοποιώντας το κουμπί Διόρθωση μορφών , μπορείτε να προσθέσετε ή να διορθώσετε τις προσαρμοσμένες μορφές Bounce-backs. Αφού οι μορφές ηλεκτρονικού ταχυδρομείου bounce-back σχεδιάστηκαν αρχικά για να διαβάζονται απο ανθρώπους, το SendBlaster μπορεί να επεξεργαστεί τις πιο κοινές bounce-back μορφές χρησιμοποιώντας την εσωτερική του βάση δεδομένων, αλλά μπορεί να συναντήσετε και κάποιες μορφές που δεν αναγνωρίζονται απο το λογισμικό. Αν συμβεί αυτό, μπορείτε να τις προσθέσετε χειροκίνητα στη bounce-back βάση δεδομένων για να μπορεί το SendBlaster να επεξεργαστεί τις νέες bounce-back μορφές επίσης.

Πρόγραμμα του SendBlaster:

  • Αυτός ο τομέας σας επιτρέπει να προεπιλέξετε να ξεκινάει το SendBlaster με την μορφή Προγράμματος (ελαχιστοποιημένο στην σειρά προβολής εργαλείων) όταν ξεκινούν τα Windows. Με αυτό τον τρόπο οι προγραμματισμένες λειτουργίες αποστολής μπορούν να ξεκινήσουν αυτόματα και αυτές.

Αποστολή:

  • Αυτός ο τομέας θα σας επιτρέψει να επιλέξετε αν θέλετε να στέλνει το SendBlaster διπλά ηλεκτρονικά μηνύματα, όταν οι επαφές υπάρχουν στην λίστα πάνω απο μία φορές. Συνήθως, δεν χρειάζεται να στέλνετε πάνω απο ένα αντίγραφο μηνυμάτων της κάθε ηλεκτρονικής διεύθυνσης στις επαφές σας, και σας συμβουλεύουμε να μην επιλέξετε αυτή τη δυνατότητα , εκτός αν έχετε κάποιον πολύ συγκεκριμένο και ειδικό λόγο να στέλνετε διπλά μηνύματα σε διπλές επαφές.

Πρόσθετα εξαρτήματα:

  • Αυτός ο τομεας σας επιτρέπει να ενεργοποιήσετε ή να απενεργοποιήσετε τα πρόσθετα εξαρτήματα που έχουν εγκατασταθεί στο SendBlaster (χωρίς να τα απεγκαθιστάτε ακριβώς, κάτι το οποίο μπορείτε να κάνετε απο το Control Panel των Windows ή χρησιμοποιώντας την απεγκατάσταση κάθε πρόσθετου εξαρτήματος ξεχωριστά). Μπορείτε επίσης να 'κατεβάσετε' περισσότερα πρόσθετα εξαρτήματα για το SendBlaster χρησιμοποιώντας το κουμπί Περισσότερα πρόσθετα εξαρτήματα .

Βάση Δεδομένων:

  • Χρησιμοποιώντας το κουμπί Φάκελος Βάσης Δεδομένων , θα ανοίξετε τον φάκελο των windows όπου αποθηκεύονται τα δεδομένα. Αυτό είναι χρήσιμο για την συντήρηση, το backup και άλλες λειτουργίες της βάσης δεδομένων.

Πληροφορίες και άδεια:

  • Εδώ μπορείτε να διαβάσετε πληροφορίες για την έκδοση του λογισμικού και την άδεια. Δεν υπάρχει τίποτα να ρυθμιστεί εδώ (αν, για κάποιο λόγο, θέλετε να αλλάξετε τον κωδικό της άδειας, μπορείτε να το κάνετε απο την οθόνη του SendBlaster).
