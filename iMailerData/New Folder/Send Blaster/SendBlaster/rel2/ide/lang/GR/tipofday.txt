Με προσωρινή ενεργοποίηση/απενεργοποίηση ορισμένων επαφών, μπορείτε να στοχεύσετε ένα υποσύνολο της λίστας σας. Σε συνδυασμό με την ταξινόμηση/ομαδοποίηση, μπορείτε να εφαρμόσετε ισχυρά φίλτρα στην αποστολή μηνυμάτων.Για να δημιουργήσετε ένα φίλτρο, πρώτα απενεργοποιείτε όλες τις διευθύνσεις, μετά τις ομαδοποιείτε και τέλος, ενεργοποιείτε μόνο τις επιθυμητές διευθύνσεις.
Ένα από τα πλέον προηγμένα χαρακτηριστικά του SendBlaster είναι η πολλαπλή ομαδοποίηση. Μην παραβλέψετε αυτή τη μοναδική δυνατότητα, προσπαθήστε να πειραματιστείτε μαζί της: σύρετέ πολλαπλούς τίτλους στηλών στην περιοχή πάνω από την λίστα, και δείτε τι συμβαίνει.
Ποτέ δεν πρέπει να ανησυχείτε για τις διπλοεγγραφές: Το SendBlaster τις τακτοποιεί αυτόματα. Εάν η ίδια διεύθυνση απαντάται περισσότερες από μία φορά σε μια λίστα, τότε μόνο ένα μήνυμα θα παραδοθεί σε αυτή τη διεύθυνση e-mail - τα διπλότυπα φιλτράρονται αυτόματα κατά την αποστολή.
Εάν καταγράφετε πλήρη στοιχεία από τους χρήστες σας, και όχι μόνο το e-mail τους, χρησιμοποιείστε τη συγχώνευση ετικετών για την αποστολή εξατομικευμένων μηνυμάτων: Το "Αγαπητέ John" ακούγεται πολύ καλύτερα από ένα γενικό "Αγαπητέ χρήστη". Δεν χρειάζεται να ενεργοποιήσετε ή να ρυθμίσετε τίποτα, απλά επιλέξτε μια ετικέτα από τη λίστα στη φόρμα "Σύνθεση μηνύματος", και θα εισαχθεί στην τρέχουσα θέση μέσα στο μήνυμα.
Η ενσωματωμένη λειτουργία "Προεπισκόπηση" χρησιμοποιεί το προεπιλεγμένο σας πρόγραμμα-πελάτη ηλεκτρονικού ταχυδρομείου. Αν χρειάζεστε μεγαλύτερη ευελιξία, δημιουργήστε μια "δοκιμαστική" λίστα που να περιέχει μερικά από τα δικά σας e-mail (συμπεριλαμβανομένων Gmail, Hotmail κτλ.) και στείλτε το μήνυμά σας στη δοκιμαστική λίστα πριν κάνετε την κανονική αποστολή.
Μπορείτε να ξεκαθαρίζετε περιοδικά τις λίστες σας με την αυτόματη λήψη και επεξεργασία των απορριφθέντων μηνυμάτων.\r\rΟι διευθύνσεις στις οποίες δεν παραδόθηκαν μηνύματα θα διαγραφούν από τη λίστα, κάνοντας έτσι τις επόμενες αποστολές ταχύτερες, καθώς δεν θα σπαταλιέται χρόνος για τις ανύπαρκτες διευθύνσεις.
Δεν είστε υποχρεωμένοι να χρησιμοποιείτε μόνο τα ενσωματωμένα πρότυπα e-mail. Οποιοδήποτε αρχείο .Htm ή. Eml μπορεί εύκολα να μετατραπεί σε προσαρμοσμένο πρότυπο, με δύο απλά βήματα: πρώτα εισαγωγή, και στη συνέχεια αποθήκευση ως νέο πρότυπο.
Μην νομίζετε ότι τα πρότυπα χρησιμοποιούνται μόνο ως designs: αν έχετε γράψει ένα κείμενο που θα χρειαστεί να επαναχρησιμοποιηθεί στο μέλλον, μπορείτε να το αποθηκεύσετε ως νέο πρότυπο. Με τον τρόπο αυτό, την επόμενη φορά δεν θα χρειαστεί να το πληκτρολογήσετε ξανά από το μηδέν.
Μπορείτε να εισαγάγετε τις επαφές σας από οποιαδήποτε πηγή ικανή να εξάγει τα στοιχεία ως μορφοποιημένα αρχεία κειμένου.\rΤο μορφοποιημένο κείμενο (επίσης γνωστό ως .Csv, "comma separated values") είναι μια πολύ δημοφιλής μορφή ανταλλαγής δεδομένων μεταξύ διαφόρων προγραμμάτων.\r\rΓια παράδειγμα, μπορείτε να εξάγετε τις επαφές σας από το Outlook ή από ένα αρχείο Excel, και να κάνετε εισαγωγή τους στο SendBlaster.
Ο απλούστερος τρόπος να προσθέσετε ένα σύνδεσμο για εγγραφή στο web site σας, είναι να βάλετε ένα σύνδεσμο προς τη διεύθυνση email διαχείρισης της λίστας, με τη λέξη "Subscribe" στον τίτλο. Αυτό μάλιστα δεν απαιτεί καθόλου κώδικα στο server σας.\rΑν η ιστοσελίδα σας υποστηρίζει PHP ή ASP, μπορείτε να προσθέσετε ένα ολοκληρωμένο σύστημα καταγραφής στοιχείων των χρηστών σας. Μπορείτε να κατεβάσετε δωρεάν φόρμες και scripts από την ιστοσελίδα μας.
Είναι καλύτερα να χρησιμοποιείτε μια ειδική διεύθυνση email για τη διαχείριση των λιστών σας (π.χ. "subscriptions@mysite.com"), αντί για την ίδια διεύθυνση από την οποία στέλνετε (δηλαδή "news@mysite.com»): Με τον τρόπο αυτό, οι απαντήσεις από τους χρήστες σας δεν ανακατεύονται με τις συνδρομές και τις ακυρώσεις, και έτσι η επεξεργασία τους είναι ταχύτερη.
Αν ρυθμίσετε το SendBlaster να κατεβάζει συνδρομές/ακυρώσεις από την ίδια ηλεκτρονική διεύθυνση που χρησιμοποιείτε και με άλλα προγράμματα-πελάτες ηλεκτρονικού ταχυδρομείου, πάντα να τσεκάρετε τη ρύθμιση "Παραμονή αντιγράφου του μηνύματος στον εξυπηρετητή" στην περιοχή "Διαχείριση συνδρομών" και "Διαχείριση μη παραδοθέντων μηνημάτων".\rΜε τον τρόπο αυτό θα μπορείτε να κάνετε λήψη των μηνυμάτων και από το εξωτερικό σας πρόγραμμα.
Οι προηγούμενες μαζικές σας αποστολές καταγράφονται στο "Ιστορικό". Μπορείτε επαναχρησιμοποιήσετε ένα μήνυμα που έχει ήδη σταλεί, ακόμη και αν δεν το είχατε αποθηκεύσει, με την επιλογή "’νοιγμα e-mail από το Ιστορικό" στη "Σύνθεση μηνύματος".
Αντί να προσθέτετε συνημμένα στο μήνυμά σας, ανεβάστε τα αρχεία στο web site σας, και στη συνέχεια τοποθετήστε συνδέσμους προς τα αρχεία μέσα στο μήνυμά σας: Έτσι, η παράδοση θα είναι πολύ πιο γρήγορη και οι παραλήπτες δε θα αναγκάζονται να κατεβάζουν μεγάλα μηνύματα.
Για να βελτιώσετε το ποσοστό παράδοσης μηνυμάτων, να αποφεύγετε την προσθήκη εικόνων στα μηνύματά σας: Τα σύγχρονα φίλτρα antispam γίνονται καχύποπτα όταν αντιμετωπίζουν ένα μήνυμα που περιέχει πολλές εικόνες, και μπορεί να χαρακτηρίσουν το μήνυμά σας ως spam ή και να το διαγράψουν.
Τα διάφορα προγράμματα ηλεκτρονικού ταχυδρομείου παρουσιάζουν το HTML με διαφορετικό τρόπο. Προσπαθήστε να αποφεύγετε την πολύπλοκη μορφοποίηση στα μηνύματά σας: Να τα διατηρείτε απλά, ώστε να είναι ευανάγνωστα σε όλες τις πλατφόρμες (και η εμφάνισή τους να είναι πάντα η ίδια).
Για να βελτιώσετε το ποσοστό παράδοσης και να μη σας μπλοκάρουν τα συστήματα antispam, πάντοτε να συμπεριλαμβάνεται και "απλό κείμενο" στο μήνυμά σας. Το SendBlaster μπορεί και δημιουργεί αυτόματα απλό κείμενο με βάση το HTML, όταν κάνετε κλικ στο κουμπί "Κείμενο από HTML". Αυτό θα πρέπει να είναι η τελευταία σας ενέργεια πριν από την αποστολή των μηνυμάτων.
Όταν στέλνετε ένα newsletter, πάντα να συμπεριλαμβάνετε ένα σύνδεσμο ακύρωσης συνδρομής στο τέλος του μηνύματος: Απλά προσθέστε ένα σύνδεσμο προς τη διεύθυνση διαχείρισης της λίστας σας με τίτλο "Unsubscribe". Αν δεν είστε εξοικειωμένοι με τους συνδέσμους, μπορείτε να ρίξτε μια ματιά στις οδηγίες χρήσης.
Το SendBlaster στέλνει κάθε μήνυμα ως ένα ξεχωριστό e-mail, δεν χρησιμοποιεί "Κοινοποίηση" ή "Bcc": Οι αποδέκτες δεν μπορούν να δουν τις άλλες διευθύνσεις e-mail στις οποίες το μήνυμα έχει σταλεί, και δεν λαμβάνουν μηνύματα με αόριστο παραλήπτη.
Εάν πρέπει να χρησιμοποιήσετε μια ειδική κωδικοσελίδα (διαφορετική από την προεπιλεγμένη), να την επιλέξετε πριν ξεκινήσετε τη σύνθεση του μηνύματός σας: Η επιλογή "charset" θα πρέπει να είναι η πρώτη ενέργεια που κάνετε, έτσι ώστε οι χαρακτήρες που πληκτρολογείτε να φαίνονται σωστά στο παράθυρο προεπισκόπησης.
Αν θέλετε να στείλετε ένα μήνυμα με απλό κείμενο μόνο, χωρίς HTML, δημιουργήστε ένα νέο μήνυμα και επεξεργαστείτε μόνο το κομμάτι "Απλού κειμένου", αφήνοντας το παράθυρο του επεξεργαστή HTML κενό: Το SendBlaster αυτόματα θα αναγνωρίσει ότι το HTML είναι κενό, και θα στείλει e-mail απλού κειμένου. \r\rΠριν την αποστολή, πάντα κάνετε προεπισκόπηση στο μήνυμά σας για να βεβαιωθείτε ότι δεν μπήκε κάτι στο HTML κατά λάθος.
Η ταχύτητα του SendBlaster σε μεγάλο βαθμό εξαρτάται από τη σύνδεσή σας στο Internet και την απόδοση του εξυπηρετητή SMTP: Αν δεν είστε σίγουροι για το πόσο γρήγορα μπορούν να σταλούν τα μηνύματα, ξεκινήστε με πιο αργές ρυθμίσεις (επιλέξτε ένα αργό προφίλ στη φόρμα "Αποστολής"). Στη συνέχεια, κάντε βαθμιαία αύξηση της ταχύτητας (επιλέξτε ένα ταχύτερο προφίλ ή μικρότερες παύσεις ή αύξηση των συνδέσεων).
Αντί να χρησιμοποιείτε μόνο SMTP ή μόνο ’μεση Αποστολή, δοκιμάστε να χρησιμοποιήσετε ένα συνδυασμό των δύο μεθόδων: Επιλέξτε SMTP ως την κύρια μέθοδο και ενεργοποιήστε την επιλογή "Επανάληψη με ’μεση Αποστολή σε περίπτωση λάθους".
Στη φόρμα "Αποστολής", χρησιμοποιήστε κωδικοποίηση "Quoted printable" αν τα μηνύματά σας αποτελούνται κυρίως από κείμενο. Επιλέξτε "base64" μόνο αν περιέχουν πολλές εικόνες ή συνημμένα (το μέγεθος μηνύματος θα είναι μικρότερο, αλλά ορισμένα φίλτρα antispam μπορεί να δημιουργήσουν προβλήματα).
Οι λίστες επαφών μπορεί να περιέχουν πολλά πεδία, αλλά ίσως να μην τα χρειάζεστε όλα: Στη "Διαχείριση λιστών", χρησιμοποιείστε τη λειτουργία "Ορατά πεδία" για να αποκρύψετε τα πεδία δεν χρειάζεστε.
Η ολοκλήρωση αποστολής μηνυμάτων μέσα από το "Ιστορικό" είναι ένας πολύ καλός τρόπος για να αντιμετωπίσετε περιορισμούς στον εξυπηρετητή SMTP που διαφορετικά δεν θα σας επέτρεπαν αποστολή πολλών μηνυμάτων μικρό χρονικό διάστημα. Ξεκινήστε την αποστολή, και διακόψτε τη μετά από λίγο. Ολοκληρώστε την αποστολή αργότερα - το SendBlaster θα αρχίσει την αποστολή, από το σημείο που είχε διακοπεί (δεν θα αποσταλούν διπλά μηνύματα).
"Εισαγωγή" δεν σημαίνει απλά προσθήκη νέων επαφών σε μια λίστα. Μπορείτε να πετύχετε περισσότερα με τα εξωτερικά σας δεδομένα: Για παράδειγμα, αν έχετε ένα αρχείο που περιέχει ανύπαρκτες/απαγορευμένες διευθύνσεις, μπορείτε να το εισαγάγετε έτσι ώστε οι διευθύνσεις να αφαιρεθούν από τη λίστα σας.
Οι Μαύρες Λίστες είναι απαραίτητες για να αποφύγετε εύκολα και σίγουρα την αποστολή μηνυμάτων σε ορισμένους παραλήπτες, ή για να αποτρέψετε ανεπιθύμητες συνδρομές: Αν μια διεύθυνση e-mail ή όνομα έχει προστεθεί στη μαύρη λίστα, τότε δεν πρόκειται ποτέ να σταλεί σε αυτήν μήνυμα, ανεξάρτητα από το αν η διεύθυνση συμπεριλαμβάνεται στις λίστες σας.
