Benvenuti in SendBlaster, il software semplice e potente per la gestione di mailing list direttamente dal PC dell'utente:

  • Gestisce liste multiple
     
  • Crea newsletter con allegati ed immagini incorporate
       I più recenti sistemi operativi non consentono di visualizzare eventuali immagini remote richiamate nei messaggi: le immagini sono perciò incorporate nel corpo della mail, evitando qualsiasi problema di lettura del messaggio
     
  • Importa pagine HTML e messaggi composti con un client di posta elettronica (formato .eml)
       Le immagini esterne referenziate dal codice HTML vengono incorporate nel messaggio, così come vengono conservati eventuali allegati presenti nel file .eml
     
  • Permette di inviare messaggi personalizzati
       Ogni destinatario può ricevere un messaggio personalizzato: il testo della mail può contenere marcatori che, in fase di invio, sono sostituiti con dati prelevati da ogni indirizzo
     
  • Consente l'invio diretto di messaggi senza usare un server SMTP
       Con la modalità di invio diretto, se si usano più connessioni ad Internet non sarà necessario modificare nessun parametro per connettersi al servizio SMTP del provider
     
  • Invia molti messaggi velocemente
       Grazie al motore SmartSend possono essere inviati più messaggi contemporaneamente
     
  • Gestisce iscrizioni e cancellazioni inviate ad un indirizzo determinato
       Gli utenti possono iscriversi e cancellarsi semplicemente inviando all'indirizzo scelto un messaggio con oggetto "Subscribe" o "Unsubscribe ". SendBlaster scarica i messaggi via POP3 ed aggiorna le liste
     
  • Importa dalla rubrica di Outlook Express
     
  • Importa liste di indirizzi in formato .csv e preleva gli indirizzi via web
       Se il proprio sito utilizza un sistema di iscrizione / cancellazione e dispone di uno script per l'esportazione degli indirizzi, SendBlaster può richiamarlo direttamente ed aggiornare il database locale
     
  • E' facile da integrare in qualsiasi sito
       Inserire nel proprio sito funzionalità registrazione alla mailing list è semplice: non servono script, è sufficiente un semplice link all'indirizzo e-mail di gestione
