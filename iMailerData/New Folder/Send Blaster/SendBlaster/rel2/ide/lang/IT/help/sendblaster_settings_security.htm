Sicurezza

E' possibile proteggere l'accesso al programma richiedendo l'uso di una password*.

  • Per impostare una nuova password (se in precedenza i dati non erano protetti) digitarla nel campo Nuova password e ripeterla identica nella casella Conferma nuova password, infine premere il pulsante Salva.
     
  • Per modificare una password inserire la password corrente nel campo Vecchia password, quindi digitare la nuova password nel campo Nuova password e ripeterla identica nella casella Conferma nuova password, infine premere il pulsante Salva.
     
  • Per rimuovere una password (rendendo i dati liberamente accessibili) digitare la password corrente nel campo Vecchia password, lasciare vuoti i campi Nuova password e Conferma nuova password, quindi premere il pulsante Salva.

Le impostazioni avranno effetto al riavvio del programma.

Importante: Se si decide di proteggere i dati, conservare con cura la password! In caso di smarrimento non sono disponibili funzionalità per recuperarla e non sarà più possibile accedere al programma. Per poter accedere nuovamente al programma sarà necessario disinstallarlo, cancellare manualmente tutti i file residui (e tutte le sottocartelle) nella cartella in cui era installato (per ragioni di sicurezza gli archivi non vengono infatti cancellati automaticamente), ed infine reinstallarlo.

*La protezione con password è disponibile solo nell'edizione Pro
