Поля

В этом окне Вы можете выставить несколько опций, относящихся к программе.

Установить названия полей: SendBlaster позволяет изменять названия 15 доступных полей для каждого адреса (поле Email выделено красным, потому что вне зависимости от названия оно будет использовано для отправки сообщений, и оно должно всегда содержать email-адреса реципиентов). Просто введите новые названия в соответствующие поля и нажмите на кнопку Сохранить.

Осторожно: После смены названий полей, метки составления текста также сменятся; следовательно, текст сообщений, возможно, придётся также изменить для соответствия.

Мои личные данные: В этой секции Вы можете указать свои личные детали, которые будут использованы в функции слияния при предпросмотре на экране составления сообщения, эта функция даст Вам более подробное представление о том, как будет выглядеть письмо.
