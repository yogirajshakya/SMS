Составить сообщение

Окно составления сообщения позволяет Вам создавать сообщение, которое будет отправлено в список рассылки.

Составление сообщений производится сходным с обыкновенными почтовыми клиентами образом: наберите Subject (тему) сообщения, прикрепите все необходимые Приложения*, укажите реципиента электронного письма (используется только для предпросмотра) и введите сам текст сообщения. Если тексту требуется алфавит, отличный от латинницы, то сначала надо будет установить Кодировку (кодировка по умолчанию может быть изменена в окне Прочих настроек).

Метки могут быть вставлены и использованы нажатием кнопок меток (#) рядом с полями Subject и To (кому) для отправки персонализированных заголовков писем.

Внимание: кодировка сообщения должна быть указана до набора сообщения для получения корректных результатов. Если этого не сделать, текст может сохраниться в нечитаемом виде.

На странице есть две текстовых области ввода:

  • В главной области (где расположена длинная панель инструментов) Вы можете вставлять форматированный текст для генерации HTML-сообщений. Вы также можете вставить изображения и ссылки, и в любое время проверить (или изменить) автоматически сгенерированный HTML-код нажатием кнопки Показать код на верхней панели.
     
  • В области Простой текст Вы можете набрать сообщения простым неформатированным текстом. Некоторые получатели могут не иметь возможности просмотреть сообщение в формате HTML, следовательно, важно предоставлять альтернативный формат для содержимого, без изображений и ссылок. Простой текст может быть автоматически сгенерирован из текста с форматированием нажатием кнопки Текст из HTML после того, как Вы закончите набирать сообщение.

Для отправки произвольных сообщений (в отличие от обощённых) в теме и теле сообщения (как в области HTML, так и в области простого текста) Вы можете вставлять Метки. Каждая метка на самом деле является заменяемой ссылкой на поле: когда сообщение отправляется, метка заменяется на значение, прочитанное из текста реципиентов указанного поля. Например, если список включает в себя поле «НазваниеБизнеса» и поле «Адрес», (названия полей могут быть настроены), при вводе следующего текста сообщения:

Уважаемые #businessname#
#address#

реципиент получит сообщение (пример):

Уважаемые The Coca Cola Company
Atlanta, GA

Так как поля могут быть изменены, возможно использование меток, независимых от названия поля, которые ссылаются на него по номеру (#1# #2#, и до #10#). Номера полей могут быть найдены в окне Полей. Также существуют три специальных метки: #listname#, который заменяется на название списка рассылки, в который отправляется сообщение, #managingemail#, который заменяется на email-адрес, на который отправляются управляющие сообщения (подписка/отписка для предоставления подписчикам возможности легко отказываться от получения корреспонденции из списка), и #systemdate#, который заменяется на текущую дату, отформатированную согласно настройкам Windows.

     

Как создать ссылку на прекращение подписки

SendBlaster может управлять списками рассылки при помощи получения сообщений из специального почтового ящика, различного для каждого списка (Управление подписками). Следовательно, при помощи меток легко включить в HTML-код ссылку на прекращение подписки:

<a href="mailto:#managingemail#?Subject=Unsubscribe">Прекратить подписку</a>

Этот HTML-код может легко быть включён в HTML-сообщение нажатием кнопки Вставить ссылку на прекращение подписки с панели кнопок, которая также позволяет указать текст, который ведёт в ссылку на прекращение подписки.

      

Отредактированное сообщение может быть сохранено в формате .eml (совместимом с большинством почтовых программ) нажатием кнопки Сохранить.

Чтобы создать новое сообщение, Вы можете или начать с пустого текста (кнопки Новое или Выбрать шаблон) или, нажав на кнопку Открыть, открыть сохранённое ранее сообщение:

  • Импортировать .eml-файл (например, созданный в Outlook) нажатием Открыть / Открыть .eml-файл
  • Загрузить отправленное ранее сообщение при помощи Открыть / Открыть email-сообщение из Истории
  • Импортировать форматированный текст из HTML-страницы на локальном жёстком диске, включая изображения: Открыть / Импортировать из HTML-файла
  • Импортировать простой текст: Открыть / Импортировать простой текст из файла

Кнопка Текст из HTML позволяет Вам извлечь простой текст из форматированного HTML-текста. Нажимать её следует по окончанию составления текста сообщения.

Функции отслеживания email-сообщений могут быть добавлены в письмо при помощи кнопок Google Analytics и/или TrackReports на панели HTML; ссылки в сообщении автоматически изменяются в согласии с выбранной системой, добавляется весь необходимый код. Добавление кода Google Analytics или TrackReports должно всегда быть последней операцией, производимой над сообщением — оба метода изменяют конечный код, следовательно, пользователи должны сначала завершить редактирование. Отслеживание сообщения обеими системами возможно; в этом случае, пользователь должен сначала добавить Google Analytics.

Перед отправкой сообщения возможно (и настоятельно рекомендуется) проверить, как сообщение будет выглядеть в почтовых клиентах, при помощи кнопки Предпросмотр электронного письма. Предпросмотр сообщения будет использовать информацию об отправителе по умолчанию (определяемую в секции Мои личные данные на экране Поля.)

Возможно также на лету во время редактирования производить подсчёт количества спам-очков нажатием кнопки Спам-проверка; спам-очки высчитываются на основе популярной программы SpamAssassin, одним из наиболее распространённых антиспам-фильтров при email-серверах. Используя эту продвинутую функциональность SendBlaster, Вы избегаете случайной классификации Вашего сообщения как спама.

*Функциональность, связанная с приложениями, доступна в версии Pro.
