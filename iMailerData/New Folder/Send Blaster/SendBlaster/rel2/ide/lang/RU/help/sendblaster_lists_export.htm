Экспортирование

SendBlaster содержит функцию экспорта данных из списка рассылки* во врешний файл в формате .csv. После выбора экспортируемого списка Вы можете выбрать один из доступных методов экспорта:

  1. Экспортировать все адреса: все адреса в списке будут экспортированы.
     
  2. Экспортировать только подписанные адреса: будут экспортированы только адреса с действительной подпиской на выбранный список рассылки (адреса, отмеченные символом  в окне Управление списками).
     
  3. Экспортировать только неподписанные адреса: экспортировать только адреса в отключённой на данный момент подпиской на текущий список (адреса, отмеченные иконкой  в окне Управление списками).
     
  4. Экспортировать только включённые адреса: экспортировать только включённые адреса. С использованием опций сложных фильтров из окна Управление списками можно делать очень специфичное экспортирование данных.

Если нажать на кнопку Экспортировать, покажется окно, в котором можно выбрать название файла, в который будет производиться экспортирование. Разделитель полей — указание символа для разделения столбцов. Также можно установить, будет ли Первая строка содержит названия полей. Наконец, Вы сможете выбрать поля, которые будут экспортированы. После нажатия кнопки Далее начнётся процесс экспорта.

* Функции экспорта будут доступны в Pro-версии.
