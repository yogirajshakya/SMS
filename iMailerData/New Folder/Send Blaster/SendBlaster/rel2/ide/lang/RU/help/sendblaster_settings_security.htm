Безопасность

Доступ к программе может быть закрыт паролем*.

  • Для установки нового пароля (если доступ не был защищён ранее) введите новый пароль в поле Новый пароль, и ещё раз в поле Подтвердите пароль, а потом нажмите кнопку Сохранить.
     
  • Для изменения пароля наберите текущий пароль в поле Старый пароль, потом наберите новый пароль в поле Новый пароль, наберите его ещё раз в поле Подтвердите пароль, и, наконец, нажмите кнопку Сохранить.
     
  • Для снятия пароля (приводящего к тому, что программа будет доступна всем), введите текущий пароль в поле Старый пароль, оставьте поля Новый пароль и Подтвердите пароль пустыми, а потом щёлкните по кнопке Сохранить.

Новые настройки будут доступны, когда программа будет запущена в следующий раз.

Осторожно: Если Вы решите защитить программу паролем, хорошо позаботьтесь о его сохранении! Возможности получить доступ к программе при утере пароля не предусмотрено, и если он потеряется, Вы не сможете польше получить доступ к старым данным. В этом случае придётся деинсталлировать программу, вручную удалить оставшиеся файлы и директории в месте, в которое она была установлена, и установить программу заново. Все старые данные будут утеряны!

*Защита паролем доступна только в версии Pro
