Release Notes

  • Это бета-версия программы; некоторые функции работают не в полной мере.
  • Документация не была обновлена и относится к версии 1.

Что нового в sendBlaster 2

Разработка SendBlaster 2 заняла более одного года; на каждом шаге разработки программы мы прислушивались к пожеланиям пользователей и их мнениям.

  • Улучшен и исправлен пользовательский интерфейс SendBlaster, работа с которым стала еще проще, чем раньше. Пользователи могут осуществить выбор между классической левой панелью, которая стала еще более привлекательна внешне. Помимо этого пользователь может выбрать размер отображающихся иконок на ней. Либо пользователь может выбрать новую панель, которая сохраняет горизонтальное место на экране. Обе панели теперь можно пролистать, используя колесо мыши; списки расположены в алфавитном порядке, помимо этого выбор списков может быть синхронизирован во всех панелях; изображение всех иконок были изменены ; помимо этого были осуществлены другие улучшения пользовательского интерфейса.
     
  • Рассылка по расписанию Теперь пользователи могут создать рассылку и настроить программу таким образом, что рассылка будет осуществлена позже автоматически, в установленное время и день. Пользователи могут даже "приостановить" рассылку, а затем запустить ее снова. Теперь SendBlaster может работать в состоянии "по расписанию" и запускаться при запуске Windows.
     
  • E-mail Отслеживание Теперь в SendBlaster 2 встроена поддержка, мастер настройки и панель для Google Analytics (бесплатная услуга от Google для мониторинга проведенных кампаний), а так же TrackReport (наша дополнительная онлайн услуга по отслеживанию рассылок).
     
  • Обновленные и улучшенные фильтры для информации Был добавлен улучшенный фильтр, который позволяет пользователям настроить их рассылку путем выбора определенных параметров, удовлетворяющих заданным условиям. Фильтр позволяет задавать несколько параметров, поэтому пользователи смогут создать SQL запрос без написания кода.Повторяющиеся адреса могут быть отфильтрованы автоматически или пользователь сможет сам управлять ими через мощную систему поиска.
     
  • Больше функций для рассылки Пользователи теперь при отправке сообщений могут добавить индивидуальные заголовки, запросить уведомления о прочтении письма, установить высокую степень важности писем, определить кому будет предназначено письмо: заголовок, поддерживающий объединенные тэги, возможность включить или выключить автоматическую фильтрацию пофторяющихся адресов.
     
  • Дополнительные функции для правки сообщений Множество новых иконок на панели позволяют пользователю с легкостью добавлять в сообщение ссылки на отмену подписки, добавлять специальные ссылки для Google Analytics и TrackReports, а так же добавлять объединенные тэги в тему письма и кому оно адресовано. Теперь рассылка, содержащая только текст (в отличие от HTML), может быть создана всего лишь добавлением галочки в один из чекбоксов.
     
  • Проверка на спам Теперь стало возможным при корректировки сообщения проверить его на спам; проверка на спам основывается на известном SpamAssassin, одном из наиболее часто встречающихся анти-спам фильтре, использующемся на e-mail серверах.
     
  • Просмтор с объединенными тэгами Функция предварительного просмотра теперь также поддерживает замененные объединенные тэги, таким образом, пользователь сможет проверить вид письма перед отправкой.
     
  • Высокая скорость SendBlaster теперь поддерживает до 50 соединений - это в значительной степени может повлиять на скорость осуществления рассылки при использовании быстрого SMTP сервера. Более того, теперь SendBlaster предоставляет возможность отмены прерывания рассылки при временном прекращении работы Internet соединения. Данная опция может быть полезной при автоматической рассылке.
     
  • Индивидуальные формы для осуществления подписки на рассылку и автоматических сообщений Теперь пользователи могут настроить тему для подписки и отмены подписки на расылку, а так же для простоты добавить форму для автоматических ответов, сообщений которые еще находятся в процессе доставки, а также для недоставленных сообщений.
     
  • SMTP и POP3 профили Использовать различные настройки SMTP и POP3 стало еще проще, чем раньше - информация об аккаунте может быть сохранена в нескольких профилях, таким образом, все SMTP и/или POP3 настройки могут быть изменены всего несколькими кликами. Эта особенность является очень полезной при рассылки писем от лица кого-либо другого.
     
  • Snapshots Сохраните все настройки для любой панели и используйте их снова, вызвав их всего одним кликом - это проще, чем сделать snapshot. Снэпшоты расширяют концепцию "профиля" до нового уровня - практически каждая группа относящихся к делу настроек может быть сохранена как snapshot.
     
  • Поддержка внешних плагинов Пользователи могут загрузить бесплатные плагины (небольшие инструменты, выполняющие определенные задачи,например, такие как осуществление импорта данных из различных e-mail клиентов, и т.д.) с нашего сайта, позже еще будут добавлены дополнительные плагины. Plugins API являются разработчиками плагинов и будут рады созданию новых плагинов.
     
  • Новые шаблоны, которые легче исправить Вы найдете в программе SendBlaster 2 более, чем 100 новых специально разработанных шаблонов, все они основаны на макетах, которые с легкостью можно исправить.
     
  • Доступ к файлам с информацией стал проще Прямой доступ к файлам с базами данных стал проще, т.к. теперь они сохраняются в папке Документы
     
  • Не требуется прав администратора Программа SendBlaster 2 может быть использована обычным пользователем в любой системе Windows
     
  • Устранены недочеты Было устранено множество мелких недочетов, а также была улучшена работа программы по различным параметрам вцелом.
     
  • Поддержка Windows 7 Программа была протестирована на совместимость со следующей операционной системой Microsoft; программа работает на операционной системе Windows 7 без дополнительных настроек.
     
  • И многое другое! SendBlaster 2 претерпела множество других различных изменений, которые только улучшили работу программы.
