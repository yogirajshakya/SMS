FAQ

Když posílám zprávu několika adresátům, vidí oni své adresy navzájem?

Ne. SendBlaster posílá každou zprávu zvlášť: soukromí každého adresáta je zachováno. Použití programu SendBlaster je vhodné pro zabránění tomu, aby adresáti viděli své adresy navzájem, což odporuje respektování soukromí a někde i zákonu.
 

Jak mohu filtrovat seznam?

Použitím funkcí pro seskupování/třídění v sekci Spravovat seznamy a dočasným povolením/zakázáním skupiny adres můžete posílat zprávu podskupině všech adres v současném seznamu. Taky můžete použit tlačítko Filtr.

Jak mohu do těla zprávy vložit odkaz na odhlášení se z distribučního seznamu?

SendBlaster umí své distribuční seznamy spravovat stahováním zpráv z dedikované e-mailové adresy, která je pro každý seznam různá (viz Spravovat seznamy). Použitím tlačítka Vložit odkaz pro odhlášení, při sestavovaní zpravy, vy vložite do odkazu nasledujici kód:

<a href="mailto:#managingemail#?Subject=Unsubscribe">Odhlásit</a>

Kód HTML musíte vložit do HTML části zprávy při editaci/psáni kliknutím na Zobrazit kód

Když posílám zprávu distribučnímu seznamu, některé zprávy se nedoručí a vracejí se s chybou.

Když posíláte na mnoho adres, je normální, že se ne všechny zprávy odešlou úspěšně: některé adresy se mohly stát neaktivní nebo jsou schránky přeplněné, nebo může být spojení dočasně přerušené nebo příliš pomalé.

Pokud je procento úspěšně odeslaných zpráv nízké nebo se dokonce ani jedna zpráva neodešle úspěšně, musíte se podívat na nastavení rozesílání.

Může být nutné změnit režim rozesílání (z přímého rozesílání na rozesílání pomocí SMTP nebo naopak) a, pokud používá Váš server SMTP filtry proti rozesílání spamu, i snížit počet simultánních spojení a zvýšit čekací dobu mezi zprávami. Server SMTP může mít i omezení, která zabrání odesílání většího než určitého počtu zpráv do určitého množství času (např. daný server nedovoluje rozesílání více než 50 zpráv za hodinu); v takovém případě, po prvním rozesílání dokončíte operaci později pomocí funkcí pro Historii.

Pokud není možné přebít antispamová omezení SMTP serveru a rozesílání funguje jenom při velmi pomalých nastaveních, můžete zkusit přímé rozeslání nebo “tolerantnější“ server SMTP (na Internetu je možné si připlatit i za velmi málo peněz hodně účinné služby SMTP).

Pokud vznikne nějaký problém při používání režimu přímého rozesílání, pak může být nutné ručně určit IP adresu serveru DNS.
 

Ztratil jsem heslo a už nemám k programu přístup. Co mohu udělat?

Mějte na vědomí, že není možné znovu získat ztracená data. Následující procedura umožní pouze znovu získat přístup k programu.

1. Odinstalujte program v Ovládacích panelů Windows.

2. Ručně odstraňte instalační složku (tj. C:\Program files\SendBlaster), včetně všech podsložek; z bezpečnostních důvodů se při odinstalaci programu neodstraňuje databáze (proto je třeba ji odstranit ručně).

3. Instalujte program znovu.
