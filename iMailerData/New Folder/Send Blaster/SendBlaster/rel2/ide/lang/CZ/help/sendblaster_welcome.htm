Vítejte v programu SendBlaster, jednoduchém a dovedném nástroji pro správu distribučních seznamů z vašeho počítače:

  • Správa více seznamů

     
  • Generování letáků s přílohami a vloženými obrázky
       Nejnovější programy pro čtení pošty nedovolují zobrazení vzdálených obrázků: proto vkládá SendBlaster obrázky do těla zprávy a tím se vyhýbá všem problémům s čtením zpráv s obrázky
     
  • Import HTML stránek, tvorba zpráv v obyčejném programu pro e-mail (formát .eml)
       Externí obrázky, na které se odkazuje v kódu HTML jsou vložené do zprávy a všechny přílohy nacházející se v souboru .eml se zachovají.
     
  • Rozesílání personalizovaných zpráv
       Každý adresát může dostávat zprávu napsanou jemu na klíč: zprávy mohou obsahovat "značky", které se nahrazují údaji o adresátovi
     
  • Rozeslílání zpráv přímo bez serveru SMTP
       Pomocí režimu přímého rozesílání můžete využívat více připojení na internet najednou bez změny nastavení pro napojení na server SMTP poskytovatele internetu: zprávy se posílají přímo poštovním serverům adresátů
     
  • Rychlé rozeslání mnoha zpráv
       Díky jádru SmartSend můžete posílat hodně zpráv zároveň a tím silně zvýšit rychlost rozesílání

     
  • Správa přihlášení a odhlášení z dedikované e-mailové adresy
       Uživatelé se mohou přihlásit nebo odhlásit jednoduše tím, že pošlou zprávu uživatelsky definované adrese s předmětem zprávy "Subscribe" nebo "Unsubscribe ". SendBlaster stahuje zprávy pomocí serveru POP3 a pak seznamy zaktualizuje
     
  • Import z adresáře programu Outlook Express
     
  • Import seznamu ze souborů .csv a z webových zdrojů
       Pokud už má Váš web systém pro přihlášení a odhlášení a má skript pro export, můžete využít toho, že SendBlaster umí přímo vyzvedávat a načíst data a pak obnovit Vaši místní databázi
     
  • Snadná integrace do webových stránek
       Je snadné přidat na svém webu funkce pro přihlášení/odhlášení: není třeba skriptů, stačí jen přidat odkaz.
