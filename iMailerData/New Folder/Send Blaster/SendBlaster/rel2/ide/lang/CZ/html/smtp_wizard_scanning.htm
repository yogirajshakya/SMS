Skenování

Průvodce nyní bude skenovat vaše PC na hledání dostupných portů, pokusí se připojit k SMTP serveru a poslat testovací zprávu. Toto může trvat několik minut, závisí na tom jak rychlý a dostupný port je nalezen.

Jestli u vás běží bezpečnostní software (firewall / antivirový program), to program se vás může ptát jestli chcete povolit SendBlasteru přístup na Internet; na dotaz prosím odpovězte "ano" nebo "povolit" aby udělit programu SendBlaster úplný přístup k Internetu. Bylo by vhodné, aby váš antivirový program pro kontrolu pošty k odeslání byl dočasně zastaven.

Prozatím, můžete se podívat na SMTP.com služby navržené jen pro uživatele SendBlaster.
