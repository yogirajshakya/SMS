Release Notes

  • This is a beta release, still under development; some of the features are not fully working at present.
  • Documentation has not been updated and still refers to release 1.

What's new in sendBlaster 2

Development of SendBlaster 2 took more than one year; during every stage of development, we listened carefully to our users' suggestions, requests and opinions.

  • Improved and redesigned user interface SendBlaster is even easier to use than before. Users can choose between the classic left panel, redesigned to be visually more appealing and now supporting large or small icons, and a new toolbar which saves horizontal space on the screen. All panels are now scrollable with the mouse wheel; list selection can be synchronized across all the panels and are alphabetically ordered; all icons have been redesigned; and many more small user interface improvements have been added.
     
  • Scheduled mailings Now users can create a mailing and let SendBlaster 2 send it later automatically, at a given day and time. Or, users can even "park" their mailing so that they can manually launch it later. Now SendBlaster can run in a minimized schedule mode and start when Windows starts.
     
  • E-mail tracking SendBlaster 2 has built-in support, wizards and panels for Google Analytics, Google's campaign monitoring free service, and for TrackReports, our advanced tracking and reporting online service.
     
  • New, powerful data filters A powerful filter has been added, which lets users fine tune their mailing by selecting and enabling or disabling records matching some conditions. It supports multiple conditions and complex operators, so that they can visually build a powerful SQL query without writing code. Duplicates can now be filtered out automatically, as in release 1, or managed by the user through a new, powerful search engine.
     
  • More options for sending out Users can now add custom headers, request read receipts, set a high priority for their message, define a customized To: header supporting merge tags, enable or disable automatic filtering out of duplicate addresses when sending out.
     
  • More options for editing messages Many new buttons in the composer panel let users easily add removal links in the message, add special links for Google Analytics and TrackReports, and add merge tags in the subject and To: field. Now text-only (as opposed to HTML) messages can more easily be created by just checking one box.
     
  • Spam checking It is now possible to perform a spam-score check on the fly while editing a message; spam score checking is based on popular SpamAssassin engine, one of the most common antispam filters with e-mail servers.
     
  • Preview with merge tags Previewing now supports substituting merge tags too, so that users can have a more exact idea of what will be sent.
     
  • More speed SendBlaster now supports up to 50 connections - this could dramatically improve the speed of mailings when using a fast SMTP server. Furthermore, SendBlaster now gives users the option to not interrupt a mailing should the Internet connection temporarily stop working, which may help with unattended mailings.
     
  • Customizable subscription and bounce-back formats Users can now customize subscription and cancellation subjects and add custom bounce-back formats for easier, automatic processing of undelivered messages.
     
  • SMTP and POP3 profiles Using multiple SMTP and POP3 settings is easier than before - account info can be saved in multiple profiles, so that all SMTP and/or POP3 settings can be changed with a couple of clicks. This is expecially useful when sending out mailings on behalf of someone else.
     
  • Snapshots Save all settings for any given panel and reuse them later with just one click - it's as easy as taking a snapshot. Snapshots extend the concept of "profiles" to a new level - almost every group of related settings in SendBlaster can be saved as a snapshot.
     
  • Support for external plug-ins Users can download free plug-ins (small tools performing specific tasks, as importing from various e.mail clients, etc) from our web site, and many more will be added in the future. Plugins API is public and third party developers are encouraged to create new, useful plugins.
     
  • New, easier-to-edit templates SendBlaster 2 has more than 100 new professionally designed templates, all of them using flexible layouts which can be easily customized.
     
  • Easier access to data files Directly accessing database files for backup purposes is now easier, as they are saved inside Documents folder
     
  • No administration privileges required SendBlaster 2 can be run by standard users on any Windows system
     
  • Debug and fine tuning Many minor bugs have been solved, and performance has been fine-tuned in many areas.
     
  • Ready for Windows 7 The software is constantly being tested with Microsoft's next operating system; it runs natively on it and does not require any compability workaround.
     
  • And much more! SendBlaster 2 has many more small improvements which let users work better and faster.
