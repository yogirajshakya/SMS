Could not connect to the server

We are sorry connecting with the server and sending a test message was not successful. This may have several causes:

  1. All ports are blocked by some firewall running on your PC or on your Internet gateway
  2. An antivirus software is blocking outgoing traffic
  3. A temporary network or server failure is also possible, therefore we suggest that you check your Internet connection and try again in a few minutes

If the problem persists, please configure your firewall and antivirus so that SendBlaster is allowed to connect to the Internet and send e-mails (you may have to ask for support to your network administrator) and make sure your Internet Service Provider is not disallowing this feature or blocking any ports. You can find additional information here: Troubleshooting network problems. After the issue has been solved, you can run the Wizard again clicking on "SMTP.com Wizard" button in SendBlaster's "Send" form.
