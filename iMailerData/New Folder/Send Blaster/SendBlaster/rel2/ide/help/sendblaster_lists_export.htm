Export

SendBlaster allows you to export data from a distribution list* to a .csv format file. After selecting the list to be exported, you can choose one of the available exporting methods:

  1. Export all addresses: all the addresses in the list will be exported.
     
  2. Export only subscribed addresses: only currently subscribed addresses will be exported from the selected list (addresses marked with the  symbol in the Manage lists window).
     
  3. Export only unsubscribed addresses: only currently unsubscribed addresses will be exported from the selected list (addresses marked with the  icon in the Manage lists window).
     
  4. Export only enabled addresses: only currently enabled addresses will be exported. Using the advanced filtering options found in the Manage lists window, this method allows you to define selective and specific exports.

Clicking on the Export button will show a window in which to set the name of the exporting file. The Field delimiter will also need to be specified (the column separator character) and you will also be able to specify whether First row contains field names. Finally, you’ll be able to select the fields to be exported and, after clicking on the Next button, exporting process will start.

* Exporting functions are only available in the Pro edition.
