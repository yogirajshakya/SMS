Fields

In this window you can set a few program related options.

Set field names: SendBlaster allows you to customize the names of the 15 available fields for each address (the E-mail field is highlighted in red since, regardless of its name, it is the field used to send messages and must always contain the recipients' e-mail addresses). Simply type the new names in the corresponding boxes and click on the Save button.

Warning: After changing field names, the message composition tags will also change; therefore message text might also need to be modified accordingly.

My personal data: In this section you can add your personal details that will be used for merging feature while previewing an e-mail message in the compose screen, this feature will provide you a more exact idea of the e-mail message that will be sent.
