Manage lists

SendBlaster can manage multiple distribution lists*.

The current list is selected through the drop-down box found in the upper side of the window. Next to it, you can find 3 buttons that will let you to create a new list (button New), rename the current list (Rename), delete it (Delete) or send it to the printer (Print). These tasks can also be performed by clicking on the drop-down box with the right mouse button.

The remaining area of the window is dedicated to the address list of the selected distribution list.

Each address shows: the subscription state (marked by the and icons), the activation state (marked by a checkbox), the contents of the 15 custom fields (the first one is always the e-mail address), the presence of comments (marked by the icon, clicking on it will open the comments). Columns order can be modified by moving columns (simply drag the column header). The columns can also be resized.

During a send operation, the addresses marked as unsubscribed or inactive will be skipped. The sending operation will therefore affect only those addresses that are currently both subscribed and active. The address list can be freely sorted by clicking on column headers. Dragging one or more column headers in the area above the list (Drag a column header in this area to group data) will let you to group addresses based on the contents of the selected fields. The Total checkbox will also be shown, enabling you (when checked) to find out the number of addresses included in each group.

     

How to filter addresses

Temporairily activating/deactivating one or more items (without having to unsubscribe them) will allow you to send a message to a subgroup of addresses in the list. With this feature you will be able to create powerful filtering to be applied to mailing processes.

For instance, if a "City" field is defined you can send a message only to those that live in a specific city, temporarily disabling all other addresses. If a "Job" field is defined you'll be able to send a message only to certain professional categories (it will also be possible to combine these 2 filters.

SendBlaster provides a powerfull data filter builder which lets users fine tune their mailing by selecting and enabling or disabling records matching some conditions. It supports multiple conditions and complex operators, so that they can visually build a powerful SQL query without writing code. This filter builder can be accessed using the Filter button.

Disabling of one or more addresses will also enable you to create exclusion lists (addresses to which no message will ever be sent, not even if the user will spontaneously subscribe to the list).

      

On top of the address list you can find the list management toolbar. You can access these commands also through the context menu that is displayed when right-clicking with the mouse on the list.

You can insert a new address (button New), modify the selected address (Modify) or delete one or more addresses (Delete). You can select multiple addresses using both the keyboard and the mouse: clicking on an address while the Ctrl key is pressed will add the address to the selection, clicking on an address while the Shift key is pressed will add a group of addresses.

With the Cut / Copy / Paste buttons you can delete the addresses or copy them to/from the clipboard (if you need to select all records right click on any record and choose Select all).

The buttons Subscribe selected addresses and Unsubscribe selected addresses will allow you to manually manage subscriptions to the selected list (subscribing an address will cause a modification of the subscription date). The buttons Enable selected addresses and Disable selected addresses will let you to activate/deactivate an addresses regardless of its subscription state.

The Filter button allows users to filter data using up to 5 filtering rules, much like a SQL query would do but without the complexities; it also allows to specify an action to perform on all contacts that match the rules: Activate/Deactivate them, or Subscribe/Unsubscribe them, you can also choose to Deactivate non matching contacts, or just leave them as they are.

The History button will show a list of previously sent messages to the selected address.

Clicking on the Notes button will show a new window in which to insert a free text to be associated with the current address. The addresses associated with a note are marked by the icon in the last column (clicking on this icon will also show the Notes window).

The Search** button will show a window that will let you to search a text in a specific field or in multiple fields (including notes). By selecting Match whole word the searching operation will display all addresses that match exactly the value specified. Otherwise the searching operation will display all addresses that contain the value even if it doesn't exactly match the value specified. To move through the resulting list use the Find first, Next and Previous buttons in the search window.

The Find Duplicates button will show a window that allows to manage duplicate email addresses across lists. You can search for duplicates in selected lists, and remove or edit duplicate contacts from the duplicates window. Notice that you can have duplicate e-mail addresses in your lists and choose wether you want to send messages to duplicate e-mail addresses in the Advanced Settings window.

The Toggle view button enables you to switch between the normal view and the "Card view" (and back). In the "Card view" each address will be shown as a separate card with a cutomizable width. Navigation is made possible by the horizontal scroll bar.

You may want to use only a subset of the 15 available fields; in order to save space on screen, by clicking on Visible fields you can choose which fields will be displayed.

*The Free edition can manage up to 2 distribution lists.
**Search is only available in the Pro edition
