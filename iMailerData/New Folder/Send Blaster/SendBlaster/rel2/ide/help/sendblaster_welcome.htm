Welcome to SendBlaster, the simple and powerful software to manage mailing lists from your PC:

  • Manage multiple lists
     
  • Generate newsletters with attachments and embedded images
       The most recent operating environments don't allow displaying of remote images: images are therefore embedded in the message body, avoiding all message-reading related problems
     
  • Import HTML pages and e-mail client generated messages (.eml format)
       External images referenced in the HTML code get embedded in the message and all attachments found in the .eml file are preserved.
     
  • Send personalized messages
       Each recipient can receive a personalized message: e-mail messages can contain tags that are replaced with the recipient's data
     
  • Enables you to directly send messages with no SMTP server
       Through direct send mode, if you use several internet connections you won't need to modify any settings to connect to the provider's SMTP service: messages will be directly dispatched to recipients' email servers
     
  • Send many messages fast
       Thanks to the powerful SmartSend engine many messages can be sent simultaneously, dramatically increasing sending speed
     
  • Manage subscriptions and unsubscriptions from a dedicated e-mail address
       Users can subscribe or unsubscribe by simply sending to a user-defined e-mail address a message with "Subscribe" or "Unsubscribe " in the subject line. SendBlaster downloads the messages via POP3 and updates the lists
     
  • Import from Outlook Express address book
     
  • Import list from .csv files and web resources
       If your web site is already equipped with a subscribe/unsubscribe system and an exporting script, SendBlaster can directly retrieve and load data and then update your local database
     
  • Easy to integrate with any web site
       Adding mailing list subscription/unsubscription functionalities to your web site is easy: no need for scripts, just add a simple link
