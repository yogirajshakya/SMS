Advanced Settings

In this window you can set a few program advanced options.

Interface:

  • Appearance: allows to specify if the side panel should be shown with an Explorer Bar style (optionally with Small Icons in order to save space), or Toolbar style.
     
  • Show TrackReports link in panel: allows to show/hide TrackReports link in the main panel; if you do not plan to use TrackReports, you can hide this link to save space on the bar.
     
  • Show Google Analytics link in panel: allows to show/hide Google Analytics link in the main panel; if you do not plan to use Google Analytics, you can hide this link to save space on the bar.
     
  • Show tips at startup: enable tips window to be shown when SendBlaster starts. It's a good idea to leave this option enabled untill you get familiar with SendBlaster.
     
  • Syncronize list selection across panels: this is a very powerful option, which allows you to keep working on the same list across all panels until you change it. As an example, if you select a list in the Send screen, then go to the Manage lists section in order to add a new address, the list you selected in the Send screen, will be already selected in Manage lists section. The drawback is that, if you have very large lists and a low-end pc, this option may slow down the program a little: if you experience any unwanted delay in the responsiveness of the interface, just disable it.
     
  • Use the Language button to change SendBlaster's user interface language, the selected language will be used next time SendBlaster starts.

Messages:

  • Default encoding for new messages: This option allows you to set the default character encoding used for the composition of new messages.

Email subscriptions:

  • This section allows to specify the subject lines that should be used for subscription and unsubscription messages. You can also choose whether messages must match them exactly, in order to be treated as subscriptions/cancellations, or just need to include the specified words. For backward compatibility reasons, "Subscribe" and "Unsubscribe" subjects will always work too (in addition to the subjects you define here).

Bounce-backs custom formats:

  • Using the Edit formats button, you can add or edit custom bounce-back formats. Since bounce-back email formats were originally designed to be read by humans, and there is no standard, SendBlaster can process most common bounce-back formats using its internal database, but you may encounter some formats which are not recognized by the software. If this happens, you can manually add them to SendBlaster's bounce-backs database so SendBlaster can process this new bounce-back formats too.

Schedule:

  • This section allows to configure SendBlaster to start in schedule mode (minimized in the tray system area) when Windows starts; this way scheduled mailing operations can be automatically start when due.

Send:

  • This section allows to specify wether you want SendBlaster to send duplicate e-mails to your contacts when contacts exist in lists more than once. Usually, you do not need to send more than one copy of each e-mail message to your contacts, we strongly advice you leave this option unchecked, unless you have a very particular and specific reason to send duplicate emails to duplicate contacts.

Plugins:

  • This section allows you to enable or disable currently installed SendBlaster Plugins (without actually uninstalling them, which you can do from Windows' Control Panel or using each plugin's uninstaller). You can also download more SendBlaster plugins using the More Plugins button.

Database:

  • Using the Database folder button, will open the windows folder where user data is stored; this is useful for mainteinance, backup and other database operations.
