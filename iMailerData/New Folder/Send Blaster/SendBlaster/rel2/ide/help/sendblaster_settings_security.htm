Security

Software access can be password protected*.

  • To set a new password (in case software access wasn't previously protected) type it in the New password field and type it again in the Confirm password field, finally click on the Save button.
     
  • To modify a password type the current password in the Old password field, then type the new password in the New password field and type it again in the Confirm new password field, finally click on the Save button.
     
  • To remove a password (making software access available to anyone) type the current password in the Old password field, leave the New password and Confirm new password fields empty, then click on the Save button.

New settings will be available the next time the software will be run.

Warning: If you decide to protect software access take good care of your password! No special functionalities are provided to gain access to the password if it gets lost and you will be no longer able to access the software. To re-gain access to the software you will need to uninstall it, manually remove all files and directories in the installation path (for security reasons database files don't get uninstalled), and finally install it again. In this case, all your data will be lost!

*Password protection is available in the Pro edition
