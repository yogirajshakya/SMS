Google Analytics

This panel lets you easily navigate Google Analytics website without opening an external browser window. The navigation bar is hidden by default, but you can make it visible by clicking on the Navigate bar; the familiar Back, Forward, Home, Reload and Stop buttons will then appear.

Google Analytics is the most widespread web analytics solution; it gives you rich insights into your website traffic and marketing effectiveness. If you already use Analytics on your website, you will be able to easily track your e-mail campaigns too: just use Google Analytics button in SendBlaster's message editing panel:

  1. Compose your message and insert links
  2. Once you're finished, click on Google Analytics button
  3. Enter your campaign details (please see Google Analytics website if you need help)

All links in the message will be automatically updated so that they include the needed Google Analytics code. Since the original HTML code is modified, adding Google Analytics code must always happen only after the editing of the message is finished (in other words, it must be tha last thing you do when you compose your message)*.

Please remember that Google Analytics must already be installed on your website (or on the website recipients are redirected to). You need to be familiar with the service in order to be able to understand the tracking information it provides.

*Except when you want some links not to be tracked: in that case, you need to add the un-tracked links after Google Analytics code has been added to the message. Furthermore, if you want to use both Google Analytics and TrackReports, add Google Analytics first and TrackReports last (as it uses a redirection system).
