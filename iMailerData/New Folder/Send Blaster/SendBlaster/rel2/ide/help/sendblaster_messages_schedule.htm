Schedule

SendBlaster allows to schedule messages, so they can be sent at a specified time. Mailings can also be "parked": they are ready for being sent, but will not be automatically launched - user will manually launch a parked mailing at a later time.

Important: Scheduled mailings are only launched when SendBlaster is in "Schedule mode" - in other words, when it's minimized in the tray area. No mailings, even if the scheduled time has elapsed, will be automatically launched if the software is in "normal" mode. Click on Run in schedule mode button when you want SendBlaster to be minimized in the tray area and process scheduled mailings (you can revert back to normal mode by clicking on the its icon in the system tray).

The Schedule screen shows a toolbar and the list of scheduled mailing operations.

Each scheduled mailing operation shows: whether the scheduled operation is active or not, the date and time when mailing operation will start, the mailing operation title, the destination list for the mailing operation and the subject of the scheduled mail message. Columns can be resized.

The scheduled mailing operations can be freely sorted by clicking on column headers.

On top of the address list you can find the schedule toolbar. You can access these commands also through the context menu that is displayed when right-clicking with the mouse on the list.

You can schedule current mailing operation (button New), change schedule date for the selected scheduled mailing operation (Edit date), preview the selected scheduled message (E-mail preview), start the selected mailing operation (Send Now), or delete one or more mailing operations (Delete).
